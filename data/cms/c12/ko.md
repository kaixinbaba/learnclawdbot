# 생산성을 높이는 OpenClaw 필수 플러그인 10가지

기본 상태의 OpenClaw도 훌륭한 AI 어시스턴트입니다. 하지만 진정한 위력은 **플러그인**에서 나옵니다. 플러그인은 매일 사용하는 실제 도구들과 어시스턴트를 연결하는 모듈식 확장 기능입니다. 웹 검색, 코드 저장소 관리, WhatsApp 메시지 전송 등 어떤 것이든 해당하는 플러그인이 있습니다.

## OpenClaw 플러그인이란?

OpenClaw 플러그인은 `openclaw.plugin.json` 매니페스트 파일로 정의됩니다. 각 플러그인이 선언하는 것:

- **이름**과 **설명** (AI가 언제 호출할지 판단하는 데 사용)
- 노출하는 **액션** (예: `search`, `create_issue`, `send_message`)
- 필요한 **인증** 정보 (API 키, OAuth 토큰)
- 선택적 **트리거** (웹훅, 스케줄)

AI는 실행 시 이러한 매니페스트를 읽고 요청과 관련이 있다고 판단할 때 사용 가능한 플러그인 액션을 호출합니다. 커뮤니티 플러그인은 [ClawdHub](https://clawdhub.com)에서 탐색하고 설치할 수 있습니다.

## 필수 플러그인 TOP 10

### 1. Google Search (구글 검색)

**기능**: 실시간 웹 검색을 수행하고 구조화된 결과, 뉴스, 스니펫을 대화 컨텍스트에 직접 가져옵니다.

**필요한 이유**: AI 어시스턴트에게 실시간 정보 접근 능력을 부여합니다 — 주가, 뉴스, 학습 데이터 이후의 문서 등.

**최적 활용**: "Node.js 현재 LTS 버전이 뭐야?" 또는 "[제품명] 최근 리뷰 찾아줘"

---

### 2. Browser Relay (브라우저 릴레이)

**기능**: 헤드리스 브라우저를 제어해 JavaScript 의존 사이트나 단순 스크래퍼를 차단하는 사이트를 포함한 모든 웹페이지의 콘텐츠를 가져오고 렌더링하며 추출합니다.

**필요한 이유**: 검색이 링크를 반환하지만 페이지의 전체 콘텐츠가 필요할 때.

**최적 활용**: "[URL] 기사 요약해줘" 또는 "[경쟁사 웹사이트]에서 가격표 추출해줘"

---

### 3. File System Manager (파일 시스템 관리자)

**기능**: 로컬 머신이나 연결된 드라이브에서 파일과 디렉토리를 읽기, 쓰기, 이동, 복사, 삭제합니다.

**필요한 이유**: AI 어시스턴트가 실제 파일을 관리할 수 있게 됩니다 — 다운로드 정리, 사진 일괄 이름 변경, 로그 파일 읽기로 문제 진단 등.

**최적 활용**: "/var/log에서 오늘 수정된 모든 .log 파일을 찾아서 오류를 요약해줘"

---

### 4. GitHub Integration (GitHub 연동)

**기능**: GitHub 저장소와 상호작용 — 이슈 생성, PR 열기, 파일 내용 읽기, 코드 검색, 댓글 작성.

**필요한 이유**: AI 어시스턴트를 실제로 변경사항을 푸시하고 작업을 추적할 수 있는 코딩 협업자로 만듭니다.

**최적 활용**: "방금 논의한 버그에 대한 GitHub 이슈 만들어줘" 또는 "내 리뷰를 기다리는 오픈 PR 있어?"

---

### 5. Twilio Voice (Twilio 음성)

**기능**: Twilio API를 통해 전화 발신/수신, SMS 전송, 전화와 AI 어시스턴트 간 브리지 연결.

**필요한 이유**: 진정한 음성 우선 인터페이스를 가능하게 합니다 — 어떤 전화기에서도 AI 어시스턴트에게 전화할 수 있습니다.

**최적 활용**: TTS/STT 프로바이더와 결합해 완전 음성 활성화 어시스턴트 구축 ([음성 어시스턴트 튜토리얼](/blog/voice-assistant-openclaw) 참조)

---

### 6. Calendar (캘린더, Google / Outlook 지원)

**기능**: Google Calendar와 Microsoft Outlook에서 캘린더 이벤트를 읽기, 생성, 업데이트, 삭제합니다.

**필요한 이유**: AI 어시스턴트가 일정을 파악하고 계획을 도울 수 있습니다 — "내일 오전에 딥 워크용으로 2시간 블록해줘" 또는 "이번 주 회의가 어떻게 돼?"

**최적 활용**: "다음 주 화요일 오전 10시~12시 사이에 John과 30분 미팅 잡아줘"

---

### 7. Email (이메일, Gmail / Outlook 지원)

**기능**: 연결된 계정에서 이메일을 읽기, 초안 작성, 전송, 정리합니다.

**필요한 이유**: 이메일 분류와 초안 작성을 AI에 위임합니다. "지난 24시간 미읽음 이메일 요약해줘" 또는 "지난 금요일에 보낸 제안서 후속 이메일 초안 써줘"

**최적 활용**: Calendar 플러그인과 결합해 받은 편지함에서 캘린더까지의 전체 워크플로 자동화

---

### 8. WhatsApp

**기능**: WhatsApp Business API 또는 지원되는 서드파티 브리지를 통해 WhatsApp 메시지를 송수신합니다.

**필요한 이유**: 많은 팀과 가족이 주로 WhatsApp으로 소통합니다 — AI를 그 대화에 참여시키세요.

**최적 활용**: 가족이 OpenClaw 전체 인터페이스 없이 WhatsApp으로 질문할 수 있는 개인 AI 어시스턴트 구축

---

### 9. Slack

**기능**: Slack에서 메시지 게시, 채널 기록 읽기, 워크스페이스 콘텐츠 검색, 멘션에 응답합니다.

**필요한 이유**: AI 어시스턴트를 팀 워크플로에 통합합니다. "#engineering에 스탠드업 업데이트 게시해줘" 또는 "지난 주 API 장애에 관한 Slack 논의 검색해줘"

**최적 활용**: 매일 아침 주요 채널 자동 요약을 DM으로 배달

---

### 10. ClawdHub Marketplace

**기능**: 수동 파일 편집 없이 OpenClaw 내에서 직접 플러그인을 탐색, 설치, 업데이트, 관리합니다.

**필요한 이유**: 플러그인 생태계는 계속 성장합니다. ClawdHub는 한 명령으로 최신 상태를 유지해 줍니다: "ClawdHub에서 Notion 플러그인 설치해줘"

**최적 활용**: Jira, Linear, Notion, Airtable 등 틈새 도구를 위한 커뮤니티 제작 플러그인 발견

## 플러그인 설치 방법

OpenClaw에 플러그인을 설치하는 것은 세 단계입니다:

**1단계: ClawdHub 탐색**

[clawdhub.com](https://clawdhub.com)을 방문해 원하는 플러그인을 찾습니다. 각 목록 페이지에는 플러그인의 액션, 필요한 자격 증명, 커뮤니티 평점이 표시됩니다.

**2단계: Dashboard에서 설치**

1. `http://localhost:3000`에서 OpenClaw Dashboard 열기
2. **플러그인 → 탐색** 메뉴 이동
3. 플러그인 이름으로 검색
4. **설치** 클릭
5. 필요한 API 키를 입력하거나 OAuth로 인증

**3단계: 테스트**

설치 후 OpenClaw에게 사용해 보도록 요청합니다:

> "GitHub 플러그인 써서 내 오픈 이슈 목록 알려줘"

AI가 자동으로 플러그인을 호출하고 결과를 반환합니다.

**수동 설치** (커스텀 또는 프라이빗 플러그인용):

```bash
# 플러그인 폴더를 여기에 배치:
/home/user/openclaw/plugins/my-plugin/
# openclaw.plugin.json이 있는지 확인 후 재시작:
npm run restart
```
