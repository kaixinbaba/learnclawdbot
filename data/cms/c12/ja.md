# 生産性を高めるOpenClawの必須プラグイン10選

OpenClawはそのままでも優れたAIアシスタントです。しかし、その真の力は**プラグイン**によって発揮されます。プラグインとは、毎日使うリアルなツールとアシスタントを接続するモジュール式の拡張機能です。ウェブ検索、コードリポジトリの管理、WhatsAppメッセージの送信など、あらゆるニーズに対応するプラグインが揃っています。

## OpenClawプラグインとは？

OpenClawプラグインは`openclaw.plugin.json`というマニフェストファイルによって定義されます。各プラグインが宣言するのは：

- **名前**と**説明**（AIがいつ呼び出すかを判断するために使用）
- 公開する**アクション**（例: `search`、`create_issue`、`send_message`）
- **認証**要件（APIキー、OAuthトークン）
- オプションの**トリガー**（Webhook、スケジュール）

AIはこれらのマニフェストを実行時に読み込み、リクエストに関連すると判断した場合に任意のプラグインアクションを呼び出します。コミュニティプラグインは[ClawdHub](https://clawdhub.com)でブラウズ・インストールできます。

## 必須プラグインTOP 10

### 1. Google Search（グーグル検索）

**できること**：リアルタイムのウェブ検索を実行し、構造化された結果・ニュース・スニペットをチャットコンテキストに直接取り込みます。

**必要な理由**：AIアシスタントにリアルタイム情報へのアクセスを与えます — 株価、ニュース、学習データのカットオフ以降のドキュメントなど。

**最適な使い方**：「Node.jsの現在のLTSバージョンは？」や「[製品名]の最近のレビューを探して」

---

### 2. Browser Relay（ブラウザリレー）

**できること**：ヘッドレスブラウザを制御して、JavaScriptに依存するサイトや単純なスクレイパーをブロックするサイトを含む、あらゆるウェブページのコンテンツを取得・レンダリング・抽出します。

**必要な理由**：検索がリンクを返しても、ページのフルコンテンツが必要なとき。

**最適な使い方**：「[URL]の記事を要約して」や「[競合サイト]の料金表を抽出して」

---

### 3. File System Manager（ファイルシステムマネージャー）

**できること**：ローカルマシンや接続されたドライブ上でファイルやディレクトリの読み書き・移動・コピー・削除を行います。

**必要な理由**：AIアシスタントが実際のファイルを管理できるようになります — ダウンロードの整理、写真の一括リネーム、ログファイルの読み取りによる問題診断など。

**最適な使い方**：「`/var/log`内で今日更新された`.log`ファイルをすべて見つけてエラーをまとめて」

---

### 4. GitHub Integration（GitHub連携）

**できること**：GitHubリポジトリとのやり取り — Issue作成、PRのオープン、ファイル内容の読み取り、コード検索、コメント投稿。

**必要な理由**：AIアシスタントを実際に変更をプッシュして作業を追跡できるコーディング協力者に変えます。

**最適な使い方**：「さっき話したバグのGitHub Issueを作って」や「私のレビュー待ちのオープンPRは？」

---

### 5. Twilio Voice（Twilio音声）

**できること**：TwilioのAPIを通じて電話の発着信・SMSの送受信・AIアシスタントとの電話ブリッジを実現します。

**必要な理由**：真の音声ファーストインターフェースを可能にします — どの電話からでもAIアシスタントに電話できます。

**最適な使い方**：TTS/STTプロバイダーと組み合わせて完全な音声起動アシスタントを構築（[音声アシスタントチュートリアル](/blog/voice-assistant-openclaw)を参照）

---

### 6. Calendar（カレンダー、Google / Outlook対応）

**できること**：Google CalendarとMicrosoft Outlookをまたいでカレンダーイベントを読み込み・作成・更新・削除します。

**必要な理由**：AIアシスタントがあなたのスケジュールを把握して計画を手伝えるようになります — 「明日の午前中に集中作業用に2時間ブロックして」や「今週の会議は？」

**最適な使い方**：「来週火曜日の10時〜12時の間にJohnと30分の通話をスケジュールして」

---

### 7. Email（メール、Gmail / Outlook対応）

**できること**：接続されたアカウントからメールを読み取り・下書き・送信・整理します。

**必要な理由**：メールのトリアージや下書きをAIに委任できます。「過去24時間の未読メールを要約して」や「先週金曜日に送った提案書のフォローアップを下書きして」

**最適な使い方**：Calendarプラグインと組み合わせて受信トレイからカレンダーまでのワークフロー全体を自動化

---

### 8. WhatsApp

**できること**：WhatsApp Business APIまたはサポートされているサードパーティブリッジを通じてWhatsAppメッセージを送受信します。

**必要な理由**：多くのチームや家族が主にWhatsAppでコミュニケーションしています — AIをそれらの会話に参加させましょう。

**最適な使い方**：家族がOpenClawのフルインターフェースなしにWhatsApp経由で問い合わせられる個人AIアシスタントを構築

---

### 9. Slack

**できること**：Slackでメッセージを投稿し、チャンネル履歴を読み取り、ワークスペースのコンテンツを検索し、メンションに返答します。

**必要な理由**：AIアシスタントをチームのワークフローに統合します。「#engineeringにスタンドアップ更新を投稿して」や「先週のAPI障害に関するSlackの議論を検索して」

**最適な使い方**：主要チャンネルの自動日次サマリーを毎朝DM配信

---

### 10. ClawdHub Marketplace

**できること**：手動でファイルを編集することなく、OpenClaw内から直接プラグインをブラウズ・インストール・更新・管理します。

**必要な理由**：プラグインエコシステムは常に成長しています。ClawdHubはワンコマンドで最新の状態を保ちます：「ClawdHubからNotionプラグインをインストールして」

**最適な使い方**：Jira、Linear、Notion、Airtableなどのニッチなツール向けコミュニティ製プラグインの発見

## プラグインのインストール方法

OpenClawへのプラグインインストールは3ステップです：

**ステップ1：ClawdHubを閲覧する**

[clawdhub.com](https://clawdhub.com)にアクセスして必要なプラグインを探します。各リストページにはプラグインのアクション・必要な認証情報・コミュニティ評価が表示されています。

**ステップ2：Dashboardからインストール**

1. `http://localhost:3000`でOpenClaw Dashboardを開く
2. **プラグイン → ブラウズ** に移動
3. プラグイン名で検索
4. **インストール** をクリック
5. 必要なAPIキーを入力するかOAuthで認証

**ステップ3：テストする**

インストール後、OpenClawに使わせてみましょう：

> 「GitHubプラグインを使って私のオープンIssueを一覧表示して」

AIが自動的にプラグインを呼び出して結果を返します。

**手動インストール**（カスタムまたはプライベートプラグイン向け）：

```bash
# プラグインフォルダをここに配置：
/home/user/openclaw/plugins/my-plugin/
# openclaw.plugin.jsonがあることを確認して再起動：
npm run restart
```
