# Создаём первого голосового ИИ-ассистента с OpenClaw

Идея о Джарвис-подобном ИИ — которому можно позвонить с любого телефона, спросить что угодно и услышать голосовой ответ — десятилетиями оставалась достоянием научной фантастики. С OpenClaw, Twilio и современными провайдерами распознавания (STT) и синтеза речи (TTS) это можно построить за один вечер.

Это руководство охватывает полную настройку: от конфигурации STT/TTS-провайдера до подключения телефонного моста и настройки личности ассистента для голосового взаимодействия.

## Технологический стек

| Компонент | Варианты |
|---|---|
| **ИИ-шлюз** | OpenClaw (self-hosted или облако) |
| **Телефонный мост** | Twilio Voice |
| **STT (речь→текст)** | Deepgram Nova-2, OpenAI Whisper |
| **TTS (текст→речь)** | ElevenLabs, OpenAI TTS, Google Cloud TTS |
| **ИИ-модель** | DeepSeek-V3, GPT-4o, Claude или любая поддерживаемая OpenClaw модель |

Не обязательно использовать всё одновременно. Минимальная рабочая конфигурация: **OpenClaw + Twilio + Deepgram (STT) + OpenAI TTS**.

## Предварительные требования

- OpenClaw установлен и доступен по публичному URL (или туннелирован через [ngrok](https://ngrok.com))
- Аккаунт Twilio с номером телефона
- API-ключ Deepgram (или OpenAI API-ключ для Whisper)
- API-ключ TTS-провайдера

## Шаг 1: Настройка STT/TTS-провайдера

### Вариант А: Deepgram (рекомендуется для STT)

Deepgram Nova-2 обеспечивает отличную точность в разговорном английском при низкой задержке — идеальный вариант для телефонных звонков.

```bash
# Добавьте в .env OpenClaw:
STT_PROVIDER=deepgram
DEEPGRAM_API_KEY=your_deepgram_key_here
DEEPGRAM_MODEL=nova-2
DEEPGRAM_LANGUAGE=en-US
```

### Вариант Б: OpenAI Whisper

Whisper — самая точная универсальная STT-модель с поддержкой 50+ языков:

```bash
STT_PROVIDER=openai_whisper
OPENAI_API_KEY=your_openai_key_here
WHISPER_MODEL=whisper-1
```

### Настройка TTS

Для голосовых ответов настройте TTS-провайдера. ElevenLabs создаёт наиболее естественно звучащий голос:

```bash
TTS_PROVIDER=elevenlabs
ELEVENLABS_API_KEY=your_elevenlabs_key_here
ELEVENLABS_VOICE_ID=21m00Tcm4TlvDq8ikWAM  # "Rachel" — чёткий, профессиональный голос
ELEVENLABS_MODEL=eleven_turbo_v2           # Модель с низкой задержкой для голоса в реальном времени
```

Или используйте OpenAI TTS для более простой конфигурации «всё в одном»:

```bash
TTS_PROVIDER=openai_tts
OPENAI_API_KEY=your_openai_key_here
OPENAI_TTS_VOICE=nova  # Варианты: alloy, echo, fable, onyx, nova, shimmer
OPENAI_TTS_MODEL=tts-1  # tts-1-hd для высшего качества (выше задержка)
```

## Шаг 2: Подключение телефонного моста через Twilio

Twilio использует вебхуки для переадресации входящих звонков на ваш сервер. Плагин Twilio Voice для OpenClaw обрабатывает это автоматически.

### Установка плагина Twilio Voice

В Dashboard OpenClaw:
1. Перейдите в **Плагины → Обзор**
2. Найдите «Twilio Voice»
3. Нажмите **Установить** и введите учётные данные:

```bash
# Добавьте в .env:
TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
TWILIO_AUTH_TOKEN=your_auth_token_here
TWILIO_PHONE_NUMBER=+15551234567
```

### Настройка Twilio Webhook

В консоли Twilio:
1. Перейдите в **Phone Numbers → Active Numbers**
2. Нажмите на ваш номер
3. В разделе **Voice & Fax** укажите вебхук:

```
https://your-openclaw-domain.com/api/twilio/voice
```

Убедитесь, что ваш экземпляр OpenClaw доступен из интернета. Для локальной разработки используйте ngrok:

```bash
ngrok http 3000
# Скопируйте https URL и используйте его в Twilio
```

### Проверка подключения

Позвоните на свой номер Twilio. Если всё настроено правильно, OpenClaw ответит, обработает вашу речь через STT-провайдер, отправит текст в ИИ-модель и воспроизведёт ответ через TTS. Первый звонок может занять 2–3 секунды на холодный старт; последующие будут быстрее.

## Шаг 3: Настройка личности под голосовой формат

Текстовые и голосово-оптимизированные ИИ-промпты отличаются. Голосовые разговоры требуют коротких предложений, естественной разговорной речи и понимания специфики телефонного формата.

### Редактирование системного промпта для голоса

В Dashboard OpenClaw перейдите в **Настройки → Голосовой режим → Системный промпт** и настройте:

```
Ты голосовой ИИ-ассистент. Пользователь звонит тебе по телефону.

Правила голосовых ответов:
- Отвечай коротко. Цель — 2–4 предложения на ответ, если не просят детальный разбор.
- Не используй маркированные списки, Markdown или перечисления — говори естественными абзацами.
- Избегай фраз-заглушек типа «Конечно!», «Безусловно!» и т.п.
- Если не знаешь что-то — скажи об этом кратко и предложи другую помощь.
- На вопросы, требующие развёрнутого ответа, сначала изложи главное, затем предложи рассказать подробнее.
- Говори естественно, как с другом — а не зачитывай документ.

Тебя зовут [имя ассистента]. Ты полезен, прямолинеен и иногда остроумен.
```

### Советы по настройке личности

**Для профессионального ассистента** (рабочие задачи, расписание, почта):
- Добавьте: «Используй официальный, но дружелюбный тон. При обсуждении расписания и важных деталей точность важнее краткости.»

**Для неформального личного ассистента** (общие вопросы, напоминания, разговоры):
- Добавьте: «Говори непринуждённо. Предпочитай короткие ответы. Используй разговорный стиль.»

**Для многоязыковой поддержки**:
- Добавьте: «Определяй язык пользователя и отвечай на том же языке.»

## Результат: звонок своему ИИ с любого телефона

После настройки ощущения удивительно естественные:

1. **Вы звоните** на номер Twilio — с любого телефона в мире
2. **OpenClaw отвечает** и начинает запись
3. **Вы говорите** свой вопрос или команду
4. **Deepgram/Whisper в реальном времени транскрибирует** речь в текст
5. **ИИ-модель генерирует** ответ
6. **ElevenLabs/OpenAI TTS преобразует** текст в речь
7. **Вы слышите ответ** через телефон — как правило, через 2–4 секунды

Можно попросить что-то найти (с плагином Google Search), управлять календарём, отправлять сообщения или просто поговорить. Работает с любого телефона по всему миру — звонящему не нужно устанавливать приложение или создавать аккаунт.

### Примеры разговоров

**Расписание**: «Что у меня завтра?» → считывает Google Calendar и зачитывает сводку.

**Информация**: «Какая погода в Москве на выходных?» → ищет и даёт голосовой прогноз.

**Действие**: «Напиши жене в WhatsApp, что буду дома к семи» → отправляет сообщение и устно подтверждает.
