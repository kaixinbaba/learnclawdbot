# Clawdia Phone Bridge - Создайте голосового ИИ-ассистента с Vapi и OpenClaw

Хотели когда-нибудь позвонить своему ИИ-ассистенту? С **Clawdia Phone Bridge** это возможно. Этот проект создаёт мост в реальном времени между голосовым ИИ Vapi и OpenClaw, позволяя совершать почти мгновенные телефонные разговоры с вашим ИИ.

## Что такое Clawdia Phone Bridge?

[Clawdia Phone Bridge](https://github.com/alejandroOPI/clawdia-bridge) — это HTTP-мост, соединяющий Vapi (платформу голосового ассистента) с OpenClaw. Он позволяет:

- Звонить вашему ИИ-ассистенту
- Получать ответы голосом в реальном времени
- Получать доступ ко всем навыкам OpenClaw (календарь, почта, погода и т.д.) через голос

## Как это работает

Архитектура проста:

1. **Вы** совершаете телефонный звонок
2. **Vapi** захватывает ваш голос и отправляет на мост
3. **Clawdia Bridge** пересылает запрос в OpenClaw через WebSocket
4. **OpenClaw** обрабатывает запрос с помощью ИИ-агента
5. **Ответ** возвращается через мост к Vapi
6. **Vapi** озвучивает ответ вам

```
Вы (телефонный звонок)
    ↓
Clawdia (Голосовой ИИ Vapi)
    ↓ POST /ask (вызов инструмента)
Clawdia Bridge
    ↓ WebSocket к Gateway
Clawdius (обрабатывает запрос)
    ↓ возвращает ответ
Clawdia Bridge
    ↓ возвращает Vapi
Clawdia
    ↓ говорит ответ
Вы
```

## Быстрый старт

### Предварительные требования

- Установленный Node.js
- Аккаунт Vapi
- Запущенный OpenClaw Gateway

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/alejandroOPI/clawdia-bridge.git
cd clawdia-bridge

# Установить зависимости
npm install

# Запустить мост
npm start
```

### Переменные окружения

| Переменная | По умолчанию | Описание |
|------------|--------------|----------|
| BRIDGE_PORT | 3847 | Порт для прослушивания |
| GATEWAY_URL | ws://127.0.0.1:18789 | WebSocket URL OpenClaw Gateway |

### Настройка Vapi

1. **Создать ассистента**: В панели Vapi создайте ассистента "Clawdia"
2. **Настроить голос**: Выберите женский голос (например, Lily от Vapi)
3. **Добавить инструмент**: Добавьте инструмент функции `ask_clawdius`
4. **Назначить номер**: Подключите номер телефона Vapi

### Публикация в интернет

Для продакшена:

```bash
# Использовать Tailscale Funnel (рекомендуется)
npm start
tailscale funnel 3847

# Или использовать ngrok
npm start
ngrok http 3844
```

## API эндпоинты

### POST /ask

Основной эндпоинт, который Vapi вызывает для связи с OpenClaw:

```json
{
  "question": "Какая сегодня погода?"
}
```

Ответ:

```json
{
  "answer": "Сейчас ясно, температура 22°C."
}
```

### GET /health

Эндпоинт проверки здоровья:

```json
{
  "status": "ok",
  "mode": "gateway-ws"
}
```

## Почему это важно

Этот мост открывает невероятные возможности:

- **Голосовые рабочие процессы**: Взаимодействуйте с ИИ без рук
- **Телефонные ИИ-ассистенты**: Создавайте ИИ-ассистентов, доступных по обычному телефону
- **Доступность**: Сделайте помощь ИИ доступной для нетехнических пользователей
- **Бизнес-приложения**: Клиентская поддержка, запись на приём, поиск информации

## Варианты использования

- **Персональный ИИ-ассистент**: Звоните ИИ для проверки календаря, погоды или отправки писем
- **Бизнес-линия**: Клиентская поддержка на основе ИИ по телефону
- **Помощь пожилым**: Простой голосовой интерфейс для доступа к ИИ
- **Производительность без рук**: Выполняйте задачи за рулём или готовкой

## Заключение

Clawdia Phone Bridge демонстрирует мощное сочетание голосового ИИ и возможностей агента OpenClaw. Объединяя Vapi и OpenClaw через мост, можно создавать сложных голосовых ИИ-ассистентов, использующих все навыки и интеграции OpenClaw.

Готовы создать своего голосового ИИ? Полная документация доступна в [GitHub репозитории](https://github.com/alejandroOPI/clawdia-bridge).
