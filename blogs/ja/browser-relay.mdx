# OpenClaw Browser Relay - インストールと使用ガイド

OpenClaw Browser Relayは、ブラウザとOpenClaw AI間のシームレスな通信を可能にする強力なChrome拡張機能です。この拡張機能により、OpenClawにWebページとのやり取り、反復的なタスクの自動化、複雑なブラウザベースの操作の実行を許可できます。

## Browser Relayとは？

Browser Relayは、ChromeブラウザとOpenClaw AI間の橋渡し役として機能します。AIが以下のことを可能にする「リモコン」と考えてください：

- **Webページのナビゲーション** - URLを開く、リンクをクリック、Webサイトを閲覧
- **フォームの入力** - Webフォームにデータを自動的に入力
- **情報の抽出** - Webページからコンテンツを読み取り、分析
- **要素との対話** - ボタンをクリック、オプションを選択、ページ要素を操作
- **スクリーンショット** - Webサイトから視覚情報をキャプチャ
- **自動化の実行** - 異なるWebサイト間で複数ステップのワークフローを実行

この拡張機能は安全で制御された方法で動作します - 完全なコントロールを維持し、AIが行うすべてをリアルタイムで見ることができます。

## インストール手順

Browser Relay拡張機能のインストールには、解凍された拡張機能として手動で読み込む必要があります。手順は以下の通りです：

### ステップ1：拡張機能ファイルの場所を確認

Browser Relay拡張機能はOpenClawにバンドルされています。以下の場所で拡張機能ファイルを見つけてください：

```
~/.openclaw/extensions/browser-relay/
```

このディレクトリが表示されない場合は、最新バージョンのOpenClawがインストールされていることを確認してください。

### ステップ2：Chrome拡張機能を開く

1. Google Chromeを開く
2. アドレスバーに`chrome://extensions/`と入力して移動
3. または、3点メニューをクリック → **その他のツール** → **拡張機能**

### ステップ3：デベロッパーモードを有効にする

拡張機能ページの右上隅で、**デベロッパーモード**をオンに切り替えます。

### ステップ4：拡張機能を読み込む

1. デベロッパーモードを有効にした後に表示される**パッケージ化されていない拡張機能を読み込む**ボタンをクリック
2. `~/.openclaw/extensions/browser-relay/`ディレクトリに移動
3. フォルダを選択し、**フォルダの選択**（または一部のシステムでは**開く**）をクリック

### ステップ5：インストールの確認

読み込み後、拡張機能リストにOpenClaw Browser Relay拡張機能が表示されるはずです。また、Chromeツールバー（通常はアドレスバーの近く）に新しいアイコンが表示されます。

## Browser Relayの使用方法

インストールが完了したら、Browser Relayの使用は簡単です：

### タブの接続

OpenClawに特定のブラウザタブを制御させるには：

1. **OpenClawとやり取りさせたいページに移動**
2. **ChromeツールバーのBrowser Relayアイコンをクリック**
3. 拡張機能アイコンが変化し、現在のタブに**接続**されていることを示します
4. バッジまたはインジケーター（通常は「ON」）が接続を確認します

**重要：** OpenClawは明示的に接続されたタブのみを制御できます。このセキュリティ対策により、AIがアクセスできるタブを常に把握できます。

### タブの切断

OpenClawをタブから切断するには：

1. Browser Relayアイコンを再度クリック
2. バッジがオフになり、タブが接続されていないことを示します

### 複数のタブでの作業

異なる時間に異なるタブを接続できますが、通常は一度に1つのタブのみが制御されます。OpenClawにブラウザアクションの実行を依頼すると、現在接続されているタブが使用されます。

## 一般的な使用例

OpenClawとBrowser Relayを活用する実用的な方法をいくつか紹介します：

### 1. Web調査とデータ抽出

**例：** 「このeコマースページからすべての商品価格を抽出して比較表を作成してください。」

OpenClawは商品リストをナビゲートし、構造化データを抽出し、整理された形式で提示できます。

### 2. フォーム入力と送信

**例：** 「私の情報でこの連絡フォームに記入して送信してください。」

反復的なフォーム送信、調査、または登録プロセスに最適です。

### 3. 自動テスト

**例：** 「これらの資格情報を使用してこのWebサイトのログインフローをテストしてください。」

QAエンジニアは、OpenClawを使用してブラウザベースのテストシナリオを自動化できます。

### 4. コンテンツ監視

**例：** 「このページを1時間ごとにチェックして、『在庫切れ』ステータスが変更されたら通知してください。」

特定の変更や更新についてWebサイトを自動的に監視します。

### 5. 複数ステップのワークフロー

**例：** 「これら3つのWebサイトでニューヨークからロンドンへのフライトを検索し、最も安いオプションを比較してください。」

複数のアクションを連鎖させて、異なるサイト間で複雑なタスクを完了します。

### 6. スクリーンショットとドキュメント

**例：** 「このチュートリアルの各ステップのスクリーンショットを撮ってフォルダに保存してください。」

プロセスを体系的に文書化したり、視覚情報をキャプチャしたりします。

## ヒントとベストプラクティス

### セキュリティに関する考慮事項

- **信頼できるタブのみを接続** - OpenClawにアクセス権を与えるサイトには注意してください
- **アクションを確認** - 特に機密情報を扱うサイトでOpenClawが行うことを監視してください
- **完了したら切断** - OpenClawで使用しなくなったタブを切断してください

### パフォーマンスのヒント

- **不要なタブを閉じる** - Chromeが過負荷でない場合、Browser Relayは最適に機能します
- **具体的な指示を使用** - 明確なコマンドは、OpenClawがより正確にアクションを実行するのに役立ちます
- **ページの読み込みを待つ** - 複雑なページは、やり取りする前に完全に読み込まれるまで時間が必要な場合があります

### トラブルシューティング

**拡張機能アイコンが表示されない？**
- デベロッパーモードが有効になっていることを確認してください
- `chrome://extensions/`から拡張機能を再読み込みしてみてください

**タブを接続できない？**
- ページを更新して再試行してください
- 一部の保護されたページ（`chrome://` URLなど）は制御できません

**アクションが機能しない？**
- タブがまだ接続されていることを確認してください（「ON」バッジを確認）
- OpenClawが実行中で接続されていることを確認してください

## OpenClawでのプロファイル選択

OpenClawでブラウザコマンドを使用する場合、使用するブラウザプロファイルを指定する必要があります：

- **`profile="chrome"`** - Browser Relay拡張機能を使用して既存のChromeタブを引き継ぎます
- **`profile="openclaw"`** - OpenClawの隔離された管理されたブラウザインスタンスを使用します

Browser Relayの場合は、常に`profile="chrome"`を使用してください：

```
profile="chrome"のブラウザを使用して現在のタブでフォームに記入してください。
```
