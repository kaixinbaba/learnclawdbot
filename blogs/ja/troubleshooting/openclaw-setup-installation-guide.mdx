# 完全ガイド: Windows、macOS、Linuxで OpenClaw をセットアップする

システムで OpenClaw を実行することは簡単なはずですが、異なるオペレーティングシステムには独自の癖があります。このガイドでは、**Windows**、**macOS**、**Linux** でのインストールを順を追って説明し、一般的な落とし穴と修正方法をカバーします。

## 前提条件: 最初に必要なもの

OpenClaw をインストールする前に、次のものがあることを確認してください:

### 1. Node.js (v18 以上)

OpenClaw には最新の Node.js ランタイムが必要です。バージョンを確認:

```bash
node --version
```

Node.js がないか、古いバージョンの場合はインストール:

**macOS (Homebrew を使用):**
```bash
brew install node
```

**Windows (winget を使用):**
```powershell
winget install OpenJS.NodeJS
```

**Linux (Ubuntu/Debian):**
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

**Linux (Fedora/RHEL):**
```bash
sudo dnf install nodejs
```

### 2. npm または pnpm

npm は Node.js と一緒にインストールされますが、pnpm の方が高速で効率的です:

```bash
npm install -g pnpm
```

## OpenClaw のインストール

### ステップ 1: npm 経由でインストール

```bash
npm install -g @anthropic-ai/openclaw
```

または pnpm を使用:

```bash
pnpm add -g @anthropic-ai/openclaw
```

### ステップ 2: インストールの確認

```bash
openclaw --version
```

バージョン番号(例: `0.5.2`)が表示されるはずです。

## プラットフォーム固有のセットアップ

### 🪟 Windows

#### 問題 #1: PowerShell 実行ポリシー

**症状:** `openclaw: ファイル...を読み込めません。このシステムではスクリプトの実行が無効になっています`

**修正:**
```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

#### 問題 #2: 長いパスのサポート

Windows には 260 文字のパス制限があり、npm インストールが壊れる可能性があります。

**修正:** レジストリで長いパスを有効化
```powershell
New-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Control\FileSystem" -Name "LongPathsEnabled" -Value 1 -PropertyType DWORD -Force
```

または WSL2 (Windows Subsystem for Linux) を使用:
```powershell
wsl --install
```

次に WSL 内で OpenClaw をインストール:
```bash
# WSL 内で
npm install -g @anthropic-ai/openclaw
```

#### 問題 #3: Node-gyp コンパイルエラー

**症状:** インストール中にビルドツールがないというエラー

**修正:** Windows ビルドツールをインストール
```powershell
npm install -g windows-build-tools
```

または Microsoft から Visual Studio ビルドツールを手動でインストール。

### 🍎 macOS

#### 問題 #1: アクセス拒否

**症状:** `npm install -g` 実行時に `EACCES: permission denied`

**修正:** npm で `sudo` を使用しないでください。代わりに、ユーザーディレクトリを使用するよう npm を設定:

```bash
mkdir -p ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.zshrc
source ~/.zshrc
```

次に再インストール:
```bash
npm install -g @anthropic-ai/openclaw
```

#### 問題 #2: インストール後にコマンドが見つからない

**症状:** インストール成功後も `openclaw: command not found`

**修正:** npm グローバル bin が PATH にあるか確認:
```bash
echo $PATH | grep npm
```

ない場合は、`~/.zshrc` に追加:
```bash
export PATH="$(npm config get prefix)/bin:$PATH"
```

#### 問題 #3: Rosetta が必要 (M1/M2 Mac)

一部の依存関係は Apple Silicon で Rosetta 2 が必要な場合があります:

```bash
softwareupdate --install-rosetta
```

### 🐧 Linux

#### 問題 #1: グローバルインストール時の EACCES

**症状:** macOS と同様の権限エラー

**修正:** 同じ解決策 - ユーザーディレクトリを使用:
```bash
mkdir -p ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
npm install -g @anthropic-ai/openclaw
```

#### 問題 #2: 依存関係の欠落 (Ubuntu/Debian)

**症状:** ネイティブモジュールのビルドエラー

**修正:**
```bash
sudo apt-get update
sudo apt-get install -y build-essential python3
```

#### 問題 #3: Snap Node.js の問題

Snap 経由で Node.js をインストールした場合、権限の問題が発生する可能性があります。

**修正:** Snap バージョンをアンインストールし、公式リポジトリを使用:
```bash
sudo snap remove node
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

## 初期設定

インストール後、OpenClaw を設定:

### 1. API キーの設定

```bash
openclaw configure
```

プロンプトに従って Anthropic API キーを追加。

### 2. Gateway の起動

```bash
openclaw gateway start
```

### 3. セットアップのテスト

```bash
openclaw chat "こんにちは、OpenClaw!"
```

応答が表示されたら、おめでとうございます! 🎉

## インストール後の一般的な問題

### Gateway が起動しない

**ログを確認:**
```bash
tail -f ~/.openclaw/logs/gateway.log
```

**一般的な修正:**
- ポート 3000 が既に使用中 → 設定でポートを変更
- 設定ファイルがない → `openclaw configure` を再実行
- ファイアウォールがブロック → ファイアウォール設定で Node.js を許可

### バージョンが古い

OpenClaw を更新:
```bash
npm update -g @anthropic-ai/openclaw
```

または pnpm で:
```bash
pnpm update -g @anthropic-ai/openclaw
```

### 完全な再インストール

すべてが失敗した場合:
```bash
# アンインストール
npm uninstall -g @anthropic-ai/openclaw

# キャッシュをクリア
npm cache clean --force

# 再インストール
npm install -g @anthropic-ai/openclaw
```
