# Moltbot 用語集

このガイドでは、Moltbot を使用する際に出会う重要な用語と概念について説明します。

## コアコンセプト

### Agent (エージェント)
タスクを実行し、決定を下し、さまざまなツールや環境と対話できる自律型 AI エンティティ。

### Skill (スキル)
Moltbot に追加できる特定の能力や機能。スキルは、新しい API へのアクセス、特定の計算の実行、他のサービスとの統合など、Moltbot の能力を拡張します。

### Workspace (ワークスペース)
Moltbot が実行される環境。ワークスペースには、ファイル、設定、コンテキストが含まれます。

### Task (タスク)
Moltbot に割り当てられた特定の作業単位。タスクは単純なもの（例：「ファイルを作成する」）から複雑なもの（例：「Web サイトを構築する」）まであります。

## 上級用語

### Context Window (コンテキストウィンドウ)
Moltbot がリクエストを処理する際に一度に考慮できる情報量（テキスト、コード、履歴）。

### Planner (プランナー)
高レベルの目標を実行可能なステップに分解する Moltbot のコンポーネント。

## コミュニティ用語

### ClawdHub
Moltbot のスキルや拡張機能を閲覧・インストールするための公式マーケットプレイス。
