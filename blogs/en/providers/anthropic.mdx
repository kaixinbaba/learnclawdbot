## Configuration Methods

Run `openclaw onboard` and select Anthropic. You'll see two authentication methods:

<img src="/images/blog/providers/anthropic/auth-method-select.png" alt="Select Anthropic authentication method" style="max-width: 100%" />

### Method 1: Setup Token (Recommended for Claude Subscribers)

If you have a Claude Pro/Max subscription, you can directly use `claude setup-token` to generate a long-term token.

**1. Run in terminal:**

```bash
claude setup-token
```

The system will generate an OAuth token valid for 1 year:

<img src="/images/blog/providers/anthropic/setup-token-generated.png" alt="Generate setup-token" style="max-width: 100%" />

**2. Copy the token and paste it into OpenClaw:**

<img src="/images/blog/providers/anthropic/paste-setup-token.png" alt="Paste setup-token into OpenClaw" style="max-width: 100%" />

Done!

### Method 2: API Key (Recommended for Pay-as-you-go API Users)

If you use Anthropic API with pay-as-you-go billing, you need to create an API Key.

**1. Visit [Anthropic Console](https://console.anthropic.com/), go to API keys page, and click Create Key:**

<img src="/images/blog/providers/anthropic/console-api-keys.png" alt="Anthropic Console API Keys" style="max-width: 100%" />

**2. Enter Key name and click Add:**

<img src="/images/blog/providers/anthropic/create-api-key-dialog.png" alt="Create API Key" style="max-width: 70%" />

**3. Copy the generated API Key (shown only once!):**

<img src="/images/blog/providers/anthropic/save-api-key.png" alt="Save API Key" style="max-width: 70%" />

**4. In `openclaw onboard`, select Anthropic API key and paste the Key:**

<img src="/images/blog/providers/anthropic/paste-api-key.png" alt="Paste API Key into OpenClaw" style="max-width: 100%" />
