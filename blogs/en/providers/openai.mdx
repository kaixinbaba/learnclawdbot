## Configuration Methods

### Method 1: OAuth One-Click Login (Recommended)

Run `openclaw onboard` and select OpenAI:

<img src="/images/blog/providers/openai/oauth-select.png" alt="Select OpenAI" style="max-width: 60%" />

Follow the prompts to complete OAuth login in the opened browser. After login completes, copy the URL from the browser address bar:

<img src="/images/blog/providers/openai/oauth-copy-url.png" alt="Copy browser address" style="max-width: 100%" />

Paste it into the terminal:

<img src="/images/blog/providers/openai/oauth-paste-terminal.png" alt="Paste into terminal" style="max-width: 100%" />

Select the model you want to use (recommend `gpt-5.2-codex`):

<img src="/images/blog/providers/openai/model-select.png" alt="Select model" style="max-width: 100%" />


### Method 2: API Key Configuration

1. Visit [OpenAI Platform](https://platform.openai.com/)

<img src="/images/blog/providers/openai/api-platform.png" alt="OpenAI Platform" style="max-width: 100%" />

2. Create API Key

<img src="/images/blog/providers/openai/api-create-key.png" alt="Create API Key" style="max-width: 70%" />

Remember to save it properly (shown only once!):

<img src="/images/blog/providers/openai/api-save-key.png" alt="Save API Key" style="max-width: 70%" />

3. In `openclaw onboard`, select OpenAI and paste the API Key:

<img src="/images/blog/providers/openai/api-paste-openclaw.png" alt="Paste API Key into OpenClaw" style="max-width: 60%" />
