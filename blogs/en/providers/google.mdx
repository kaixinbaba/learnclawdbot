Google Gemini is a multimodal AI model that supports text, image, video, and audio input, with generous free quotas. This tutorial shows you how to configure and use Gemini in OpenClaw.

## Why Choose Gemini?

- **Free Quota** ‚Äî Google AI Studio provides free API call quotas
- **Native Multimodal** ‚Äî Supports image, video, and audio understanding
- **Long Context** ‚Äî Gemini 1.5 Pro supports 1M token context

## Configuration Methods

Run `openclaw onboard` and select Google. You'll see three authentication methods:

<img src="/images/blog/providers/google/auth-method-select.png" alt="Select Google authentication method" style="max-width: 60%" />

### Method 1: API Key (Recommended, Simplest)

**1. Open [Google AI Studio](https://aistudio.google.com/apikey) and click "Create API key":**

<img src="/images/blog/providers/google/ai-studio-api-keys.png" alt="Google AI Studio API keys page" style="max-width: 100%" />

**2. Enter key name, select project, and click "Create key":**

<img src="/images/blog/providers/google/create-key-dialog.png" alt="Create API key" style="max-width: 80%" />

**3. After creation succeeds, click the copy button to copy the API Key:**

<img src="/images/blog/providers/google/copy-api-key.png" alt="Copy API Key" style="max-width: 100%" />

**4. Paste the API Key in OpenClaw:**

<img src="/images/blog/providers/google/paste-api-key.png" alt="Paste API Key into OpenClaw" style="max-width: 100%" />

**5. Select default model (recommend `gemini-3-pro-high`):**

<img src="/images/blog/providers/google/model-select.png" alt="Select Gemini model" style="max-width: 100%" />

Done!

### Method 2: Antigravity OAuth

After selecting "Google Antigravity OAuth", a browser will open for Google account authorization. After successful authorization, you'll see:

<img src="/images/blog/providers/google/antigravity-oauth-complete.png" alt="Antigravity OAuth complete" style="max-width: 100%" />

Return to the terminal and continue selecting the model.

### Method 3: Gemini CLI OAuth (Advanced)

Selecting "Google Gemini CLI OAuth" requires installing Gemini CLI first and configuring a Google Cloud project. After successful authorization, you'll see:

<img src="/images/blog/providers/google/gemini-cli-oauth-complete.png" alt="Gemini CLI OAuth complete" style="max-width: 100%" />

> ‚ö†Ô∏è This method requires setting the `GOOGLE_CLOUD_PROJECT` environment variable. Configuration is complex; recommend using API Key method first.
