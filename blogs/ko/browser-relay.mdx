# OpenClaw Browser Relay - 설치 및 사용 가이드

OpenClaw Browser Relay는 브라우저와 OpenClaw AI 간의 원활한 통신을 가능하게 하는 강력한 Chrome 확장 프로그램입니다. 이 확장 프로그램을 통해 OpenClaw가 웹 페이지와 상호 작용하고, 반복적인 작업을 자동화하며, 대신 복잡한 브라우저 기반 작업을 수행하도록 허용할 수 있습니다.

## Browser Relay란 무엇인가요?

Browser Relay는 Chrome 브라우저와 OpenClaw AI 간의 다리 역할을 합니다. AI가 다음을 수행할 수 있도록 하는 "리모컨"이라고 생각하세요:

- **웹 페이지 탐색** - URL 열기, 링크 클릭 및 웹사이트 탐색
- **양식 작성** - 웹 양식에 데이터를 자동으로 입력
- **정보 추출** - 웹 페이지에서 콘텐츠를 읽고 분석
- **요소와 상호 작용** - 버튼 클릭, 옵션 선택 및 페이지 요소 조작
- **스크린샷 촬영** - 웹사이트에서 시각적 정보 캡처
- **자동화 실행** - 여러 웹사이트에 걸쳐 다단계 워크플로우 수행

이 확장 프로그램은 안전하고 통제된 방식으로 작동합니다 - 전체 제어권을 유지하고 AI가 수행하는 모든 작업을 실시간으로 볼 수 있습니다.

## 설치 단계

Browser Relay 확장 프로그램을 설치하려면 압축 해제된 확장 프로그램으로 수동으로 로드해야 합니다. 방법은 다음과 같습니다:

### 1단계: 확장 프로그램 파일 찾기

Browser Relay 확장 프로그램은 OpenClaw와 함께 번들로 제공됩니다. 다음 위치에서 확장 프로그램 파일을 찾으세요:

```
~/.openclaw/extensions/browser-relay/
```

이 디렉토리가 보이지 않으면 최신 버전의 OpenClaw가 설치되어 있는지 확인하세요.

### 2단계: Chrome 확장 프로그램 열기

1. Google Chrome 열기
2. 주소 표시줄에 `chrome://extensions/` 입력하여 이동
3. 또는 점 세 개 메뉴 클릭 → **도구 더보기** → **확장 프로그램**

### 3단계: 개발자 모드 활성화

확장 프로그램 페이지의 오른쪽 상단 모서리에서 **개발자 모드**를 켭니다.

### 4단계: 확장 프로그램 로드

1. 개발자 모드를 활성화한 후 나타나는 **압축해제된 확장 프로그램을 로드합니다** 버튼 클릭
2. `~/.openclaw/extensions/browser-relay/` 디렉토리로 이동
3. 폴더를 선택하고 **폴더 선택**(또는 일부 시스템에서는 **열기**) 클릭

### 5단계: 설치 확인

로드 후 확장 프로그램 목록에 OpenClaw Browser Relay 확장 프로그램이 표시되어야 합니다. Chrome 도구 모음(보통 주소 표시줄 근처)에 새 아이콘도 나타납니다.

## Browser Relay 사용 방법

설치가 완료되면 Browser Relay 사용은 간단합니다:

### 탭 연결

OpenClaw가 특정 브라우저 탭을 제어하도록 허용하려면:

1. **OpenClaw가 상호 작용하기를 원하는 페이지로 이동**
2. **Chrome 도구 모음에서 Browser Relay 아이콘 클릭**
3. 확장 프로그램 아이콘이 변경되어 현재 탭에 **연결**되었음을 표시합니다
4. 배지 또는 표시기(보통 "ON")가 연결을 확인합니다

**중요:** OpenClaw는 명시적으로 연결된 탭만 제어할 수 있습니다. 이 보안 조치는 AI가 액세스할 수 있는 탭을 항상 알 수 있도록 보장합니다.

### 탭 분리

탭에서 OpenClaw를 분리하려면:

1. Browser Relay 아이콘을 다시 클릭
2. 배지가 꺼져 탭이 더 이상 연결되지 않았음을 나타냅니다

### 여러 탭 작업

다른 시간에 다른 탭을 연결할 수 있지만 일반적으로 한 번에 하나의 탭만 제어됩니다. OpenClaw에게 브라우저 작업 수행을 요청하면 현재 연결된 탭을 사용합니다.

## 일반적인 사용 사례

OpenClaw와 Browser Relay를 활용하는 실용적인 방법은 다음과 같습니다:

### 1. 웹 연구 및 데이터 추출

**예:** "이 전자상거래 페이지에서 모든 제품 가격을 추출하고 비교 표를 만드세요."

OpenClaw는 제품 목록을 탐색하고, 구조화된 데이터를 추출하며, 정리된 형식으로 제시할 수 있습니다.

### 2. 양식 작성 및 제출

**예:** "내 정보로 이 연락처 양식을 작성하고 제출하세요."

반복적인 양식 제출, 설문조사 또는 등록 프로세스에 완벽합니다.

### 3. 자동화된 테스트

**예:** "이러한 자격 증명을 사용하여 이 웹사이트의 로그인 흐름을 테스트하세요."

QA 엔지니어는 OpenClaw를 사용하여 브라우저 기반 테스트 시나리오를 자동화할 수 있습니다.

### 4. 콘텐츠 모니터링

**예:** "이 페이지를 매시간 확인하고 '품절' 상태가 변경되면 알려주세요."

특정 변경 사항이나 업데이트에 대해 웹사이트를 자동으로 모니터링합니다.

### 5. 다단계 워크플로우

**예:** "이 세 웹사이트에서 뉴욕에서 런던으로 가는 항공편을 검색하고 가장 저렴한 옵션을 비교하세요."

여러 작업을 연결하여 다양한 사이트에서 복잡한 작업을 완료합니다.

### 6. 스크린샷 및 문서화

**예:** "이 튜토리얼의 각 단계 스크린샷을 찍어 폴더에 저장하세요."

프로세스를 체계적으로 문서화하거나 시각적 정보를 캡처합니다.

## 팁과 모범 사례

### 보안 고려 사항

- **신뢰할 수 있는 탭만 연결** - OpenClaw에 액세스 권한을 부여하는 사이트에 주의하세요
- **작업 검토** - 특히 민감한 정보를 처리하는 사이트에서 OpenClaw가 수행하는 작업을 관찰하세요
- **완료 후 분리** - OpenClaw와 더 이상 사용하지 않는 탭을 분리하세요

### 성능 팁

- **불필요한 탭 닫기** - Chrome이 과부하되지 않을 때 Browser Relay가 가장 잘 작동합니다
- **구체적인 지침 사용** - 명확한 명령은 OpenClaw가 더 정확하게 작업을 수행하는 데 도움이 됩니다
- **페이지 로드 대기** - 복잡한 페이지는 상호 작용 전에 완전히 로드되는 데 시간이 필요할 수 있습니다

### 문제 해결

**확장 프로그램 아이콘이 나타나지 않나요?**
- 개발자 모드가 활성화되어 있는지 확인하세요
- `chrome://extensions/`에서 확장 프로그램을 다시 로드해 보세요

**탭을 연결할 수 없나요?**
- 페이지를 새로 고침하고 다시 시도하세요
- 일부 보호된 페이지(`chrome://` URL 등)는 제어할 수 없습니다

**작업이 작동하지 않나요?**
- 탭이 여전히 연결되어 있는지 확인하세요("ON" 배지 확인)
- OpenClaw가 실행 중이고 연결되어 있는지 확인하세요

## OpenClaw의 프로필 선택

OpenClaw에서 브라우저 명령을 사용할 때 사용할 브라우저 프로필을 지정해야 합니다:

- **`profile="chrome"`** - Browser Relay 확장 프로그램을 사용하여 기존 Chrome 탭을 인수합니다
- **`profile="openclaw"`** - OpenClaw의 격리된 관리형 브라우저 인스턴스를 사용합니다

Browser Relay의 경우 항상 `profile="chrome"`을 사용하세요:

```
profile="chrome"인 브라우저를 사용하여 현재 탭의 양식을 작성하세요.
```
