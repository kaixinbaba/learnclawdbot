한 줄의 명령어로 설치하고, 한 줄로 설정하면 바로 사용할 수 있습니다.

OpenClaw(구 ClawdBot / MoltBot)는 오픈소스 AI 어시스턴트 프레임워크로, Telegram, WhatsApp, Discord 등 여러 채팅 플랫폼에 연결하여 일상적인 작업을 처리할 수 있습니다. 이 튜토리얼은 처음부터 시작하여 10분 만에 실행하는 방법을 안내합니다.

## 사전 요구사항

- **Node.js 22+**(필수, [설치 가이드](https://nodejs.org/))
- **macOS / Linux / Windows(WSL2)**

> Windows 사용자는 반드시 [WSL2](https://learn.microsoft.com/ko-kr/windows/wsl/install)를 사용해야 하며, 네이티브 Windows 환경은 지원하지 않습니다.

## 1단계: OpenClaw 설치

[터미널](/blog/how-to-open-terminal)을 엽니다(터미널이 무엇인지 모르시나요? [여기를 클릭하세요](/blog/how-to-open-terminal)). 다음 명령어를 실행하세요:

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

스크립트가 자동으로 시스템 환경을 감지하고, 의존성을 설치하며, 권한을 설정합니다. 전체 과정에서 수동 개입이 필요하지 않습니다.

설치가 완료되면 확인해보세요:

```bash
openclaw --version
```

> 📖 다른 설치 방법(npm 글로벌 설치, 소스 코드 설치 등)은 [공식 설치 문서](https://docs.openclaw.ai/install#quick-install-recommended)를 참조하세요.

## 2단계: 설정 마법사 실행

```bash
openclaw onboard --install-daemon
```

보안 위험에 동의합니다(동의하지 않으면 종료됩니다). QuickStart를 선택하는 것을 권장합니다.
<img src="/images/blog/install/agree.webp" alt="보안 위험 동의" style="max-width: 75%" />

마법사가 필요한 모든 설정을 안내합니다:

**1. 모델 인증**

AI 모델 제공업체를 선택하고 로그인합니다:

**국제 모델:**
[OpenAI](/blog/providers/openai) ·
[Anthropic (Claude)](/blog/providers/anthropic) ·
[Google (Gemini)](/blog/providers/google) ·

**로컬 모델:** [Ollama](/blog/openclaw-ollama)(무료, 완전 오프라인)

<img src="/images/blog/install/onboard-model-auth.png" alt="OpenClaw 모델 인증 선택"  style="max-width: 45%" />

**2. 채널 연결(선택사항)**

마법사가 채팅 플랫폼에 즉시 연결할지 물어봅니다. 지금 연결하거나 나중에 설정할 수 있습니다:

- **Telegram** — Bot Token 입력([@BotFather](https://t.me/botfather)에서 받기)
- **WhatsApp** — QR 코드 스캔
- **Discord** — Bot Token 입력

> 📖 각 플랫폼의 상세 설정 가이드: [Telegram](/blog/integrations/telegram) · [WhatsApp](/blog/integrations/whatsapp) · [Discord](/blog/integrations/discord)

## 3단계: 대화 시작

설정이 완료되면 OpenClaw와 상호작용하는 두 가지 방법이 있습니다:

### 방법 1: 터미널 TUI

```bash
openclaw tui
```

터미널에서 직접 AI와 대화합니다. 개발자와 명령줄 사용자에게 적합합니다.

<img src="/images/blog/install/tui-chat.png" alt="OpenClaw TUI 터미널 인터페이스" />

### 방법 2: Web Chat

브라우저에서 다음 주소를 여세요:

```
http://localhost:18789
```

그래픽 채팅 인터페이스로 더 친숙한 경험을 제공합니다.

<img src="/images/blog/install/webchat.webp" alt="OpenClaw WebChat 인터페이스" />

🎉 **완료!** 이제 실행 중인 AI 어시스턴트를 갖게 되었습니다.

## 자주 묻는 질문

### `openclaw` 명령어를 찾을 수 없나요?

npm 글로벌 경로가 `PATH`에 없습니다. 다음 명령어를 실행하여 수정하세요:

```bash
export PATH="$PATH:$(npm config get prefix)/bin"
source ~/.zshrc  # 또는 ~/.bashrc
```

또는 설치 스크립트를 다시 실행하면 자동으로 처리됩니다.

### Gateway에 액세스할 수 없나요?

포트가 사용 중인지 확인하세요:

```bash
openclaw gateway status
```
