# OpenClaw 용어집

이 가이드는 OpenClaw를 사용할 때 접하게 될 주요 용어와 개념을 설명합니다.

## 핵심 개념

### Agent
작업을 실행하고, 결정을 내리며, 다양한 도구 및 환경과 상호 작용할 수 있는 자율 AI 엔티티입니다.

### Skill
OpenClaw에 추가할 수 있는 특정 능력 또는 기능입니다. Skill은 새로운 API 액세스, 특정 계산 수행 또는 다른 서비스와의 통합과 같은 OpenClaw의 기능을 확장합니다.

### Workspace
OpenClaw가 실행되는 환경입니다. 워크스페이스에는 파일, 구성 및 컨텍스트가 포함됩니다.

### Task
OpenClaw에 할당된 특정 작업 단위입니다. 작업은 간단할 수도 있고(예: "파일 생성") 복잡할 수도 있습니다(예: "웹사이트 구축").

## 고급 용어

### Context Window
OpenClaw가 요청을 처리할 때 한 번에 고려할 수 있는 정보(텍스트, 코드, 기록)의 양입니다.

### Planner
상위 수준 목표를 실행 가능한 단계로 나누는 역할을 담당하는 OpenClaw의 구성 요소입니다.

## 커뮤니티 용어

### ClawdHub
OpenClaw 스킬 및 확장 프로그램을 탐색하고 설치할 수 있는 공식 마켓플레이스입니다.
