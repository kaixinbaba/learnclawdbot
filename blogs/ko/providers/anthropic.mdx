Anthropic의 Claude는 현재 가장 강력한 AI 모델 중 하나로, 안전성과 긴 컨텍스트 능력으로 유명합니다. 이 튜토리얼은 OpenClaw에서 Claude를 설정하고 사용하는 방법을 안내합니다.

## Claude를 선택하는 이유는?

- **긴 컨텍스트** — 200K token 컨텍스트 윈도우 지원
- **강력한 코드 능력** — Claude Sonnet/Opus는 프로그래밍 작업에서 뛰어난 성능을 발휘합니다
- **안전하고 신뢰할 수 있음** — Anthropic의 Constitutional AI 설계

## 설정 방법

`openclaw onboard`를 실행하고 Anthropic을 선택하면 두 가지 인증 방법이 표시됩니다:

<img src="/images/blog/providers/anthropic/auth-method-select.png" alt="Anthropic 인증 방법 선택" style="max-width: 100%" />

### 방법 1: Setup Token(Claude 구독자에게 권장)

Claude Pro/Max 구독이 있는 경우 `claude setup-token`을 사용하여 장기 token을 직접 생성할 수 있습니다.

**1. 터미널에서 실행:**

```bash
claude setup-token
```

시스템이 유효기간 1년의 OAuth token을 생성합니다:

<img src="/images/blog/providers/anthropic/setup-token-generated.png" alt="setup-token 생성" style="max-width: 100%" />

**2. token을 복사하여 OpenClaw에 붙여넣기:**

<img src="/images/blog/providers/anthropic/paste-setup-token.png" alt="OpenClaw에 setup-token 붙여넣기" style="max-width: 100%" />

완료!

### 방법 2: API Key(유료 API 사용자에게 권장)

Anthropic API를 종량제로 사용하는 경우 API Key를 생성해야 합니다.

**1. [Anthropic Console](https://console.anthropic.com/)에 방문하여 API keys 페이지로 이동하고 Create Key를 클릭합니다:**

<img src="/images/blog/providers/anthropic/console-api-keys.png" alt="Anthropic Console API Keys" style="max-width: 100%" />

**2. Key 이름을 입력하고 Add를 클릭합니다:**

<img src="/images/blog/providers/anthropic/create-api-key-dialog.png" alt="API Key 생성" style="max-width: 70%" />

**3. 생성된 API Key를 복사합니다(한 번만 표시됩니다!):**

<img src="/images/blog/providers/anthropic/save-api-key.png" alt="API Key 저장" style="max-width: 70%" />

**4. `openclaw onboard`에서 Anthropic API key를 선택하고 Key를 붙여넣습니다:**

<img src="/images/blog/providers/anthropic/paste-api-key.png" alt="OpenClaw에 API Key 붙여넣기" style="max-width: 100%" />

## 모델 선택

| 모델 | 특징 | 적용 시나리오 |
|------|------|----------|
| Claude Opus | 가장 강력하고 비쌈 | 복잡한 추론, 창의적 글쓰기, 고난도 작업 |
| Claude Sonnet | 균형 잡힌 가성비 | 일상 대화, 코드 생성, 대부분의 작업 |
| Claude Haiku | 가장 빠르고 저렴함 | 간단한 작업, 빠른 응답, 대량 호출 |

## 자주 묻는 질문

### setup-token과 API Key 중 어떤 것을 사용해야 하나요?

- **Claude Pro/Max 구독 보유** → setup-token 사용, 무료 할당량 내에서 추가 비용 없음
- **구독 없음, 종량제** → API Key 사용

### token이 만료되면 어떻게 하나요?

`claude setup-token`을 다시 실행하여 새 token을 생성한 후 OpenClaw에서 다시 설정합니다.
