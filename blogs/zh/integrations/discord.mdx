# Discord 集成是什么？（社区活跃气氛组）

> Discord 集成是一种【服务器插件】，用于【增强社区互动和自动化管理】，可以帮助【社区主和游戏玩家】在【多人聊天频道】下【创造更有趣、更智能的交流体验】。

---

## 一句话先懂

- **它解决的问题**：把 AI 引入多人聊天室，让它扮演角色、生成表情包或回答新人问题。
- **它的核心价值**：多模态互动（画图、语音）、角色扮演（Roleplay）、频道隔离。
- **典型使用场景**：Midjourney 式的画图频道、TRPG 跑团助手、服务器自动客服。

---

## 一、Discord 集成是什么？

Discord 是全球最受欢迎的社区聊天平台之一，在社交类应用中长期占据前列位置。

![Discord App Store](/images/integrations/discord/1.png)
*Discord 在 App Store 社交榜长期排名前列，拥有 334 万+ 评分*

Discord 集成本质上是**一个拥有特定权限的服务器成员（Bot）**。

你可以把它理解成**一个不知疲倦的版主兼陪聊**：
- 当只有它被允许的频道有人说话时
- 它会根据你设定的"人设"（比如一个傲娇的魔法师）来回复

---

## 二、它能做什么？

通常，Discord 集成可以用来：

1. **AI 绘图**（一句话）：输入 `/imagine` 加上描述，它直接在频道里生成高质量图片。
2. **角色扮演**：给它设定"赛博朋克酒保"的人设，让它在 `#bar` 频道陪大家聊天。
3. **语音伴侣**：加入语音频道，把大家说的话转成文字，或者参与语音聊天。

在实际产品中，常见的用法包括：
- **Mod 辅助**：检测恶意言论或垃圾广告，自动禁言或踢人（需高级权限）。
- **游戏攻略查询**：连接游戏 Wiki 知识库，玩家问"这个 BOSS 怎么打"，它秒回攻略。
- **每日话题生成**：每天早上自动在闲聊频道发一个有趣的讨论话题，防止冷群。

---

## 三、怎么用？（快速上手）

一般来说，使用 Discord 集成可以分为 3 步：

1. **第一步：邀请入驻**
   在 OpenClaw 后台点击 `Add to Discord`，选择你有管理权限的服务器。

2. **第二步：设置权限**
   Discord 会弹窗询问权限（如发送消息、读取历史），建议勾选最小必要权限，然后确认。

3. **第三步：频道配置**
   为了不打扰大家，建议先创建一个专用频道（如 `#ai-chat`），并在后台设置 Bot 只能在该频道回复。

如果你是第一次使用：
- **建议从【Slash 命令】开始**：输入 `/` 看看它有哪些专属命令（如 `/ask`, `/help`）。
- **不需要一开始就【全频道开启】**：否则你会发现它在每个人的对话下面都插嘴，非常烦人。

## 四、手把手教程：创建你自己的 Discord Bot

如果你想完全掌控自己的 Bot（自定义名称、头像、权限），跟着下面的步骤操作：

### 第 1 步：打开 Discord Developer Portal

访问 [Discord Developer Portal](https://discord.com/developers/applications)，登录你的 Discord 账号。

在 Applications 页面，点击右上角的 **New Application** 按钮。

![创建应用](/images/integrations/discord/2.png)
*点击 "New Application" 创建一个新的 Discord 应用*

### 第 2 步：创建应用

在弹出的对话框中：
1. 输入你的应用名称（比如 `MyAIBot`）
2. 勾选同意开发者条款
3. 点击 **Create** 按钮

![创建应用弹窗](/images/integrations/discord/3.png)
*填写应用名称并同意条款，点击 Create 创建*

### 第 3 步：进入 Bot 设置

创建成功后，在左侧菜单找到 **Bot** 选项并点击进入。

这里是你的 Bot 配置中心，可以设置：
- Bot 的用户名和头像
- 权限和 Gateway Intents
- 最重要的：**Bot Token**

![Bot 设置页面](/images/integrations/discord/4.png)
*左侧点击 "Bot"，然后点击 "Reset Token" 生成新的 Token*

### 第 4 步：获取 Bot Token

点击 **Reset Token** 按钮，Discord 会生成一个新的 Token。

**⚠️ 重要提示**：Token 只会显示一次！点击 **Copy** 按钮复制并妥善保存。

![复制 Token](/images/integrations/discord/5.png)
*点击 Copy 复制 Token，这是 Bot 的"密码"，请妥善保管*

> 🔐 **安全提示**：Bot Token 相当于你 Bot 的密码，绝对不要公开分享或上传到 GitHub！

### 第 5 步：开启必要的权限（Intents）

在同一个 Bot 页面，向下滚动找到 **Privileged Gateway Intents** 部分，开启：

- ✅ **Message Content Intent** — 必须开启，否则 Bot 读不到消息内容
- ✅ **Server Members Intent** — 推荐开启，用于成员查询

### 第 6 步：在 OpenClaw 中配置

最后一步，把 Token 配置到 OpenClaw。

运行配置向导或直接编辑配置文件，输入你的 Bot Token：

![配置 OpenClaw](/images/integrations/discord/6.png)
*在 OpenClaw 配置中输入你的 Discord Bot Token*

配置文件示例：

```json5
{
  channels: {
    discord: {
      enabled: true,
      token: "你的Bot Token"
    }
  }
}
```

或者使用环境变量：

```bash
export DISCORD_BOT_TOKEN="你的Bot Token"
```

### 第 7 步：邀请 Bot 到服务器

回到 Developer Portal，进入 **OAuth2** → **URL Generator**：

1. **Scopes** 勾选：`bot` 和 `applications.commands`
2. **Bot Permissions** 勾选：
   - View Channels
   - Send Messages
   - Read Message History
   - Embed Links
   - Add Reactions

复制生成的链接，在浏览器打开，选择你的服务器，完成邀请！

### 验证是否成功

1. 启动 OpenClaw：`openclaw gateway start`
2. 在你的 Discord 服务器 @你的Bot 发一条消息
3. 如果 Bot 回复了，恭喜你配置成功！ 🎉

**常见问题排查**：
- Bot 不回复 → 检查 Message Content Intent 是否开启
- 提示权限不足 → 检查 Bot 在频道的权限设置

## 五、为什么要这样用 Discord 集成？

之所以推荐在 Discord 中使用集成，主要有几个原因：

- **原因 1：社区隔离**：Discord 优秀的频道权限管理，让 AI 的活动范围完全可控。
- **原因 2：Slash Command**：斜杠命令是目前最规范的人机交互方式，参数清晰，不会误触。
- **原因 3：多模态展示**：Discord 对图片、Embed 消息卡片的支持极好，展示效果华丽。

如果你选择在微信群或 QQ 群：
- 很难控制它什么时候说话，什么时候闭嘴
- 生成的图片经常被压缩或屏蔽

## 六、它适合谁？不适合谁？

**适合：**
- **游戏公会**：需要查攻略、记分数、活跃气氛。
- **Web3/DAO 组织**：需要自动化回答社区常见问题（FAQ）。
- **兴趣小组**：一起玩 AI 画图，分享 Prompt。

**不太适合：**
- **正式商务沟通**：Discord 的画风偏娱乐，不适合严肃的即时通讯。
- **1 对 1 私密咨询**：虽然支持 DM，但 Discord 的强项在于公开频道。
