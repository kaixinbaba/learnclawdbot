# Moltbot 名词解释

本指南解释了在使用 Moltbot 时会遇到的关键术语和概念。

## 核心概念

### Agent (代理)
一个能够执行任务、做出决策并与各种工具和环境交互的自主 AI 实体。

### Skill (技能)
可以添加到 Moltbot 的特定能力或功能。技能扩展了 Moltbot 的能力，例如访问新的 API、执行特定计算或与其他服务集成。

### Workspace (工作区)
Moltbot 运行的环境。工作区包含您的文件、配置和上下文。

### Task (任务)
分配给 Moltbot 的特定工作单元。任务可以是简单的（例如“创建一个文件”），也可以是复杂的（例如“建立一个网站”）。

## 进阶术语

### Context Window (上下文窗口)
Moltbot 在处理请求时一次可以考虑的信息量（文本、代码、历史记录）。

### Planner (规划器)
Moltbot 中负责将高层目标分解为可执行步骤的组件。

## 社区术语

### ClawdHub
浏览和安装 Moltbot 技能及扩展的官方市场。
