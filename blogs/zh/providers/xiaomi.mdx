小米大模型是小米自研的 AI 大语言模型，与小米生态深度集成。本教程教你如何在 OpenClaw 中配置和使用小米大模型。

## 为什么选择小米大模型？

- **小米生态集成** — 与小米手机、IoT 设备无缝连接，智能家居场景友好
- **性能稳定** — 基于小米澎湃大模型，性能表现稳定可靠
- **国内优化** — 针对国内用户使用习惯优化，响应速度快

## 配置方式

运行 `openclaw onboard`，选择 Xiaomi 进行配置。

{/* 截图：OpenClaw onboard 选择 Xiaomi */}

### 获取 API Key

**1. 访问 [小米 AI 开放平台](https://ai.xiaomi.com/)，使用小米账号登录：**

{/* 截图：小米 AI 开放平台首页 */}

**2. 进入控制台，点击"API Key 管理"：**

{/* 截图：控制台导航栏 */}

**3. 点击"创建 API Key"按钮：**

{/* 截图：API Key 管理页面 */}

**4. 填写 Key 名称和用途说明，点击创建：**

{/* 截图：创建 API Key 表单 */}

**5. 复制生成的 API Key（仅显示一次，请妥善保管）：**

{/* 截图：API Key 生成成功提示 */}

**6. 在 `openclaw onboard` 中选择 Xiaomi，粘贴 API Key：**

{/* 截图：OpenClaw 粘贴 Xiaomi API Key */}

### 配置示例

在 OpenClaw 配置文件中，小米大模型的配置格式如下：

```yaml
models:
  xiaomi:
    apiKey: YOUR_XIAOMI_API_KEY
    baseURL: https://api.xiaomi.com/v1  # 可选，默认值
    models:
      - xiaomi-chat-pro
      - xiaomi-chat
      - xiaomi-chat-lite
```

## 模型选择

| 模型 | 特点 | 适用场景 |
|------|------|----------|
| xiaomi-chat-pro | 专业版、能力强 | 复杂任务、专业场景、深度推理 |
| xiaomi-chat | 标准版、平衡 | 日常对话、一般任务、大多数场景 |
| xiaomi-chat-lite | 轻量版、快速 | 简单任务、快速响应、高频调用 |

## 常见问题

### 需要小米设备才能使用吗？

不需要。虽然小米大模型与小米生态集成良好，但使用 API 不需要拥有小米设备，只需要注册小米账号即可。

### 如何查看用量？

登录小米 AI 开放平台控制台，在"用量统计"页面可以查看 API 调用次数和消耗情况。

### 有免费额度吗？

新用户通常有一定的免费试用额度，具体政策请查看小米 AI 开放平台的最新公告。
