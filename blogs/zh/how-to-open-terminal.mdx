终端（Terminal）是一个用来输入命令的工具，看起来像一个黑色的窗口，你在里面打字，电脑就会执行对应的操作。很多开发者工具（包括 OpenClaw）都需要通过终端来安装和使用。

别担心，打开它只需要几秒钟。

## macOS

### 方法一：Spotlight 搜索（最快）

1. 按下 **⌘ Command + 空格键**
2. 输入 **Terminal** 或 **终端**
3. 按回车打开

<img src="/images/blog/terminal/macos-spotlight.png" alt="macOS Spotlight 搜索终端" style="max-width: 75%" />

### 方法二：从 Launchpad 打开

1. 点击 Dock 栏上的 **Launchpad** 图标（火箭图标）
2. 在顶部搜索栏输入 **Terminal** 或 **终端**
3. 点击打开

<img src="/images/blog/terminal/macos-launchpad.png" alt="macOS Launchpad 搜索终端" style="max-width: 75%" />

> 💡 **推荐：** 打开后右键点击 Dock 栏上的终端图标，选择"在 Dock 中保留"，下次直接点击就能打开。

## Linux

### Ubuntu / Debian

按下 **Ctrl + Alt + T**，终端会直接弹出来。

或者在应用菜单中搜索 **Terminal**。

### 其他发行版

大多数 Linux 发行版都可以用 **Ctrl + Alt + T** 打开终端。如果不行，在应用菜单里搜索 "Terminal" 或 "Console"。

## Windows

OpenClaw 在 Windows 上需要通过 **WSL2**（Windows Subsystem for Linux）运行，不支持原生 Windows 终端。

### 第一步：安装 WSL2

1. 按下 **Win + X**，选择 **终端（管理员）** 或 **PowerShell（管理员）**
2. 输入以下命令并回车：

```powershell
wsl --install
```

3. 等待安装完成后 **重启电脑**

### 第二步：打开 WSL 终端

重启后：
1. 按下 **Win 键**
2. 搜索 **Ubuntu**（或你安装的 Linux 发行版名称）
3. 点击打开

首次打开会要求你设置用户名和密码，设置完就可以使用了。

> 📖 更多关于 WSL2 的信息，请参考[微软官方文档](https://learn.microsoft.com/zh-cn/windows/wsl/install)。

现在你应该看到一个等待你输入的窗口，光标在闪烁。恭喜，你已经准备好了！
