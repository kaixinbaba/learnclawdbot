# OpenClaw 新手指南

欢迎使用 OpenClaw！本指南将帮助你设置属于自己的 AI 个人助手，让它真正为你做事。

## 什么是 OpenClaw?

OpenClaw 是一个开源的 AI 个人助手网关，通过你喜欢的即时通讯应用将你连接到强大的 AI 智能体。与传统的聊天机器人不同，OpenClaw 的设计目标是**执行任务**——它可以浏览网页、控制智能家居、管理日历、编写代码等等。

### 核心理念：行动胜于对话

大多数 AI 助手只会给你答案。OpenClaw 更进一步——它会采取行动。需要检查电子邮件？OpenClaw 可以做到。想要自动化你的浏览器？完成。需要部署代码？OpenClaw 处理它。这一根本差异使 OpenClaw 成为真正的**个人助手**，而不仅仅是一个聊天机器人。

### 多渠道支持

OpenClaw 充当网关，将一个 AI 智能体连接到多个消息平台：

- **Telegram** — 易于设置，非常适合初学者
- **Discord** — 适合社区和团队
- **WhatsApp** — 使用你的日常通讯工具
- **iMessage** — 原生 iOS/macOS 集成
- **Email** — 传统但强大
- 以及更多！

你只需安装一次 OpenClaw，你的 AI 助手就可以在你聊天的任何地方使用。

### 简史

OpenClaw 经历了几次迭代演进：
- **Moltbot** — 最初的实验
- **Clawbot** — 第一个稳定版本
- **OpenClaw** — 当前的开源版本

每个版本都改进了核心理念：为你提供一个无缝集成到日常工作流程中的个人 AI 智能体。

## 主要功能

### 多渠道网关

一个 OpenClaw 实例可以同时服务于多个聊天平台。设置一次，从 Telegram、Discord 或 WhatsApp 与你的助手聊天——你的对话和上下文会跟随你到任何地方。

### AI 智能体集成

OpenClaw 支持多个 AI 提供商：
- **Anthropic Claude**（推荐）— 出色的推理和工具使用能力
- **OpenAI GPT** — 多功能且强大
- **Google Gemini** — 快速且多模态
- **Pi** — 对话友好
- **自定义模型** — 使用你自己的模型

在不同模型之间切换，或为不同目的运行多个智能体。

### 浏览器自动化

OpenClaw 可以通过 [Browser Relay](/blog/browser-relay) 功能控制网页浏览器。需要填写表单、抓取数据或自动化重复的网页任务？你的智能体可以在你观看或在后台工作时完成。

### 智能家居控制

与 **Home Assistant** 集成，通过自然对话控制灯光、恒温器、摄像头等。"关闭客厅的灯"就这么简单。

### 开发者工作流

OpenClaw 擅长开发者任务：
- 读取和编写代码
- Git 操作（提交、推送、创建 PR）
- 代码审查
- CI/CD 流水线管理
- 通过 SSH 进行服务器管理

### 技能系统

OpenClaw 的功能可通过**技能系统**扩展。每个技能提供特定功能——从日历管理到图像生成。你可以创建自定义技能以满足你的独特需求。

## 你需要准备什么

在开始之前，请确保你有：

1. **一台计算机**，运行 macOS、Linux 或 Windows
2. **Node.js 18+** 或 **Docker** 用于运行 OpenClaw
3. **AI 提供商 API 密钥** — 我们建议从 [Anthropic Claude](https://console.anthropic.com/) 开始
4. **即时通讯应用账户** — Telegram、Discord 或 WhatsApp

就这些！基本设置非常简单。

## 快速安装

OpenClaw 提供多种安装方法。以下是快速概述：

### 通过 Homebrew（macOS/Linux）

```bash
brew install openclaw/tap/openclaw
```

### 通过 npm

```bash
npm install -g openclaw
```

### 通过 Docker

```bash
docker pull openclaw/openclaw:latest
```

### 从源码安装

```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
npm install
npm run build
```

有关详细的安装说明，包括 Windows 设置、故障排除和环境配置，请参阅[完整安装指南](/blog/openclaw-setup-installation-guide)。

## 首次设置：引导向导

安装后，OpenClaw 提供交互式设置向导帮助你开始：

```bash
openclaw onboard
```

此命令启动一个友好的演练，帮助你：

### 1. 选择你的 AI 提供商

向导会询问你想使用哪个 AI 服务。对于初学者，我们推荐 **Anthropic Claude**，因为它具有出色的推理和工具使用能力。

### 2. 设置你的 API 密钥

你需要提供来自所选提供商的 API 密钥：

- **Anthropic**：在 [console.anthropic.com](https://console.anthropic.com/) 获取
- **OpenAI**：访问 [platform.openai.com](https://platform.openai.com/)
- **Google**：查看 [ai.google.dev](https://ai.google.dev/)

向导会安全地将你的密钥存储在配置文件中。

### 3. 配置你的第一个渠道

接下来，你将设置第一个消息渠道。向导支持：
- Telegram（最适合初学者）
- Discord
- WhatsApp
- Email

选择一个开始——以后可以添加更多渠道！

## 连接你的第一个渠道

让我们逐步设置你的第一个渠道。我们将介绍三个最受欢迎的选项。

### 选项 A：Telegram（推荐给初学者）

Telegram 是开始使用 OpenClaw 最简单的方式。

**步骤 1：使用 BotFather 创建机器人**

1. 打开 Telegram 并搜索 `@BotFather`
2. 发送 `/newbot` 开始创建你的机器人
3. 选择一个名称（例如，"My OpenClaw Assistant"）
4. 选择一个用户名（必须以 'bot' 结尾，例如，`my_openclaw_bot`）
5. BotFather 会给你一个 **bot token** — 保存它！

**步骤 2：配置 OpenClaw**

```bash
openclaw channel add telegram
```

提示时，粘贴你的 bot token。

**步骤 3：开始聊天**

1. 在 Telegram 中找到你的机器人（搜索你创建的用户名）
2. 发送 `/start` 开始
3. 尝试说"你好！" — 你的 AI 助手应该会回复！

### 选项 B：Discord

Discord 非常适合社区和团队。

**步骤 1：创建 Discord 应用程序**

1. 前往 [discord.com/developers/applications](https://discord.com/developers/applications)
2. 点击 "New Application" 并给它命名
3. 导航到 "Bot" 选项卡
4. 点击 "Add Bot" 并确认
5. 在 "Token" 下，点击 "Reset Token" 并复制它（安全保存！）
6. 启用这些特权网关意图：
   - Presence Intent
   - Server Members Intent
   - Message Content Intent

**步骤 2：配置 OpenClaw**

```bash
openclaw channel add discord
```

提示时粘贴你的 bot token。

**步骤 3：邀请到你的服务器**

1. 在 Discord 开发者门户中，转到 "OAuth2" → "URL Generator"
2. 选择范围：`bot` 和 `applications.commands`
3. 选择权限：至少需要 `Read Messages/View Channels`、`Send Messages`、`Read Message History`
4. 复制生成的 URL 并在浏览器中打开
5. 选择你的服务器并授权

**步骤 4：开始聊天**

在机器人有访问权限的任何频道中，提及它或直接私信它以开始对话！

### 选项 C：WhatsApp

WhatsApp 需要 QR 码配对，稍微复杂一些。

**步骤 1：配置 OpenClaw**

```bash
openclaw channel add whatsapp
```

**步骤 2：扫描 QR 码**

OpenClaw 会在你的终端中显示一个 QR 码。在手机上打开 WhatsApp 并扫描它：

1. 打开 WhatsApp
2. 点击菜单 (⋮) 或设置
3. 点击 "已链接的设备"
4. 点击 "链接设备"
5. 扫描终端中的 QR 码

**步骤 3：开始聊天**

配对后，向链接的设备（你自己）发送消息以测试连接！

有关 WhatsApp 设置和故障排除的更多详细信息，请查看[官方文档](https://openclaw.org/docs)。

## 你的第一次对话

现在你的渠道已连接，让我们与你的 AI 助手进行第一次对话！

### 发送你的第一条消息

尝试一些简单的：

```
你好！你能帮我吗？
```

你的助手应该会自我介绍并询问你需要什么。OpenClaw 智能体旨在提供帮助、友好且以行动为导向。

### 理解智能体的响应

当你要求助手做某事时，你会注意到它逐步思考任务。例如：

```
你：在网上搜索有关 AI 的最新新闻
智能体：我将为你搜索最近的 AI 新闻。
[智能体使用 web_search 工具]
智能体：这是我找到的：[带有链接和摘要的结果]
```

智能体会告诉你它在做什么，使用适当的工具，并返回有用的结果。

### 基本命令

OpenClaw 支持几个内置命令：

**检查状态：**
```
/status
```
显示你当前的会话、活动模型和系统状态。

**获取帮助：**
```
/help
```
列出可用命令和基本使用说明。

**查看上下文：**
```
/context
```
显示当前会话上下文和内存。

**重启会话：**
```
/restart
```
清除当前对话并重新开始。

这些命令在所有渠道中都有效，是你管理助手的首选工具。

## 理解基础知识

### 会话

与 OpenClaw 的每次对话都发生在一个**会话**中。将会话视为对话上下文——它记住你说过的话、正在进行的任务以及相关背景信息。

**关键会话概念：**

- **隔离**：每个聊天（私信、频道、群组）都有自己的会话
- **内存**：会话记住你的对话历史
- **上下文**：智能体知道你在谈论什么
- **持久性**：会话在重启后仍然存在（它们被保存到磁盘）

如果你遇到智能体忘记事情或混淆上下文的问题，请查看[会话内存指南](/blog/troubleshooting/fix-session-memory-issues)以获取故障排除提示。

### 命令

我们在上面介绍了基本命令，但 OpenClaw 支持更多命令：

- `/thinking` — 切换推理显示
- `/model` — 切换 AI 模型
- `/skills` — 列出可用技能
- `/config` — 显示当前配置

有关完整的命令参考，请参阅[官方文档](https://openclaw.org/docs/commands)。

### 控制 UI

OpenClaw 包含一个基于 Web 的控制面板，用于管理你的设置。

**访问 UI：**

```bash
openclaw start
```

然后在浏览器中打开 `http://localhost:3000`（默认端口）。

**你可以做什么：**

- 查看所有活动会话
- 查看对话历史
- 管理渠道配置
- 监控系统健康状况
- 调整智能体设置
- 查看日志和调试信息

控制 UI 在运行多个渠道或调试问题时特别有用。

## 常见任务

让我们探索你实际上可以用 OpenClaw **做什么**。

### 代码辅助

OpenClaw 擅长帮助编程任务：

```
你：创建一个 Python 脚本，读取 CSV 文件并计算第二列中数字的平均值。

智能体：[创建并解释代码]
```

智能体可以读取你工作区中的文件、编写新代码、解释复杂概念，甚至调试错误。

### 日历和电子邮件管理

启用适当的技能后：

```
你：明天我的日历上有什么？
智能体：[检查日历并列出事件]

你：给 john@example.com 发送一封关于我们会议的电子邮件
智能体：[草拟电子邮件并在发送前请求确认]
```

### 网络搜索和研究

你的智能体可以搜索网络并总结发现：

```
你：研究 React hooks 的最佳实践并为我总结
智能体：[搜索、阅读文章并提供全面总结]
```

### 浏览器自动化

使用 [Browser Relay](/blog/browser-relay) 功能，你的智能体可以控制网页浏览器：

```
你：转到 example.com 并用我的详细信息填写联系表
智能体：[打开浏览器、导航、填写表单并提交]
```

这对于自动化重复的网页任务或测试 Web 应用程序非常强大。

### 智能家居控制

如果你已集成 Home Assistant：

```
你：关闭所有灯
智能体：[向 Home Assistant 发送命令]

你：客厅的温度是多少？
智能体：[检查传感器并报告]
```

### 文件管理和组织

```
你：按日期组织我的下载文件夹中的所有 PDF 文件
智能体：[扫描文件夹，创建基于日期的子文件夹，移动文件]
```

可能性无穷无尽！你的助手可以访问强大的工具，并可以创造性地组合它们来解决复杂的任务。
