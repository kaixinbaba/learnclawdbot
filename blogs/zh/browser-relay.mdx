# OpenClaw Browser Relay - 安装和使用指南

OpenClaw Browser Relay 是一个强大的 Chrome 扩展程序，能够在您的浏览器和 OpenClaw AI 之间建立无缝通信。通过这个扩展，您可以授权 OpenClaw 与网页交互、自动化重复性任务，并代表您执行复杂的基于浏览器的操作。

## 什么是 Browser Relay？

Browser Relay 充当 Chrome 浏览器和 OpenClaw AI 之间的桥梁。可以将它视为一个"遥控器"，允许 AI 执行以下操作：

- **导航网页** - 打开 URL、点击链接和浏览网站
- **填写表单** - 自动将数据输入到 Web 表单中
- **提取信息** - 从网页读取和分析内容
- **与元素交互** - 点击按钮、选择选项和操作页面元素
- **截屏** - 从网站捕获视觉信息
- **执行自动化** - 跨不同网站执行多步骤工作流

该扩展以安全、受控的方式运行 - 您保持完全控制，并可以实时看到 AI 所做的一切。

## 安装步骤

安装 Browser Relay 扩展需要将其作为解压扩展程序手动加载。具体步骤如下：

### 步骤 1：找到扩展文件

Browser Relay 扩展随 OpenClaw 一起提供。在以下位置找到扩展文件：

```
~/.openclaw/extensions/browser-relay/
```

如果您没有看到此目录，请确保已安装最新版本的 OpenClaw。

### 步骤 2：打开 Chrome 扩展程序页面

1. 打开 Google Chrome
2. 在地址栏中导航到 `chrome://extensions/`
3. 或者，点击三点菜单 → **更多工具** → **扩展程序**

### 步骤 3：启用开发者模式

在扩展程序页面的右上角，打开**开发者模式**。

### 步骤 4：加载扩展

1. 点击启用开发者模式后出现的**加载已解压的扩展程序**按钮
2. 导航到 `~/.openclaw/extensions/browser-relay/` 目录
3. 选择该文件夹并点击**选择文件夹**（在某些系统上为**打开**）

### 步骤 5：验证安装

加载后，您应该在扩展程序列表中看到 OpenClaw Browser Relay 扩展。您还会注意到 Chrome 工具栏中出现一个新图标（通常在地址栏附近）。

## 如何使用 Browser Relay

安装完成后，使用 Browser Relay 非常简单：

### 附加标签页

要允许 OpenClaw 控制特定的浏览器标签页：

1. **导航到您希望 OpenClaw 交互的页面**
2. **点击 Chrome 工具栏中的 Browser Relay 图标**
3. 扩展图标将改变以显示它已**附加**到当前标签页
4. 徽章或指示器（通常为"ON"）确认连接

**重要提示：** OpenClaw 只能控制明确附加的标签页。此安全措施确保您始终知道 AI 可以访问哪些标签页。

### 分离标签页

要断开 OpenClaw 与标签页的连接：

1. 再次点击 Browser Relay 图标
2. 徽章将关闭，表示该标签页不再附加

### 使用多个标签页

您可以在不同时间附加不同的标签页，但通常一次只控制一个标签页。当您要求 OpenClaw 执行浏览器操作时，它将使用当前附加的标签页。

## 常见用例

以下是使用 OpenClaw 和 Browser Relay 的一些实际方式：

### 1. 网络研究和数据提取

**示例：** "从这个电子商务页面提取所有产品价格并创建比较表。"

OpenClaw 可以导航产品列表、提取结构化数据并以有组织的格式呈现。

### 2. 表单填写和提交

**示例：** "用我的信息填写这个联系表单并提交。"

非常适合重复的表单提交、调查或注册流程。

### 3. 自动化测试

**示例：** "使用这些凭据测试此网站上的登录流程。"

将多个操作链接在一起，跨不同站点完成复杂任务。

## 提示和最佳实践

### 安全注意事项

- **仅附加可信标签页** - 谨慎对待您授予 OpenClaw 访问权限的网站
- **审查操作** - 观察 OpenClaw 的操作，尤其是在处理敏感信息的网站上
- **完成后分离** - 断开您不再与 OpenClaw 一起使用的标签页

### 性能提示

- **关闭不必要的标签页** - 当 Chrome 不超负荷时，Browser Relay 效果最佳
- **使用具体指令** - 清晰的命令有助于 OpenClaw 更准确地执行操作
- **等待页面加载** - 复杂页面在交互之前可能需要时间完全加载

### 故障排除

**扩展图标未出现？**
- 确保已启用开发者模式
- 尝试从 `chrome://extensions/` 重新加载扩展

**无法附加标签页？**
- 刷新页面并重试
- 某些受保护的页面（如 `chrome://` URL）无法控制

**操作不起作用？**
- 确保标签页仍然附加（检查"ON"徽章）
- 验证 OpenClaw 正在运行并已连接

## OpenClaw 中的配置文件选择

在 OpenClaw 中使用浏览器命令时，您需要指定要使用的浏览器配置文件：

- **`profile="chrome"`** - 使用 Browser Relay 扩展接管您现有的 Chrome 标签页
- **`profile="openclaw"`** - 使用 OpenClaw 的隔离、托管浏览器实例

对于 Browser Relay，始终使用 `profile="chrome"`：

```
使用 profile="chrome" 的浏览器填写当前标签页上的表单。
```
