# OpenClaw Browser Relay - Руководство по установке и использованию

OpenClaw Browser Relay — это мощное расширение Chrome, которое обеспечивает бесшовное взаимодействие между вашим браузером и OpenClaw AI. С помощью этого расширения вы можете предоставить OpenClaw возможность взаимодействовать с веб-страницами, автоматизировать повторяющиеся задачи и выполнять сложные операции в браузере от вашего имени.

## Что такое Browser Relay?

Browser Relay выступает в качестве моста между вашим браузером Chrome и OpenClaw AI. Думайте о нем как о «пульте дистанционного управления», который позволяет ИИ:

- **Навигация по веб-страницам** - Открывать URL-адреса, нажимать на ссылки и просматривать веб-сайты
- **Заполнение форм** - Автоматически вводить данные в веб-формы
- **Извлечение информации** - Читать и анализировать контент веб-страниц
- **Взаимодействие с элементами** - Нажимать кнопки, выбирать параметры и манипулировать элементами страницы
- **Делать скриншоты** - Захватывать визуальную информацию с веб-сайтов
- **Выполнение автоматизации** - Выполнять многоэтапные рабочие процессы на разных веб-сайтах

Расширение работает безопасным и контролируемым образом - вы сохраняете полный контроль и можете видеть все, что делает ИИ, в реальном времени.

## Шаги по установке

Установка расширения Browser Relay требует ручной загрузки в качестве распакованного расширения. Вот как это сделать:

### Шаг 1: Найдите файлы расширения

Расширение Browser Relay поставляется в комплекте с OpenClaw. Найдите файлы расширения по адресу:

```
~/.openclaw/extensions/browser-relay/
```

Если вы не видите этот каталог, убедитесь, что у вас установлена последняя версия OpenClaw.

### Шаг 2: Откройте расширения Chrome

1. Откройте Google Chrome
2. Перейдите к `chrome://extensions/` в адресной строке
3. Или нажмите меню с тремя точками → **Дополнительные инструменты** → **Расширения**

### Шаг 3: Включите режим разработчика

В правом верхнем углу страницы расширений включите **Режим разработчика**.

### Шаг 4: Загрузите расширение

1. Нажмите кнопку **Загрузить распакованное расширение**, которая появляется после включения режима разработчика
2. Перейдите в каталог `~/.openclaw/extensions/browser-relay/`
3. Выберите папку и нажмите **Выбрать папку** (или **Открыть** в некоторых системах)

### Шаг 5: Проверьте установку

После загрузки вы должны увидеть расширение OpenClaw Browser Relay в списке ваших расширений. Вы также заметите новый значок на панели инструментов Chrome (обычно рядом с адресной строкой).

## Как использовать Browser Relay

После установки использование Browser Relay очень просто:

### Подключение вкладки

Чтобы позволить OpenClaw управлять определенной вкладкой браузера:

1. **Перейдите на страницу**, с которой вы хотите, чтобы OpenClaw взаимодействовал
2. **Нажмите значок Browser Relay** на панели инструментов Chrome
3. Значок расширения изменится, показывая, что он **подключен** к текущей вкладке
4. Значок или индикатор (обычно «ON») подтверждает соединение

**Важно:** OpenClaw может управлять только явно подключенными вкладками. Эта мера безопасности гарантирует, что вы всегда знаете, к каким вкладкам имеет доступ ИИ.

### Отключение вкладки

Чтобы отключить OpenClaw от вкладки:

1. Снова нажмите значок Browser Relay
2. Значок выключится, указывая, что вкладка больше не подключена

### Работа с несколькими вкладками

Вы можете подключать разные вкладки в разное время, но обычно одновременно управляется только одна вкладка. Когда вы просите OpenClaw выполнить действия в браузере, он будет использовать текущую подключенную вкладку.

## Распространенные случаи использования

Вот несколько практических способов использования Browser Relay с OpenClaw:

### 1. Веб-исследования и извлечение данных

**Пример:** «Извлеките все цены на продукты с этой страницы электронной коммерции и создайте сравнительную таблицу».

OpenClaw может перемещаться по спискам продуктов, извлекать структурированные данные и представлять их в организованном формате.

### 2. Заполнение и отправка форм

**Пример:** «Заполните эту контактную форму моей информацией и отправьте ее».

Идеально подходит для повторяющихся отправок форм, опросов или процессов регистрации.

### 3. Автоматизированное тестирование

**Пример:** «Протестируйте процесс входа на этом веб-сайте, используя эти учетные данные».

QA-инженеры могут использовать OpenClaw для автоматизации сценариев тестирования на основе браузера.

### 4. Мониторинг контента

**Пример:** «Проверяйте эту страницу каждый час и уведомляйте меня, если статус 'Нет в наличии' изменится».

Автоматически отслеживайте веб-сайты на предмет конкретных изменений или обновлений.

### 5. Многоэтапные рабочие процессы

**Пример:** «Найдите рейсы из Нью-Йорка в Лондон на этих трех веб-сайтах и сравните самые дешевые варианты».

Связывайте несколько действий на разных сайтах для выполнения сложных задач.

### 6. Скриншоты и документация

**Пример:** «Сделайте скриншоты каждого шага в этом руководстве и сохраните их в папку».

Систематически документируйте процессы или захватывайте визуальную информацию.

## Советы и лучшие практики

### Соображения безопасности

- **Подключайте только доверенные вкладки** - Будьте осторожны с сайтами, к которым вы предоставляете доступ OpenClaw
- **Проверяйте действия** - Следите за тем, что делает OpenClaw, особенно на сайтах, обрабатывающих конфиденциальную информацию
- **Отключайте по завершении** - Отключайте вкладки, которые вы больше не используете с OpenClaw

### Советы по производительности

- **Закрывайте ненужные вкладки** - Browser Relay работает лучше всего, когда Chrome не перегружен
- **Используйте конкретные инструкции** - Четкие команды помогают OpenClaw выполнять действия более точно
- **Ждите загрузки страницы** - Сложным страницам может потребоваться время для полной загрузки перед взаимодействием

### Устранение неполадок

**Значок расширения не появляется?**
- Убедитесь, что режим разработчика включен
- Попробуйте перезагрузить расширение из `chrome://extensions/`

**Не можете подключить вкладку?**
- Обновите страницу и попробуйте снова
- Некоторые защищенные страницы (например, URL-адреса `chrome://`) не могут быть управляемы

**Действия не работают?**
- Убедитесь, что вкладка все еще подключена (проверьте значок «ON»)
- Проверьте, что OpenClaw запущен и подключен

## Выбор профиля в OpenClaw

При использовании команд браузера с OpenClaw вам нужно указать, какой профиль браузера использовать:

- **`profile="chrome"`** - Использует расширение Browser Relay для управления вашими существующими вкладками Chrome
- **`profile="openclaw"`** - Использует изолированный управляемый экземпляр браузера OpenClaw

Для Browser Relay всегда используйте `profile="chrome"`:

```
Используйте браузер с profile="chrome" для заполнения формы на текущей вкладке.
```
