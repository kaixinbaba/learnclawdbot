# Руководство для начинающих по OpenClaw

Добро пожаловать в OpenClaw! Это руководство поможет вам настроить собственного AI-персонального помощника, который действительно выполняет работу.

## Что такое OpenClaw?

OpenClaw — это шлюз персонального помощника на базе AI с открытым исходным кодом, который подключает вас к мощным AI-агентам через ваши любимые приложения для обмена сообщениями. В отличие от традиционных чат-ботов, которые просто разговаривают, OpenClaw разработан для **выполнения задач** — он может просматривать веб-страницы, управлять умным домом, управлять календарем, писать код и многое другое.

### Основная философия: действие вместо разговора

Большинство AI-помощников останавливаются на предоставлении ответов. OpenClaw идет дальше — он действует. Нужно проверить электронную почту? OpenClaw может это сделать. Хотите автоматизировать браузер? Готово. Нужно развернуть код? OpenClaw справится. Это фундаментальное различие делает OpenClaw настоящим **персональным помощником**, а не просто чат-ботом.

### Поддержка нескольких каналов

OpenClaw выступает в качестве шлюза, подключая одного AI-агента к нескольким платформам обмена сообщениями:

- **Telegram** — Легкая настройка, отлично подходит для начинающих
- **Discord** — Идеально для сообществ и команд
- **WhatsApp** — Используйте свой повседневный мессенджер
- **iMessage** — Нативная интеграция iOS/macOS
- **Email** — Традиционный, но мощный
- И многое другое!

Установите OpenClaw один раз, и ваш AI-помощник станет доступен везде, где вы общаетесь.

### Краткая история

OpenClaw прошел через несколько итераций:
- **Moltbot** — Первый эксперимент
- **Clawbot** — Первый стабильный релиз
- **OpenClaw** — Текущая версия с открытым исходным кодом

Каждая версия улучшала основную идею: предоставить вам персонального AI-агента, который бесшовно интегрируется в ваш повседневный рабочий процесс.

## Ключевые возможности

### Мультиканальный шлюз

Один экземпляр OpenClaw может одновременно обслуживать несколько чат-платформ. Настройте один раз, общайтесь с помощником из Telegram, Discord или WhatsApp — ваши разговоры и контекст следуют за вами повсюду.

### Интеграция AI-агентов

OpenClaw поддерживает несколько поставщиков AI:
- **Anthropic Claude** (рекомендуется) — Отличные возможности рассуждения и использования инструментов
- **OpenAI GPT** — Универсальный и мощный
- **Google Gemini** — Быстрый и мультимодальный
- **Pi** — Разговорный и дружелюбный
- **Пользовательские модели** — Используйте свои собственные

Переключайтесь между моделями или запускайте несколько агентов для разных целей.

### Автоматизация браузера

OpenClaw может управлять веб-браузерами через функцию [Browser Relay](/blog/browser-relay). Нужно заполнить формы, извлечь данные или автоматизировать повторяющиеся веб-задачи? Ваш агент может делать это, пока вы наблюдаете или работаете в фоновом режиме.

### Управление умным домом

Интегрируйтесь с **Home Assistant** для управления освещением, термостатами, камерами и многим другим через естественный разговор. «Выключи свет в гостиной» просто работает.

### Рабочие процессы разработчика

OpenClaw превосходен в задачах разработки:
- Чтение и написание кода
- Операции Git (коммит, пуш, создание PR)
- Проверка кода
- Управление конвейером CI/CD
- Администрирование сервера через SSH

### Система навыков

Возможности OpenClaw расширяются через **систему навыков**. Каждый навык предоставляет определенную функциональность — от управления календарем до генерации изображений. Вы можете создавать пользовательские навыки для своих уникальных потребностей.

## Что вам понадобится

Перед началом работы убедитесь, что у вас есть:

1. **Компьютер** под управлением macOS, Linux или Windows
2. **Node.js 18+** или **Docker** для запуска OpenClaw
3. **API-ключ поставщика AI** — Рекомендуем начать с [Anthropic Claude](https://console.anthropic.com/)
4. **Учетная запись приложения для обмена сообщениями** — Telegram, Discord или WhatsApp

Вот и все! Базовая настройка на удивление проста.

## Быстрая установка

OpenClaw предлагает несколько методов установки. Вот краткий обзор:

### Через Homebrew (macOS/Linux)

```bash
brew install openclaw/tap/openclaw
```

### Через npm

```bash
npm install -g openclaw
```

### Через Docker

```bash
docker pull openclaw/openclaw:latest
```

### Из исходного кода

```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
npm install
npm run build
```

Подробные инструкции по установке, включая настройку Windows, устранение неполадок и конфигурацию среды, см. в [Полном руководстве по установке](/blog/openclaw-setup-installation-guide).

## Первоначальная настройка: мастер адаптации

После установки OpenClaw предоставляет интерактивный мастер настройки для начала работы:

```bash
openclaw onboard
```

Эта команда запускает дружественное пошаговое руководство, которое помогает вам:

### 1. Выбрать поставщика AI

Мастер спросит, какой сервис AI вы хотите использовать. Для начинающих мы рекомендуем **Anthropic Claude** за его отличные возможности рассуждения и использования инструментов.

### 2. Установить API-ключ

Вам нужно будет предоставить API-ключ от выбранного поставщика:

- **Anthropic**: Получите на [console.anthropic.com](https://console.anthropic.com/)
- **OpenAI**: Посетите [platform.openai.com](https://platform.openai.com/)
- **Google**: Проверьте [ai.google.dev](https://ai.google.dev/)

Мастер безопасно сохранит ваш ключ в файле конфигурации.

### 3. Настроить первый канал

Далее вы настроите свой первый канал обмена сообщениями. Мастер поддерживает:
- Telegram (проще всего для начинающих)
- Discord
- WhatsApp
- Email

Выберите один для начала — позже вы сможете добавить больше каналов!

## Подключение первого канала

Давайте пошагово рассмотрим настройку вашего первого канала. Мы рассмотрим три самых популярных варианта.

### Вариант A: Telegram (рекомендуется для начинающих)

Telegram — самый простой способ начать работу с OpenClaw.

**Шаг 1: Создайте бота с помощью BotFather**

1. Откройте Telegram и найдите `@BotFather`
2. Отправьте `/newbot`, чтобы начать создание бота
3. Выберите имя (например, «My OpenClaw Assistant»)
4. Выберите имя пользователя (должно заканчиваться на 'bot', например, `my_openclaw_bot`)
5. BotFather даст вам **bot token** — сохраните его!

**Шаг 2: Настройте OpenClaw**

```bash
openclaw channel add telegram
```

При запросе вставьте токен бота.

**Шаг 3: Начните общение**

1. Найдите своего бота в Telegram (найдите имя пользователя, которое вы создали)
2. Отправьте `/start`, чтобы начать
3. Попробуйте сказать «Привет!» — ваш AI-помощник должен ответить!

### Вариант B: Discord

Discord отлично подходит для сообществ и команд.

**Шаг 1: Создайте приложение Discord**

1. Перейдите на [discord.com/developers/applications](https://discord.com/developers/applications)
2. Нажмите «New Application» и дайте ему имя
3. Перейдите на вкладку «Bot»
4. Нажмите «Add Bot» и подтвердите
5. В разделе «Token» нажмите «Reset Token» и скопируйте его (сохраните безопасно!)
6. Включите эти привилегированные намерения шлюза:
   - Presence Intent
   - Server Members Intent
   - Message Content Intent

**Шаг 2: Настройте OpenClaw**

```bash
openclaw channel add discord
```

При запросе вставьте токен бота.

**Шаг 3: Пригласите на свой сервер**

1. На портале разработчика Discord перейдите в «OAuth2» → «URL Generator»
2. Выберите области: `bot` и `applications.commands`
3. Выберите разрешения: Как минимум `Read Messages/View Channels`, `Send Messages`, `Read Message History`
4. Скопируйте сгенерированный URL и откройте в браузере
5. Выберите свой сервер и авторизуйте

**Шаг 4: Начните общение**

В любом канале, к которому у бота есть доступ, упомяните его или отправьте прямое сообщение, чтобы начать разговор!

### Вариант C: WhatsApp

WhatsApp требует сопряжения QR-кода и немного сложнее.

**Шаг 1: Настройте OpenClaw**

```bash
openclaw channel add whatsapp
```

**Шаг 2: Отсканируйте QR-код**

OpenClaw отобразит QR-код в вашем терминале. Откройте WhatsApp на телефоне и отсканируйте его:

1. Откройте WhatsApp
2. Нажмите Меню (⋮) или Настройки
3. Нажмите «Связанные устройства»
4. Нажмите «Связать устройство»
5. Отсканируйте QR-код из терминала

**Шаг 3: Начните общение**

После сопряжения отправьте сообщение связанному устройству (себе), чтобы протестировать соединение!

Для получения более подробной информации о настройке WhatsApp и устранении неполадок см. [официальную документацию](https://openclaw.org/docs).

## Ваш первый разговор

Теперь, когда ваш канал подключен, давайте проведем первый разговор с вашим AI-помощником!

### Отправка первого сообщения

Попробуйте что-нибудь простое:

```
Привет! Можешь мне помочь?
```

Ваш помощник должен представиться и спросить, что вам нужно. Агенты OpenClaw разработаны так, чтобы быть полезными, дружелюбными и ориентированными на действия.

### Понимание ответов агента

Когда вы просите помощника что-то сделать, вы заметите, что он пошагово обдумывает задачу. Например:

```
Вы: Поищи в интернете последние новости об AI
Агент: Я поищу последние новости об AI для вас.
[Агент использует инструмент web_search]
Агент: Вот что я нашел: [результаты со ссылками и резюме]
```

Агент сообщает вам, что он делает, использует соответствующие инструменты и возвращает полезные результаты.

### Основные команды

OpenClaw поддерживает несколько встроенных команд:

**Проверить статус:**
```
/status
```
Показывает текущий сеанс, активную модель и состояние системы.

**Получить помощь:**
```
/help
```
Перечисляет доступные команды и базовые инструкции по использованию.

**Просмотреть контекст:**
```
/context
```
Отображает текущий контекст сеанса и память.

**Перезапустить сеанс:**
```
/restart
```
Очищает текущий разговор и начинает заново.

Эти команды работают во всех каналах и являются вашими основными инструментами для управления помощником.

## Понимание основ

### Сеансы

Каждый разговор с OpenClaw происходит в **сеансе**. Думайте о сеансе как о контексте разговора — он запоминает, что вы сказали, какие задачи выполняются и соответствующую справочную информацию.

**Ключевые концепции сеанса:**

- **Изоляция**: Каждый чат (личное сообщение, канал, группа) имеет свой собственный сеанс
- **Память**: Сеансы запоминают историю разговора
- **Контекст**: Агент знает, о чем вы говорили
- **Постоянство**: Сеансы переживают перезапуски (они сохраняются на диск)

Если у вас возникают проблемы с тем, что агент забывает вещи или путает контексты, ознакомьтесь с [Руководством по памяти сеансов](/blog/troubleshooting/fix-session-memory-issues) для советов по устранению неполадок.

### Команды

Мы рассмотрели основные команды выше, но OpenClaw поддерживает намного больше:

- `/thinking` — Переключить отображение рассуждений
- `/model` — Переключить модели AI
- `/skills` — Перечислить доступные навыки
- `/config` — Показать текущую конфигурацию

Полный справочник команд см. в [официальной документации](https://openclaw.org/docs/commands).

### Панель управления

OpenClaw включает в себя веб-панель управления для управления вашей настройкой.

**Доступ к интерфейсу:**

```bash
openclaw start
```

Затем откройте браузер на `http://localhost:3000` (порт по умолчанию).

**Что вы можете делать:**

- Просматривать все активные сеансы
- Просматривать историю разговоров
- Управлять конфигурациями каналов
- Мониторить состояние системы
- Настраивать параметры агента
- Просматривать журналы и отладочную информацию

Панель управления особенно полезна при работе с несколькими каналами или отладке проблем.

## Обычные задачи

Давайте рассмотрим, что вы можете **делать** с OpenClaw.

### Помощь с кодом

OpenClaw превосходен в помощи с задачами программирования:

```
Вы: Создай Python-скрипт, который читает CSV-файл и вычисляет среднее значение чисел во втором столбце.

Агент: [Создает и объясняет код]
```

Агент может читать файлы в вашем рабочем пространстве, писать новый код, объяснять сложные концепции и даже отлаживать ошибки.

### Управление календарем и электронной почтой

При включенных соответствующих навыках:

```
Вы: Что у меня в календаре на завтра?
Агент: [Проверяет календарь и перечисляет события]

Вы: Отправь email на john@example.com о нашей встрече
Агент: [Составляет черновик email и просит подтверждения перед отправкой]
```

### Веб-поиск и исследования

Ваш агент может искать в интернете и резюмировать находки:

```
Вы: Изучи лучшие практики React hooks и резюмируй их для меня
Агент: [Ищет, читает статьи и предоставляет всеобъемлющее резюме]
```

### Автоматизация браузера

Используя функцию [Browser Relay](/blog/browser-relay), ваш агент может управлять веб-браузером:

```
Вы: Перейди на example.com и заполни контактную форму моими данными
Агент: [Открывает браузер, переходит, заполняет форму и отправляет]
```

Это невероятно мощно для автоматизации повторяющихся веб-задач или тестирования веб-приложений.

### Управление умным домом

Если вы интегрировали Home Assistant:

```
Вы: Выключи весь свет
Агент: [Отправляет команду в Home Assistant]

Вы: Какая температура в гостиной?
Агент: [Проверяет датчик и сообщает]
```

### Управление файлами и организация

```
Вы: Организуй все PDF-файлы в моей папке Загрузки по дате
Агент: [Сканирует папку, создает подпапки по датам, перемещает файлы]
```

Возможности безграничны! Ваш помощник имеет доступ к мощным инструментам и может творчески их комбинировать для решения сложных задач.

OpenClaw трансформирует способ вашего взаимодействия с AI. Вместо простого общения вы работаете **с** интеллектуальным помощником, который может предпринимать реальные действия в вашем цифровом мире.

Начните с малого — настройте один канал, попробуйте несколько задач, освойтесь. Затем расширяйтесь: добавляйте больше каналов, включайте больше навыков, автоматизируйте больше своего рабочего процесса.

Цель проста: вернуть вам время и умственную энергию, автоматизируя скучные части вашей цифровой жизни. Ваш AI-помощник здесь, чтобы помочь — заставьте его работать!

**Добро пожаловать в OpenClaw. Давайте вместе создадим что-то удивительное.**
