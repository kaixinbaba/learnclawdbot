# Clawdia Голосовой мост

Clawdia Голосовой мост позволяет взаимодействовать с OpenClaw через голосовые вызовы, получая помощь ИИ по телефону без необходимости печатать.

## Что такое Clawdia?

Clawdia — это голосовой мост, соединяющий телефонные вызовы с вашим OpenClaw Gateway. Когда вы звоните на настроенный номер, ваш голос преобразуется в текст, отправляется в OpenClaw для обработки, а ответ воспроизводится голосом.

## Как это работает

Архитектура следует трём простым шагам:

1. **Мост принимает**: мост принимает входящие вызовы через эндпоинт /ask
2. **Пересылка на Gateway**: голосовой ввод пересылается на OpenClaw Gateway через WebSocket
3. **Голосовой ответ**: текстовый ответ преобразуется в речь и воспроизводится

## Варианты использования

- **Помощь без использования рук**: получите помощь ИИ за рулём или при готовке
- **Пользователи мобильных устройств**: предпочитают голосовое взаимодействие тексту
- **Доступность**: голосовой интерфейс для пользователей, которые не могут печатать

## Справка по API

| Эндпоинт | Метод | Описание |
|----------|-------|----------|
| /ask | POST | Отправить голосовой запрос в OpenClaw |

## Конфигурация

Основные переменные окружения:
- OPENCLAW_WS_URL — ваш Gateway WebSocket URL
- VAPI_API_KEY — API ключ голосового провайдера
