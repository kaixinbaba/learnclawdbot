# Clawdia 语音桥接

Clawdia 语音桥接让您可以通过语音电话与 OpenClaw 交互，无需打字即可获得 AI 助手帮助。

## 什么是 Clawdia？

Clawdia 是一个语音桥接工具，将电话呼叫连接到您的 OpenClaw Gateway。当您拨打配置好的号码时，您的声音会被转录、发送到 OpenClaw 进行处理，然后语音播报回复内容。

## 工作原理

架构遵循三个简单步骤：

1. **桥接接收**：桥接服务通过 /ask 端点接收来电
2. **转发到 Gateway**：语音输入通过 WebSocket 转发到 OpenClaw Gateway
3. **语音回复**：文本回复被转换为语音并播放

## 使用场景

- **免提助手**：开车或做饭时获取 AI 帮助
- **移动端优先用户**：更喜欢语音交互而非文字
- **无障碍访问**：为无法打字的用户提供语音界面

## 配置

关键环境变量：
- OPENCLAW_WS_URL - 您的 Gateway WebSocket URL
- VAPI_API_KEY - 语音提供商 API 密钥
