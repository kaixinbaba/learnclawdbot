# Learn OpenClaw について

**Learn OpenClaw** へようこそ。ここは OpenClaw（旧称 Clawbot、Moltbot）を徹底的にマスターするための独立した学習リソースです。

## 私たちのミッション

私たちのミッションはシンプルです：**高度な AI 自動化を誰もが利用できるようにすること。**

OpenClaw のようなツールは、単調な作業を自動化することで私たちの働き方や生き方を変える力を持っています。しかし、急峻な学習曲線がその障壁になりがちです。私たちは、分かりやすいステップバイステップのチュートリアル、深掘り比較、そしてコミュニティ主導のアプローチでそのギャップを埋めます。

## 私たちのストーリー

Learn OpenClaw は 2026 年初頭に誕生しました。AI 愛好家と開発者のグループ（Clawbot の初期ユーザーでもあります）が、急成長する OpenClaw エコシステムに向けた包括的で初心者向けのドキュメントが不足していることに気付いたのがきっかけです。

個人的なメモや「チートシート」として始まったものが、やがてフルスケールの教育プラットフォームへと成長しました。私たちはこのプロジェクトが **Clawbot** から **Moltbot**、そして現在の堅牢な **OpenClaw** へと進化する過程を追い続けてきました。

## 独立しているということ

Learn OpenClaw は**独立したサードパーティプロジェクト**です。公式 OpenClaw 開発チームとの提携・推薦・関連は一切ありません。

**なぜそれが重要なのか？**
- **公平なレビュー：** Claude Code や AutoGPT など他のツールとの正直な比較が提供できます。
- **ユーザー視点のチュートリアル：** ガイドはユーザーの立場から作成され、実際の問題やエッジケースに焦点を当てています。
- **コミュニティファースト：** 次に学びたいことはコミュニティが決めます。

## 私たちが提供するもの

- **ステップバイステップガイド：** インストールから高度なマルチエージェントオーケストレーションまで。
- **ケーススタディ：** ユーザーが OpenClaw で日常をどう自動化しているかの実例。
- **エコシステム情報：** 最新のプラグイン、スキルマニフェスト、ロードマップ変更情報。
- **用語集：** AI エージェント技術用語の総合ガイド。

## チームメンバー

私たちは、オープンソース AI への情熱で結ばれた開発者とライターの小さなチームです。

- **"アーキテクト"**：リードテクニカルライター。CLI とコア機能に焦点を当てています。
- **"オートメーター"**：WhatsApp、Telegram、モバイル統合を専門としています。
- **"キュレーター"**：増え続けるケーススタディとコミュニティTipsのライブラリを管理しています。
