# 关于 Learn OpenClaw

欢迎来到 **Learn OpenClaw**，这里是专注于 OpenClaw（前身为 Clawbot 和 Moltbot）学习的独立资源平台。

## 我们的使命

我们的使命很简单：**让每个人都能轻松掌握先进的 AI 自动化技术。**

我们相信 OpenClaw 这样的工具有能力改变我们的工作和生活方式——通过自动化重复性任务，释放人类的创造力。但陡峭的学习曲线往往是一道门槛。我们在这里，用清晰的分步教程、深度对比分析和社区驱动的内容来填补这一空缺。

## 我们的故事

Learn OpenClaw 于 2026 年初诞生，当时一群 AI 爱好者和开发者（也是 Clawbot 的早期用户）意识到，市场上缺乏针对快速发展的 OpenClaw 生态系统的全面入门文档。

从最初的个人笔记和"速查表"，这个平台逐渐成长为一个完整的教育资源站。我们见证并记录了这个项目从最初的 **Clawbot**，到 **Moltbot**，再到如今强大的 **OpenClaw** 的品牌演变历程。

## 为什么我们是独立的

Learn OpenClaw 是一个**独立的第三方项目**，与官方 OpenClaw 开发团队没有任何隶属关系、背书或关联。

**为什么这很重要？**
- **客观评测：** 我们能够与其他工具（如 Claude Code 或 AutoGPT）进行公正对比。
- **用户视角教程：** 我们的指南从用户的角度撰写，专注于真实场景中的问题与边界情况。
- **社区优先：** 我们优先响应社区希望学习的内容。

## 我们提供什么

- **分步指南：** 从安装配置到高级多代理编排，全程覆盖。
- **案例研究：** 展示真实用户如何利用 OpenClaw 实现自动化。
- **生态更新：** 跟踪最新的插件、技能清单和路线图变化。
- **术语表：** AI 代理技术术语的全面参考。

## 认识我们的团队

我们是一个由开发者和技术写作者组成的小团队，对开源 AI 充满热情。

- **"架构师"**：首席技术写作者，专注于 CLI 和核心功能文档。
- **"自动化专家"**：专注于 WhatsApp、Telegram 和移动端集成。
- **"策展人"**：管理不断增长的案例研究库和社区技巧分享。
