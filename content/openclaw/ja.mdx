# OpenClaw とは？

OpenClaw は、Clawbot と Moltbot として知られていた AI エージェントツールの最新名称です。単なるチャットではなく、タスク実行を目的に設計されています。

質問に答えるだけの従来型チャットボットと違い、OpenClaw は次のことができます：

- **タスクを実行**：実際の操作をあなたの代わりに行う
- **文脈を理解**：プロジェクトや目標を継続的に把握
- **自律的に作業**：最小限の指示で複雑な手順を完了
- **学習と適応**：好みに合わせて改善

## OpenClaw は誰のため？

OpenClaw は以下の方に最適です：

- **開発者**：コーディング作業の自動化
- **コンテンツ制作者**：執筆や編集の支援
- **研究者**：分析や整理のサポート
- **誰でも**：実際に「やってくれる」AI アシスタントが欲しい人
