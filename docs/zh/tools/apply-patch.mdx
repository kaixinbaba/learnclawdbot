# apply_patch 工具

使用结构化补丁格式应用文件更改。这对于多文件或多块编辑非常理想，因为单个 `edit` 调用会很脆弱。

该工具接受单个 `input` 字符串，其中包含一个或多个文件操作：

```
*** Begin Patch
*** Add File: path/to/file.txt
+line 1
+line 2
*** Update File: src/app.ts
@@
-old line
+new line
*** Delete File: obsolete.txt
*** End Patch
```

## 参数

- `input`（必需）：完整的补丁内容，包括 `*** Begin Patch` 和 `*** End Patch`。

## 注意事项

- 路径相对于工作区根目录解析。
- 在 `*** Update File:` 块中使用 `*** Move to:` 重命名文件。
- `*** End of File` 在需要时标记仅 EOF 插入。
- 实验性功能，默认禁用。使用 `tools.exec.applyPatch.enabled` 启用。
- 仅限 OpenAI（包括 OpenAI Codex）。可选择通过 `tools.exec.applyPatch.allowModels` 按模型限制。
- 配置仅在 `tools.exec` 下。

## 示例

```json
{
  "tool": "apply_patch",
  "input": "*** Begin Patch\n*** Update File: src/index.ts\n@@\n-const foo = 1\n+const foo = 2\n*** End Patch"
}
```
