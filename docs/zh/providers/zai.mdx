# Z.AI

Z.AI 是 **GLM** 模型的 API 平台。它为 GLM 提供 REST API 并使用 API key 进行认证。在 Z.AI 控制台创建你的 API key。OpenClaw 使用 `zai` provider
和 Z.AI API key。

## CLI 设置

```bash
openclaw onboard --auth-choice zai-api-key
# 或非交互式
openclaw onboard --zai-api-key "$ZAI_API_KEY"
```

## 配置片段

```json5
{
  env: { ZAI_API_KEY: "sk-..." },
  agents: { defaults: { model: { primary: "zai/glm-4.7" } } }
}
```

## 注意事项

- GLM 模型可以作为 `zai/<model>` 使用(示例: `zai/glm-4.7`)。
- 查看 [/providers/glm](/providers/glm) 了解模型系列概览。
- Z.AI 使用 Bearer 认证和你的 API key。
