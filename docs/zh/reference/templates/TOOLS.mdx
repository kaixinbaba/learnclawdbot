# TOOLS.md - 本地笔记

Skills 定义工具如何工作。这个文件是用来记录你的具体情况——那些你的设置独有的东西。

## 这里记录什么

例如：

- 摄像头名称和位置
- SSH 主机和别名
- TTS 首选语音
- 扬声器/房间名称
- 设备昵称
- 任何环境特定的内容

## 示例

### 摄像头

- `living-room` → 主区域，180° 广角
- `front-door` → 入口，动作触发

### SSH

- `home-server` → 192.168.1.100, 用户: admin

### TTS

- 首选语音："Nova"（温暖，略带英式口音）
- 默认扬声器：厨房 HomePod

## 为什么要分开？

Skills 是共享的。你的设置是你自己的。将它们分开意味着你可以更新 skills 而不丢失你的笔记，也可以共享 skills 而不泄露你的基础设施。

添加任何能帮助你完成工作的内容。这是你的备忘单。
