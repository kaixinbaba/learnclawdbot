# `openclaw configure`

交互式提示以设置凭据、设备和代理默认值。

注意：**模型（Model）**部分现在包含用于
`agents.defaults.models` 白名单（在 `/model` 和模型选择器中显示的内容）的多选。

提示：不带子命令的 `openclaw config` 会打开相同的向导。使用
`openclaw config get|set|unset` 进行非交互式编辑。

相关：
- Gateway 配置参考（Gateway configuration reference）：[配置](/gateway/configuration)
- Config CLI：[Config](/cli/config)

注意：
- 选择 Gateway 运行位置始终更新 `gateway.mode`。如果这是您唯一需要的，您可以选择 "Continue" 而不选择其他部分。
- 面向渠道的服务（Slack/Discord/Matrix/Microsoft Teams）在设置期间提示渠道/房间白名单。您可以输入名称或 ID；向导在可能的情况下将名称解析为 ID。

## 示例

```bash
openclaw configure
openclaw configure --section models --section channels
```
