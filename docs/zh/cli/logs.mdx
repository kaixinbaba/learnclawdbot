# `openclaw logs`

通过 RPC 尾随 Gateway 文件日志（在远程模式下工作）。

相关：
- 日志概述（Logging overview）：[日志](/logging)

## 示例

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
