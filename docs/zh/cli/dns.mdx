# `openclaw dns`

用于广域发现（Tailscale + CoreDNS）的 DNS 帮助器。目前专注于 macOS + Homebrew CoreDNS。

相关：
- Gateway 发现（Gateway discovery）：[发现](/gateway/discovery)
- 广域发现配置（Wide-area discovery config）：[配置](/gateway/configuration)

## 设置（Setup）

```bash
openclaw dns setup
openclaw dns setup --apply
```
