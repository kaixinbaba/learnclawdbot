# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 和代理工作空间。

相关：
- 入门指南：[入门](/start/getting-started)
- 向导：[入门引导](/start/onboarding)

## 示例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

通过 setup 运行向导：

```bash
openclaw setup --wizard
```
