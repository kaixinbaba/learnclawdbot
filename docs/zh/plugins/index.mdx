# 插件

使用插件和自定义工具扩展 OpenClaw 的功能。

## 可用插件

- **[插件清单](manifest)** - 定义插件元数据和要求
- **[代理工具](agent-tools)** - 为代理创建自定义工具
- **[语音通话插件](voice-call)** - 添加语音通话功能
- **[Zalo 用户插件](zalouser)** - 与 Zalo 消息集成

## 创建插件

插件允许您向 OpenClaw 添加新工具、集成和功能。从 [插件清单](manifest) 文档开始，了解如何构建您的插件。
