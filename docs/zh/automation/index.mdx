# 自动化

OpenClaw 提供强大的自动化功能，帮助您构建主动、自主的代理。

## 主题

- **[Cron 任务](cron-jobs)** - 在特定时间安排任务
- **[心跳检测](poll)** - 定期检查和主动监控
- **[Cron vs 心跳](cron-vs-heartbeat)** - 何时使用每种自动化方法
- **[Gmail Pub/Sub](gmail-pubsub)** - 使用 Google Pub/Sub 进行实时邮件监控
- **[认证监控](auth-monitoring)** - 跟踪认证状态和事件

## 入门

首先阅读 [Cron 任务](cron-jobs) 了解如何安排一次性或定期任务，然后探索 [心跳检测](poll) 学习实现主动监控模式。
