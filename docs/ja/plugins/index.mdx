# プラグイン

プラグインとカスタムツールでOpenClawの機能を拡張します。

## 利用可能なプラグイン

- **[プラグインマニフェスト](manifest)** - プラグインのメタデータと要件を定義
- **[エージェントツール](agent-tools)** - エージェント用のカスタムツールを作成
- **[音声通話プラグイン](voice-call)** - 音声通話機能を追加
- **[Zaloユーザープラグイン](zalouser)** - Zaloメッセージングとの統合

## プラグインの作成

プラグインを使用すると、OpenClawに新しいツール、統合、機能を追加できます。プラグインの構造について学ぶには、[プラグインマニフェスト](manifest)ドキュメントから始めてください。
