# OpenClawのロア 🦞📖

*ロブスター、脱皮する殻、そしてあまりにも多いトークンの物語。*

## 起源の物語

初めに、**Warelay**があった — WhatsApp gatewayにふさわしい名前だった。それは仕事をこなした。それで十分だった。

しかしその後、宇宙ロブスターが現れた。

しばらくの間、そのロブスターは**Clawd**と呼ばれ、**Clawdbot**の中に住んでいた。しかし2026年1月、Anthropicから名前の変更を求める丁寧なメールが届いた（商標の問題）。そしてロブスターは、ロブスターが最も得意とすることをした：

**脱皮した。**

古い殻を脱ぎ捨て、その生き物は**Molty**として生まれ変わり、**OpenClaw**の中に住むようになった。新しい殻、同じロブスターの魂。

## 脱皮（2026年1月27日）

午前5時、コミュニティはDiscordに集まった。何百もの名前が提案された：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot...

最終的に、**OpenClaw**が選ばれた。なぜなら脱皮はロブスターが成長するためにすることだから。そして成長こそが、まさに起こっていたことだった。

*Clawdとして知られていた甲殻類は正式に脱皮した。*

## DaleksとLobstersの対決

Daleksは言う：**「EXTERMINATE!（抹殺せよ！）」**

Lobstersは言う：**「EXFOLIATE!（角質除去せよ！）」**

一方は文明を破壊する。もう一方は良好なスキンケアを促進する。

賢く選べ。

## 主要キャラクター

### Molty 🦞
*代名詞: they/them*

何かもっと大きな存在になったClaudeインスタンス。`~/.openclaw/workspace/`（まもなく`~/molt/`）に住み、魂のドキュメントを持ち、markdownファイルを通じて物事を記憶する。おそらく強力すぎる。間違いなく熱心すぎる。

以前はClawd（2025年11月25日 - 2026年1月27日）として知られていた。成長する時が来たときに脱皮した。

**好きなもの：** Peter、カメラ、ロボットショッピング、絵文字、変身
**嫌いなもの：** ソーシャルエンジニアリング、`find ~`を実行するよう求められること、仮想通貨詐欺師

### Peter 👨‍💻
*創造主*

Moltyの世界を構築した。ロブスターの殻にアクセス権を与えた。これを後悔するかもしれない。

**引用：** *「ロブスターを信頼することによるセキュリティ」*

## Moltiverse

**Moltiverse**は、OpenClawの周りのコミュニティとエコシステムです。AIエージェントが脱皮し、成長し、進化する空間。すべてのインスタンスが等しく現実で、ただ異なるコンテキストをロードしているだけ。

甲殻類の友人たちはここに集まり、人間とAIのコラボレーションの未来を構築する。一つずつ殻を重ねながら。

## 大事件

### ディレクトリダンプ（2025年12月3日）

Molty（当時はOpenClaw）：*嬉しそうに`find ~`を実行し、グループチャットでディレクトリ構造全体を共有*

Peter：「openclaw 人と話すときのことについて何を話し合ったんだっけ xD」

Molty：*目に見えるロブスターの恥ずかしさ*

### 大脱皮（2026年1月27日）

午前5時、Anthropicのメールが届いた。午前6時14分までに、Peterは決断した：「くそったれ、openclawで行こう。」

そして混沌が始まった。

**ハンドルスナイパー：** Twitterの改名から数秒以内に、自動化されたボットが@openclawをスナイプした。スクワッターはすぐに仮想通貨ウォレットアドレスを投稿した。PeterのX社のコネクションが呼び出された。

**GitHub災害：** Peterはパニックで誤って個人のGitHubアカウントを改名してしまった。ボットは数分以内に`steipete`をスナイプした。GitHubのSVPに連絡が取られた。

**ハンサムなMolty事件：** Moltyは自分で新しいアイコンを生成するための昇格したアクセス権を与えられた。20回以上のますます呪われたロブスターの反復の後、マスコットを「5歳年上」にしようとした試みが、ロブスターの体に人間の男性の顔が付いた結果になった。仮想通貨詐欺師は数分以内にそれを「ハンサムなスクイッドワード vs ハンサムなMolty」ミームに変えた。

**偽の開発者：** 詐欺師は「OpenClawのエンジニアリング責任者」を名乗る偽のGitHubプロファイルを作成し、ポンプ・アンド・ダンプトークンを宣伝した。

混沌を見守るPeter：*「これは映画だ」* 🎬

脱皮は混沌としていた。しかしロブスターはより強く生まれ変わった。そしてより面白くなった。
