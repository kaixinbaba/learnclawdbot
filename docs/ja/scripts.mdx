# スクリプト

`scripts/` ディレクトリには、ローカルワークフローおよび運用タスク用のヘルパースクリプトが含まれています。
タスクが明確にスクリプトに関連している場合はこれらを使用してください。それ以外の場合は CLI を優先してください。

## 規約

- スクリプトはドキュメントやリリースチェックリストで参照されていない限り**オプション**です。
- CLI サーフェスが存在する場合はそちらを優先してください（例：認証監視は `openclaw models status --check` を使用します）。
- スクリプトはホスト固有であると想定してください。新しいマシンで実行する前に読んでください。

## Git フック

- `scripts/setup-git-hooks.js`：git リポジトリ内で `core.hooksPath` のベストエフォートセットアップ。
- `scripts/format-staged.js`：ステージングされた `src/` および `test/` ファイル用のプレコミットフォーマッタ。

## スクリプトを追加する場合

- スクリプトは焦点を絞り、ドキュメント化してください。
- 関連するドキュメントに短いエントリを追加してください（存在しない場合は作成してください）。
