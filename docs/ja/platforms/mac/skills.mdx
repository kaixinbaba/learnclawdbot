# Skills（macOS）

macOS アプリは、ゲートウェイ経由で OpenClaw スキルを表示します。ローカルでスキルを解析しません。

## データソース
- `skills.status`（ゲートウェイ）は、すべてのスキルに加えて、適格性と不足している要件を返します
  （バンドルされたスキルの許可リストブロックを含む）。
- 要件は各 `SKILL.md` の `metadata.openclaw.requires` から派生します。

## インストールアクション
- `metadata.openclaw.install` はインストールオプション（brew/node/go/uv）を定義します。
- アプリは `skills.install` を呼び出して、ゲートウェイホストでインストーラーを実行します。
- ゲートウェイは、複数が提供されている場合、1つの優先インストーラーのみを表示します
  （利用可能な場合は brew、そうでない場合は `skills.install` からの node マネージャー、デフォルト npm）。

## 環境変数/API キー
- アプリは `~/.openclaw/openclaw.json` の `skills.entries.<skillKey>` にキーを保存します。
- `skills.update` は `enabled`, `apiKey`, `env` をパッチします。

## リモートモード
- インストール + 設定更新はゲートウェイホスト（ローカル Mac ではない）で行われます。
