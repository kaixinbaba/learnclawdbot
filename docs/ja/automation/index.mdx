# 自動化

OpenClawは、プロアクティブで自律的なエージェントを構築するための強力な自動化機能を提供します。

## トピック

- **[Cronジョブ](cron-jobs)** - 特定の時刻にタスクをスケジュール実行
- **[ハートビート](poll)** - 定期的なチェックとプロアクティブ監視
- **[Cron vs ハートビート](cron-vs-heartbeat)** - 各自動化手法をいつ使用するか
- **[Gmail Pub/Sub](gmail-pubsub)** - Google Pub/Subによるリアルタイムメール監視
- **[認証監視](auth-monitoring)** - 認証状態とイベントの追跡

## はじめに

まず[Cronジョブ](cron-jobs)で一度限りまたは定期的なタスクのスケジュール方法を学び、次に[ハートビート](poll)でプロアクティブ監視パターンの実装を探索してください。
