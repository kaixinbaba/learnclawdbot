# Vercel AI Gateway

[Vercel AI Gateway](https://vercel.com/ai-gateway) は、単一のエンドポイントを介して数百のモデルにアクセスできる統合 API を提供します。

- プロバイダ: `vercel-ai-gateway`
- 認証: `AI_GATEWAY_API_KEY`
- API: Anthropic Messages 互換

## クイックスタート

1) API キーを設定します（推奨: ゲートウェイ用に保存）：

```bash
openclaw onboard --auth-choice ai-gateway-api-key
```

2) デフォルトモデルを設定：

```json5
{
  agents: {
    defaults: {
      model: { primary: "vercel-ai-gateway/anthropic/claude-opus-4.5" }
    }
  }
}
```

## 非対話的な例

```bash
openclaw onboard --non-interactive \
  --mode local \
  --auth-choice ai-gateway-api-key \
  --ai-gateway-api-key "$AI_GATEWAY_API_KEY"
```

## 環境に関する注意

ゲートウェイがデーモン（launchd/systemd）として実行される場合、`AI_GATEWAY_API_KEY` がそのプロセスで利用可能であることを確認してください（例：`~/.openclaw/.env` または `env.shellEnv` 経由）。
