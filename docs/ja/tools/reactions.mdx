# リアクションツール

チャンネル間で共有されるリアクションセマンティクス：

- リアクションを追加する際には `emoji` が必須です。
- `emoji=""` はサポートされている場合、ボットのリアクションを削除します。
- `remove: true` はサポートされている場合、指定された絵文字を削除します（`emoji` が必要）。

チャンネル別の注意点：

- **Discord/Slack**: 空の `emoji` はメッセージ上のボットのすべてのリアクションを削除します。`remove: true` はその絵文字のみを削除します。
- **Google Chat**: 空の `emoji` はメッセージ上のアプリのリアクションを削除します。`remove: true` はその絵文字のみを削除します。
- **Telegram**: 空の `emoji` はボットのリアクションを削除します。`remove: true` もリアクションを削除しますが、ツールのバリデーションには空でない `emoji` が必要です。
- **WhatsApp**: 空の `emoji` はボットのリアクションを削除します。`remove: true` は空の絵文字にマッピングされます（ただし `emoji` は必要）。
- **Signal**: `channels.signal.reactionNotifications` が有効な場合、インバウンドリアクション通知はシステムイベントを発行します。
