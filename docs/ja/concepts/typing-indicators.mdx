# タイピングインジケーター

タイピングインジケーターは、実行中にチャットチャンネルに送信されます。`agents.defaults.typingMode`を使用して**いつ**タイピングが開始されるかを制御し、`typingIntervalSeconds`を使用して**どのくらいの頻度**で更新されるかを制御します。

## デフォルト
`agents.defaults.typingMode`が**未設定**の場合、OpenClawはレガシー動作を維持します：
- **ダイレクトチャット**: モデルループが開始されるとすぐにタイピングが開始されます。
- **メンション付きグループチャット**: すぐにタイピングが開始されます。
- **メンションなしグループチャット**: メッセージテキストのストリーミングが開始されたときのみタイピングが開始されます。
- **ハートビート実行**: タイピングは無効です。

## モード
`agents.defaults.typingMode`を次のいずれかに設定します：
- `never` — タイピングインジケーターを一切表示しません。
- `instant` — **モデルループが開始されるとすぐに**タイピングを開始します。実行が後でサイレント返信トークンのみを返す場合でも表示されます。
- `thinking` — **最初の推論デルタ**でタイピングを開始します（実行に`reasoningLevel: "stream"`が必要）。
- `message` — **最初の非サイレントテキストデルタ**でタイピングを開始します（`NO_REPLY`サイレントトークンを無視）。

「どれだけ早く発火するか」の順序：
`never` → `message` → `thinking` → `instant`

## 設定
```json5
{
  agent: {
    typingMode: "thinking",
    typingIntervalSeconds: 6
  }
}
```

セッションごとにモードまたはケーデンスを上書きできます：
```json5
{
  session: {
    typingMode: "message",
    typingIntervalSeconds: 4
  }
}
```

## 注意事項
- `message`モードは、サイレントのみの返信（例：出力を抑制するために使用される`NO_REPLY`トークン）ではタイピングを表示しません。
- `thinking`は、実行が推論をストリームする場合（`reasoningLevel: "stream"`）のみ発火します。モデルが推論デルタを出力しない場合、タイピングは開始されません。
- ハートビートは、モードに関係なく、タイピングを表示しません。
- `typingIntervalSeconds`は**更新ケーデンス**を制御し、開始時刻を制御しません。デフォルトは6秒です。
