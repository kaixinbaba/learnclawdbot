# コンセプト

OpenClawを支える基本的なコンセプトとアーキテクチャを理解します。

## コアコンセプト

- **[エージェント](agent)** - OpenClawエージェントとは
- **[エージェントループ](agent-loop)** - エージェントがメッセージを処理し、アクションを実行する仕組み
- **[エージェントワークスペース](agent-workspace)** - エージェントのファイルシステムとメモリ
- **[アーキテクチャ](architecture)** - システムアーキテクチャとコンポーネント
- **[チャネルルーティング](channel-routing)** - チャネル間でのメッセージルーティング
- **[セッション](sessions)** - 会話セッションとコンテキストの管理
- **[思考と推論](thinking-reasoning)** - エージェントの意思決定プロセス
- **[Markdownフォーマット](markdown-formatting)** - クロスプラットフォームメッセージフォーマット
