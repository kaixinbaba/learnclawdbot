# TOOLS.md - ローカルメモ

Skills はツールの動作方法を定義します。このファイルはあなた固有の情報——あなたのセットアップに特有のもの用です。

## ここに記録すること

以下のようなもの：

- カメラ名と場所
- SSH ホストとエイリアス
- TTS の好みの声
- スピーカー/部屋名
- デバイスのニックネーム
- 環境固有の情報

## 例

### カメラ

- `living-room` → メインエリア、180° 広角
- `front-door` → 入口、モーション検知

### SSH

- `home-server` → 192.168.1.100, ユーザー: admin

### TTS

- 好みの声：「Nova」（暖かい、少し英国風）
- デフォルトスピーカー：キッチン HomePod

## なぜ分ける？

Skills は共有されます。あなたのセットアップはあなたのものです。分けておくことで、メモを失わずに skills を更新でき、インフラを漏らさずに skills を共有できます。

仕事に役立つものを何でも追加してください。これはあなたのチートシートです。
