# `openclaw logs`

RPC経由でGatewayファイルログをテールします（リモートモードで動作します）。

関連:
- ロギング概要: [Logging](/logging)

## 例

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
