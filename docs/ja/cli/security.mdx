# `openclaw security`

セキュリティツール（監査 + オプション修正）。

関連項目：
- セキュリティガイド：[Security](/gateway/security)

## 監査

```bash
openclaw security audit
openclaw security audit --deep
openclaw security audit --fix
```

監査は、複数の DM 送信者が main セッションを共有している場合に警告し、共有インボックスには `session.dmScope="per-channel-peer"`（またはマルチアカウントチャネルの場合は `per-account-channel-peer`）を推奨します。
また、小さなモデル（`<=300B`）がサンドボックスなしで使用され、web/browser ツールが有効になっている場合にも警告します。
