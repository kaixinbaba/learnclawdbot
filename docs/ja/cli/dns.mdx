# `openclaw dns`

広域ディスカバリー用のDNSヘルパー（Tailscale + CoreDNS）。現在、macOS + Homebrew CoreDNSに焦点を当てています。

関連:
- Gatewayディスカバリー: [Discovery](/gateway/discovery)
- 広域ディスカバリー設定: [Configuration](/gateway/configuration)

## セットアップ

```bash
openclaw dns setup
openclaw dns setup --apply
```
