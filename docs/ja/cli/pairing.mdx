# `openclaw pairing`

DM ペアリングリクエストを承認または検査します（ペアリングをサポートするチャネル用）。

関連項目：
- ペアリングフロー：[Pairing](/start/pairing)

## コマンド

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
