# Moonshot AI (Kimi)

Moonshot는 OpenAI 호환 엔드포인트를 갖춘 Kimi API를 제공합니다.
provider를 설정하고 기본 모델을 `moonshot/kimi-k2.5`로 설정하거나,
Kimi Code를 `kimi-code/kimi-for-coding`과 함께 사용하세요.

현재 Kimi K2 모델 ID:
{/* moonshot-kimi-k2-ids:start */}
- `kimi-k2.5`
- `kimi-k2-0905-preview`
- `kimi-k2-turbo-preview`
- `kimi-k2-thinking`
- `kimi-k2-thinking-turbo`
{/* moonshot-kimi-k2-ids:end */}

```bash
openclaw onboard --auth-choice moonshot-api-key
```

Kimi Code:

```bash
openclaw onboard --auth-choice kimi-code-api-key
```

참고: Moonshot와 Kimi Code는 별도의 provider입니다. 키는 서로 호환되지 않으며, 엔드포인트가 다르고, 모델 참조도 다릅니다(Moonshot는 `moonshot/...`, Kimi Code는 `kimi-code/...` 사용).

## Kimi Code

```json5
{
  env: { KIMICODE_API_KEY: "sk-..." },
  agents: {
    defaults: {
      model: { primary: "kimi-code/kimi-for-coding" },
      models: {
        "kimi-code/kimi-for-coding": { alias: "Kimi Code" }
      }
    }
  },
  models: {
    mode: "merge",
    providers: {
      "kimi-code": {
        baseUrl: "https://api.kimi.com/coding/v1",
        apiKey: "${KIMICODE_API_KEY}",
        api: "openai-completions",
        models: [
          {
            id: "kimi-for-coding",
            name: "Kimi For Coding",
            reasoning: true,
            input: ["text"],
            cost: { input: 0, output: 0, cacheRead: 0, cacheWrite: 0 },
            contextWindow: 262144,
            maxTokens: 32768,
            headers: { "User-Agent": "KimiCLI/0.77" },
            compat: { supportsDeveloperRole: false }
          }
        ]
      }
    }
  }
}
```

## 참고사항

- Moonshot 모델 참조는 `moonshot/<modelId>` 형식을 사용합니다. Kimi Code 모델 참조는 `kimi-code/<modelId>` 형식을 사용합니다.
- 필요한 경우 `models.providers`에서 가격 및 컨텍스트 메타데이터를 재정의하세요.
- Moonshot가 모델에 대해 다른 컨텍스트 제한을 게시하는 경우
  `contextWindow`를 적절히 조정하세요.
- 중국 엔드포인트가 필요한 경우 `https://api.moonshot.cn/v1`을 사용하세요.
