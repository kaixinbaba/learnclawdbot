# Z.AI

Z.AI는 **GLM** 모델을 위한 API 플랫폼입니다. GLM을 위한 REST API를 제공하고 인증을 위해 API 키를 사용합니다. Z.AI 콘솔에서 API 키를 생성하세요. OpenClaw는 Z.AI API 키와 함께 `zai` provider를 사용합니다.

## CLI 설정

```bash
openclaw onboard --auth-choice zai-api-key
# 또는 비대화형
openclaw onboard --zai-api-key "$ZAI_API_KEY"
```

## Config 예시

```json5
{
  env: { ZAI_API_KEY: "sk-..." },
  agents: { defaults: { model: { primary: "zai/glm-4.7" } } }
}
```

## 참고사항

- GLM 모델은 `zai/<model>` 형식으로 사용할 수 있습니다 (예: `zai/glm-4.7`).
- 모델 패밀리 개요는 [/providers/glm](/providers/glm)을 참조하세요.
- Z.AI는 API 키와 함께 Bearer 인증을 사용합니다.
