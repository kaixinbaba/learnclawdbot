# GLM 모델

GLM은 Z.AI 플랫폼을 통해 제공되는 **모델 패밀리**입니다(회사가 아닙니다). OpenClaw에서
GLM 모델은 `zai` provider와 `zai/glm-4.7`과 같은 model ID를 통해 접근합니다.

## CLI 설정

```bash
openclaw onboard --auth-choice zai-api-key
```

## Config 예시

```json5
{
  env: { ZAI_API_KEY: "sk-..." },
  agents: { defaults: { model: { primary: "zai/glm-4.7" } } }
}
```

## 참고사항

- GLM 버전과 가용성은 변경될 수 있습니다. 최신 정보는 Z.AI의 문서를 확인하세요.
- 예시 model ID로는 `glm-4.7` 및 `glm-4.6`이 있습니다.
- provider 세부 정보는 [/providers/zai](/providers/zai)를 참조하세요.
