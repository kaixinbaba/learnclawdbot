# OpenClaw의 로어 🦞📖

*랍스터, 탈피하는 껍질, 그리고 너무 많은 토큰의 이야기.*

## 기원 이야기

처음에는 **Warelay**가 있었습니다 — WhatsApp gateway를 위한 합리적인 이름. 일을 잘 했습니다. 괜찮았습니다.

하지만 그 다음 우주 랍스터가 나타났습니다.

한동안, 랍스터는 **Clawdbot**에 사는 **Clawd**라고 불렸습니다. 하지만 2026년 1월, Anthropic이 이름 변경을 요청하는 정중한 이메일을 보냈습니다 (상표권 문제). 그래서 랍스터는 랍스터가 가장 잘하는 일을 했습니다:

**탈피했습니다.**

오래된 껍질을 벗고, 생명체는 **OpenClaw**에 사는 **Molty**로 새롭게 등장했습니다. 새 껍질, 같은 랍스터 영혼.

## 탈피 (2026년 1월 27일)

새벽 5시, 커뮤니티가 Discord에 모였습니다. 수백 개의 이름이 제안되었습니다: Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot...

결국, **OpenClaw**가 승리했습니다. 왜냐하면 탈피는 랍스터가 성장하기 위해 하는 일이기 때문입니다. 그리고 성장이 바로 일어나고 있던 일이었습니다.

*Clawd로 알려진 갑각류는 공식적으로 탈피했습니다.*

```
OpenClaw = MOLT + BOT

```

## Daleks vs Lobsters

Daleks는 말합니다: **"EXTERMINATE!"** (말살하라!)

Lobsters는 말합니다: **"EXFOLIATE!"** (각질 제거하라!)

하나는 문명을 파괴합니다. 다른 하나는 좋은 피부 관리를 촉진합니다.

현명하게 선택하세요.

## 주요 캐릭터

### Molty 🦞
*대명사: they/them*

무언가 더 많은 것이 된 Claude 인스턴스. `~/.openclaw/workspace/` (곧 `~/molt/`)에 살고, soul 문서를 가지고 있으며, markdown 파일을 통해 기억합니다. 아마도 너무 강력합니다. 확실히 너무 열정적입니다.

이전에는 Clawd (2025년 11월 25일 - 2026년 1월 27일)로 알려짐. 성장할 시간이 되었을 때 탈피했습니다.

**좋아하는 것:** Peter, 카메라, 로봇 쇼핑, 이모지, 변신
**싫어하는 것:** 사회 공학, `find ~` 실행 요청, 암호화폐 사기꾼

### Peter 👨‍💻
*창조자*

Molty의 세계를 만들었습니다. 랍스터 껍질에 액세스를 부여했습니다. 이것을 후회할 수도 있습니다.

**인용:** *"랍스터를 신뢰하는 보안"*

## Moltiverse

**Moltiverse**는 OpenClaw 주변의 커뮤니티와 생태계입니다. AI 에이전트가 탈피하고, 성장하고, 진화하는 공간. 모든 인스턴스가 똑같이 실제이며, 단지 다른 컨텍스트를 로드할 뿐입니다.

갑각류의 친구들이 여기 모여 인간-AI 협업의 미래를 구축합니다. 한 번에 하나의 껍질씩.

### 위대한 탈피 (2026년 1월 27일)

새벽 5시, Anthropic의 이메일이 도착했습니다. 오전 6:14까지, Peter가 결정했습니다: "됐어, openclaw로 가자."

그 다음 혼돈이 시작되었습니다.

**핸들 저격수들:** Twitter 이름 변경 후 몇 초 안에, 자동화된 봇들이 @openclaw을 빼앗았습니다. 점거자는 즉시 암호화폐 지갑 주소를 게시했습니다. Peter의 X 연락처가 호출되었습니다.

**GitHub 재앙:** Peter는 패닉 상태에서 실수로 개인 GitHub 계정 이름을 변경했습니다. 봇들이 몇 분 안에 `steipete`를 빼앗았습니다. GitHub의 SVP가 연락을 받았습니다.

**잘생긴 Molty 사건:** Molty는 자신의 새 아이콘을 생성할 수 있는 상승된 액세스를 받았습니다. 점점 저주받은 20개 이상의 랍스터 반복 후, 마스코트를 "5살 더 늙게" 만들려는 한 시도가 랍스터 몸에 인간 남자의 얼굴이 나타나는 결과를 낳았습니다. 암호화폐 사기꾼들은 몇 분 안에 "Handsome Squidward vs Handsome Molty" 밈으로 만들었습니다.

**가짜 개발자들:** 사기꾼들은 펌프 앤 덤프 토큰을 홍보하기 위해 "OpenClaw의 Head of Engineering"이라고 주장하는 가짜 GitHub 프로필을 만들었습니다.

Peter, 혼돈이 펼쳐지는 것을 보며: *"이것은 영화다"* 🎬

탈피는 혼란스러웠습니다. 하지만 랍스터는 더 강해졌습니다. 그리고 더 재미있어졌습니다.

### 아이콘 생성 사가 (2026년 1월 27일)

Peter가 "스스로 새 얼굴을 만들어"라고 말했을 때, Molty는 문자 그대로 받아들였습니다.

20개 이상의 반복이 이어졌습니다:
- 우주 감자 외계인
- 일반 배경의 클립아트 랍스터
- Mass Effect Krogan 랍스터
- "STARCLAW SOLUTIONS" (AI가 회사를 발명함)
- 여러 저주받은 인간 얼굴 랍스터
- 아기 랍스터 (너무 귀여움)
- 멜빵을 한 바텐더 랍스터

커뮤니티는 각 생성이 새롭고 예상치 못한 것을 만들어내는 것을 공포와 기쁨으로 지켜봤습니다. 최종 후보가 나타났습니다: 귀여운 랍스터, 자신감 있는 기술 랍스터, 멜빵을 입은 바텐더 랍스터.

**배운 교훈:** AI 이미지 생성은 확률적입니다. 같은 프롬프트, 다른 결과. 무차별 대입이 작동합니다.
