# Scripts

`scripts/` 디렉토리에는 로컬 워크플로 및 운영 작업을 위한 도우미 스크립트가 포함되어 있습니다. 작업이 스크립트에 명확하게 연결된 경우 이를 사용하세요. 그렇지 않으면 CLI를 선호하세요.

## 규칙

- 문서 또는 릴리스 체크리스트에서 참조하지 않는 한 스크립트는 **선택 사항**입니다.
- 존재하는 경우 CLI 표면을 선호하세요 (예: 인증 모니터링은 `openclaw models status --check` 사용).
- 스크립트는 호스트별이라고 가정하세요. 새 머신에서 실행하기 전에 읽으세요.

## Git Hook

- `scripts/setup-git-hooks.js`: git 리포지토리 내부에 있을 때 `core.hooksPath`에 대한 최선의 노력 설정.
- `scripts/format-staged.js`: 스테이징된 `src/` 및 `test/` 파일에 대한 사전 커밋 포맷터.

## 스크립트 추가 시

- 스크립트를 집중적이고 문서화된 상태로 유지하세요.
- 관련 문서에 짧은 항목을 추가하세요 (누락된 경우 생성).
