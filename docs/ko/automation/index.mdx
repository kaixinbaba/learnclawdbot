# 자동화

OpenClaw는 프로액티브하고 자율적인 에이전트를 구축하기 위한 강력한 자동화 기능을 제공합니다.

## 주제

- **[크론 작업](cron-jobs)** - 특정 시간에 작업 예약
- **[하트비트](poll)** - 주기적인 확인 및 프로액티브 모니터링
- **[크론 vs 하트비트](cron-vs-heartbeat)** - 각 자동화 방법을 사용할 시기
- **[Gmail Pub/Sub](gmail-pubsub)** - Google Pub/Sub을 사용한 실시간 이메일 모니터링
- **[인증 모니터링](auth-monitoring)** - 인증 상태 및 이벤트 추적

## 시작하기

먼저 [크론 작업](cron-jobs)으로 일회성 또는 반복 작업을 예약하는 방법을 배우고, [하트비트](poll)로 프로액티브 모니터링 패턴 구현을 탐색하세요.
