# 사용량 추적

## 무엇인가
- Provider의 사용량 endpoint에서 직접 provider 사용량/할당량을 가져옵니다.
- 추정 비용 없음; provider가 보고한 윈도우만.

## 표시되는 위치
- 채팅의 `/status`: session 토큰 + 추정 비용(API key만)이 있는 이모지가 풍부한 상태 카드. Provider 사용량은 사용 가능한 경우 **현재 model provider**에 대해 표시됩니다.
- 채팅의 `/usage off|tokens|full`: 응답별 사용량 footer(OAuth는 토큰만 표시).
- 채팅의 `/usage cost`: OpenClaw session 로그에서 집계된 로컬 비용 요약.
- CLI: `openclaw status --usage`는 provider별 전체 분석을 출력합니다.
- CLI: `openclaw channels list`는 provider config와 함께 동일한 사용량 스냅샷을 출력합니다(`--no-usage`를 사용하여 건너뛰기).
- macOS 메뉴 바: Context 아래 "Usage" 섹션(사용 가능한 경우에만).

## Provider + 인증 정보
- **Anthropic (Claude)**: auth 프로필의 OAuth 토큰.
- **GitHub Copilot**: auth 프로필의 OAuth 토큰.
- **Gemini CLI**: auth 프로필의 OAuth 토큰.
- **Antigravity**: auth 프로필의 OAuth 토큰.
- **OpenAI Codex**: auth 프로필의 OAuth 토큰(존재하는 경우 accountId 사용).
- **MiniMax**: API key (코딩 플랜 키; `MINIMAX_CODE_PLAN_KEY` 또는 `MINIMAX_API_KEY`); 5시간 코딩 플랜 윈도우 사용.
- **z.ai**: env/config/auth store를 통한 API key.

일치하는 OAuth/API 인증 정보가 없으면 사용량이 숨겨집니다.
