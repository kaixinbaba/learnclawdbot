# Model provider

이 페이지는 **LLM/model provider**를 다룹니다 (WhatsApp/Telegram과 같은 채팅 channel이 아님).
Model 선택 규칙은 [/concepts/models](/concepts/models) 참조.

## 빠른 규칙

- Model ref는 `provider/model`을 사용합니다 (예: `opencode/claude-opus-4-5`).
- `agents.defaults.models`를 설정하면 allowlist가 됩니다.
- CLI 헬퍼: `openclaw onboard`, `openclaw models list`, `openclaw models set <provider/model>`.

## 내장 provider (pi-ai catalog)

OpenClaw는 pi‑ai catalog과 함께 제공됩니다. 이러한 provider는 `models.providers` config가 **필요 없습니다**; auth를 설정하고 model을 선택하기만 하면 됩니다.

### OpenAI

- Provider: `openai`
- Auth: `OPENAI_API_KEY`
- 예제 model: `openai/gpt-5.2`
- CLI: `openclaw onboard --auth-choice openai-api-key`

```json5
{
  agents: { defaults: { model: { primary: "openai/gpt-5.2" } } }
}
```

### Anthropic

- Provider: `anthropic`
- Auth: `ANTHROPIC_API_KEY` 또는 `claude setup-token`
- 예제 model: `anthropic/claude-opus-4-5`
- CLI: `openclaw onboard --auth-choice token` (setup-token 붙여넣기) 또는 `openclaw models auth paste-token --provider anthropic`

```json5
{
  agents: { defaults: { model: { primary: "anthropic/claude-opus-4-5" } } }
}
```

더 많은 provider 및 설정 예제는 영문 원본을 참조하세요 ([Model providers](/concepts/model-providers)).
