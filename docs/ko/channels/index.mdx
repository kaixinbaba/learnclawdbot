# 채팅 채널

OpenClaw는 이미 사용 중인 모든 채팅 앱에서 대화할 수 있습니다. 각 채널은 Gateway를 통해 연결됩니다.
텍스트는 모든 곳에서 지원되며, 미디어와 반응은 채널에 따라 다릅니다.

## 지원되는 채널

- [WhatsApp](/channels/whatsapp) — 가장 인기 있음; Baileys를 사용하며 QR 페어링이 필요합니다.
- [Telegram](/channels/telegram) — grammY를 통한 Bot API; 그룹을 지원합니다.
- [Discord](/channels/discord) — Discord Bot API + Gateway; 서버, 채널, DM을 지원합니다.
- [Slack](/channels/slack) — Bolt SDK; 워크스페이스 앱.
- [Google Chat](/channels/googlechat) — HTTP webhook을 통한 Google Chat API 앱.
- [Mattermost](/channels/mattermost) — Bot API + WebSocket; 채널, 그룹, DM (별도 설치되는 플러그인).
- [Signal](/channels/signal) — signal-cli; 프라이버시 중심.
- [BlueBubbles](/channels/bluebubbles) — **iMessage 권장**; BlueBubbles macOS 서버 REST API를 사용하며 전체 기능 지원 (편집, 전송 취소, 효과, 반응, 그룹 관리 — 편집은 현재 macOS 26 Tahoe에서 작동하지 않음).
- [iMessage](/channels/imessage) — macOS 전용; imsg를 통한 네이티브 통합 (레거시, 새로운 설정에는 BlueBubbles 권장).
- [Microsoft Teams](/channels/msteams) — Bot Framework; 엔터프라이즈 지원 (별도 설치되는 플러그인).
- [LINE](/channels/line) — LINE Messaging API 봇 (별도 설치되는 플러그인).
- [Nextcloud Talk](/channels/nextcloud-talk) — Nextcloud Talk를 통한 자체 호스팅 채팅 (별도 설치되는 플러그인).
- [Matrix](/channels/matrix) — Matrix 프로토콜 (별도 설치되는 플러그인).
- [Nostr](/channels/nostr) — NIP-04를 통한 탈중앙화 DM (별도 설치되는 플러그인).
- [Tlon](/channels/tlon) — Urbit 기반 메신저 (별도 설치되는 플러그인).
- [Twitch](/channels/twitch) — IRC 연결을 통한 Twitch 채팅 (별도 설치되는 플러그인).
- [Zalo](/channels/zalo) — Zalo Bot API; 베트남의 인기 메신저 (별도 설치되는 플러그인).
- [Zalo Personal](/channels/zalouser) — QR 로그인을 통한 Zalo 개인 계정 (별도 설치되는 플러그인).
- [WebChat](/web/webchat) — WebSocket을 통한 Gateway WebChat UI.

## 참고사항

- 채널은 동시에 실행할 수 있으며, 여러 개를 구성하면 OpenClaw가 채팅별로 라우팅합니다.
- 가장 빠른 설정은 일반적으로 **Telegram** (간단한 봇 토큰)입니다. WhatsApp은 QR 페어링이 필요하며
  디스크에 더 많은 상태를 저장합니다.
- 그룹 동작은 채널마다 다릅니다. [Groups](/concepts/groups)를 참조하세요.
- 안전을 위해 DM 페어링 및 허용목록이 적용됩니다. [Security](/gateway/security)를 참조하세요.
- Telegram 내부 구조: [grammY 참고사항](/channels/grammy).
- 문제 해결: [채널 문제 해결](/channels/troubleshooting).
- 모델 제공자는 별도로 문서화되어 있습니다. [Model Providers](/providers/models)를 참조하세요.
