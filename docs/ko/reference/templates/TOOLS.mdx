# TOOLS.md - 로컬 노트

Skills는 도구가 어떻게 작동하는지 정의합니다. 이 파일은 당신의 구체적인 사항——당신의 설정에 고유한 것들을 위한 것입니다.

## 여기에 기록할 내용

다음과 같은 것들：

- 카메라 이름과 위치
- SSH 호스트 및 별칭
- TTS 선호 음성
- 스피커/방 이름
- 장치 별명
- 환경별 정보

## 예시

### 카메라

- `living-room` → 메인 영역, 180° 광각
- `front-door` → 입구, 동작 감지

### SSH

- `home-server` → 192.168.1.100, 사용자: admin

### TTS

- 선호 음성: "Nova" (따뜻함, 약간 영국식)
- 기본 스피커: 주방 HomePod

## 왜 분리하나요?

Skills는 공유됩니다. 당신의 설정은 당신의 것입니다. 분리해두면 노트를 잃지 않고 skills를 업데이트할 수 있고, 인프라를 노출하지 않고 skills를 공유할 수 있습니다.

일을 하는 데 도움이 되는 것이라면 무엇이든 추가하세요. 이것은 당신의 치트시트입니다.
