# `openclaw onboard`

대화형 온보딩 마법사 (로컬 또는 원격 Gateway 설정).

관련 문서:
- 마법사 가이드: [Onboarding](/start/onboarding)

## 예제

```bash
openclaw onboard
openclaw onboard --flow quickstart
openclaw onboard --flow manual
openclaw onboard --mode remote --remote-url ws://gateway-host:18789
```

플로우 참고:
- `quickstart`: 최소한의 프롬프트, gateway token 자동 생성.
- `manual`: port/bind/auth에 대한 전체 프롬프트 (`advanced`의 별칭).
- 가장 빠른 첫 채팅: `openclaw dashboard` (Control UI, channel 설정 불필요).
