# `openclaw security`

보안 도구 (감사 + 선택적 수정).

관련 문서:
- 보안 가이드: [Security](/gateway/security)

## Audit

```bash
openclaw security audit
openclaw security audit --deep
openclaw security audit --fix
```

감사는 여러 DM 발신자가 main session을 공유할 때 경고하고 공유 inbox에 대해 `session.dmScope="per-channel-peer"` (또는 다중 계정 channel의 경우 `per-account-channel-peer`)를 권장합니다.
또한 작은 model(`<=300B`)이 샌드박스 없이 web/browser tool이 활성화된 상태로 사용될 때 경고합니다.
