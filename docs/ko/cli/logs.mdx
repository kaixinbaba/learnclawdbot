# `openclaw logs`

RPC를 통해 Gateway 파일 로그를 tail합니다 (원격 모드에서 작동).

관련 문서:
- 로깅 개요: [Logging](/logging)

## 예제

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
