# `openclaw skills`

skill(번들 + workspace + 관리형 재정의)을 검사하고 적격 vs 요구 사항 누락을 확인합니다.

관련 문서:
- Skill 시스템: [Skills](/tools/skills)
- Skill config: [Skills config](/tools/skills-config)
- ClawdHub 설치: [ClawdHub](/tools/clawdhub)

## 명령

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
