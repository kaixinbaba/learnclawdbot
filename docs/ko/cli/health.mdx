# `openclaw health`

실행 중인 Gateway에서 상태를 가져옵니다.

```bash
openclaw health
openclaw health --json
openclaw health --verbose
```

참고사항:
- `--verbose`는 실시간 조사를 실행하고 여러 계정이 구성된 경우 계정별 타이밍을 출력합니다.
- 출력에는 여러 에이전트가 구성된 경우 에이전트별 세션 저장소가 포함됩니다.
