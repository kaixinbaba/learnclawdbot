# `openclaw dns`

광역 디스커버리(Tailscale + CoreDNS)를 위한 DNS 헬퍼입니다. 현재 macOS + Homebrew CoreDNS에 중점을 두고 있습니다.

관련 항목:
- Gateway 디스커버리: [Discovery](/gateway/discovery)
- 광역 디스커버리 구성: [Configuration](/gateway/configuration)

## 설정

```bash
openclaw dns setup
openclaw dns setup --apply
```
