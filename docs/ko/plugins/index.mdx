# 플러그인

플러그인과 사용자 정의 도구로 OpenClaw의 기능을 확장합니다.

## 사용 가능한 플러그인

- **[플러그인 매니페스트](manifest)** - 플러그인 메타데이터 및 요구 사항 정의
- **[에이전트 도구](agent-tools)** - 에이전트용 사용자 정의 도구 생성
- **[음성 통화 플러그인](voice-call)** - 음성 통화 기능 추가
- **[Zalo 사용자 플러그인](zalouser)** - Zalo 메시징과의 통합

## 플러그인 생성

플러그인을 사용하면 OpenClaw에 새로운 도구, 통합 및 기능을 추가할 수 있습니다. 플러그인 구조에 대해 알아보려면 [플러그인 매니페스트](manifest) 문서부터 시작하세요.
