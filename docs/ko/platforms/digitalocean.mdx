# DigitalOcean에서 OpenClaw 실행

## 목표

**월 $6**(또는 예약 가격으로 $4/월)로 DigitalOcean에서 지속적인 OpenClaw Gateway를 실행합니다.

월 $0 옵션을 원하고 ARM + 제공업체별 설정이 괜찮다면, [Oracle Cloud 가이드](/platforms/oracle)를 참조하세요.

## 비용 비교 (2026)

| 제공업체 | 플랜 | 사양 | 월 가격 | 비고 |
|----------|------|-------|----------|-------|
| Oracle Cloud | Always Free ARM | 최대 4 OCPU, 24GB RAM | $0 | ARM, 제한된 용량 / 가입 이슈 |
| Hetzner | CX22 | 2 vCPU, 4GB RAM | €3.79 (~$4) | 가장 저렴한 유료 옵션 |
| DigitalOcean | Basic | 1 vCPU, 1GB RAM | $6 | 쉬운 UI, 좋은 문서 |
| Vultr | Cloud Compute | 1 vCPU, 1GB RAM | $6 | 많은 위치 |
| Linode | Nanode | 1 vCPU, 1GB RAM | $5 | 현재 Akamai의 일부 |

**제공업체 선택:**
- DigitalOcean: 가장 간단한 UX + 예측 가능한 설정 (이 가이드)
- Hetzner: 좋은 가격/성능 ([Hetzner 가이드](/platforms/hetzner) 참조)
- Oracle Cloud: 월 $0일 수 있지만 더 까다롭고 ARM 전용 ([Oracle 가이드](/platforms/oracle) 참조)

## 전제 조건

- DigitalOcean 계정 ([가입 시 $200 무료 크레딧](https://m.do.co/c/signup))
- SSH 키 페어 (또는 비밀번호 인증 사용 의향)
- ~20분

## 1) Droplet 생성

1. [DigitalOcean](https://cloud.digitalocean.com/)에 로그인
2. **Create → Droplets** 클릭
3. 선택:
   - **Region:** 사용자(또는 사용자들)와 가장 가까운 위치
   - **Image:** Ubuntu 24.04 LTS
   - **Size:** Basic → Regular → **$6/mo** (1 vCPU, 1GB RAM, 25GB SSD)
   - **Authentication:** SSH 키 (권장) 또는 비밀번호
4. **Create Droplet** 클릭
5. IP 주소 기록

## 2) SSH를 통해 연결

```bash
ssh root@YOUR_DROPLET_IP
```

## 3) OpenClaw 설치

```bash
# 시스템 업데이트
apt update && apt upgrade -y

# Node.js 22 설치
curl -fsSL https://deb.nodesource.com/setup_22.x | bash -
apt install -y nodejs

# OpenClaw 설치
curl -fsSL https://openclaw.bot/install.sh | bash

# 확인
openclaw --version
```

## 4) 온보딩 실행

```bash
openclaw onboard --install-daemon
```

마법사가 다음을 안내합니다:
- Model 인증 (API 키 또는 OAuth)
- Channel 설정 (Telegram, WhatsApp, Discord 등)
- Gateway token (자동 생성)
- Daemon 설치 (systemd)

## 5) Gateway 확인

```bash
# 상태 확인
openclaw status

# 서비스 확인
systemctl --user status openclaw-gateway.service

# 로그 보기
journalctl --user -u openclaw-gateway.service -f
```

## 6) 대시보드 접근

Gateway는 기본적으로 loopback에 바인드됩니다. Control UI에 접근하려면:

**옵션 A: SSH 터널 (권장)**
```bash
# 로컬 머신에서
ssh -L 18789:localhost:18789 root@YOUR_DROPLET_IP

# 그런 다음 열기: http://localhost:18789
```

**옵션 B: Tailscale Serve (HTTPS, loopback 전용)**
```bash
# Droplet에서
curl -fsSL https://tailscale.com/install.sh | sh
tailscale up

# Gateway를 Tailscale Serve를 사용하도록 구성
openclaw config set gateway.tailscale.mode serve
openclaw gateway restart
```

열기: `https://<magicdns>/`

참고사항:
- Serve는 Gateway를 loopback 전용으로 유지하고 Tailscale identity 헤더를 통해 인증합니다.
- 대신 token/password를 요구하려면, `gateway.auth.allowTailscale: false`를 설정하거나 `gateway.auth.mode: "password"`를 사용하세요.

**옵션 C: Tailnet 바인드 (Serve 없음)**
```bash
openclaw config set gateway.bind tailnet
openclaw gateway restart
```

열기: `http://<tailscale-ip>:18789` (token 필요).

## 7) Channel 연결

### Telegram
```bash
openclaw pairing list telegram

```

### WhatsApp
```bash
openclaw channels login whatsapp

```

## 1GB RAM 최적화

$6 droplet은 1GB RAM만 있습니다. 원활한 실행을 위해:

### Swap 추가 (권장)
```bash
fallocate -l 2G /swapfile
chmod 600 /swapfile
mkswap /swapfile
swapon /swapfile
echo '/swapfile none swap sw 0 0' >> /etc/fstab
```

### 더 가벼운 model 사용
OOM이 발생하는 경우, 다음을 고려하세요:
- 로컬 model 대신 API 기반 model (Claude, GPT) 사용
- `agents.defaults.model.primary`를 더 작은 model로 설정

### 메모리 모니터링
```bash
free -h
htop
```

## 지속성

모든 상태는 다음에 저장됩니다:
- `~/.openclaw/` — 구성, 자격 증명, 세션 데이터
- `~/.openclaw/workspace/` — 작업 공간 (SOUL.md, memory 등)

이들은 재부팅 후에도 유지됩니다. 주기적으로 백업하세요:
```bash
tar -czvf openclaw-backup.tar.gz ~/.openclaw ~/.openclaw/workspace
```

## Oracle Cloud 무료 대안

Oracle Cloud는 월 $0로 여기의 어떤 유료 옵션보다 훨씬 더 강력한 **Always Free** ARM 인스턴스를 제공합니다.

| 제공 내용 | 사양 |
|--------------|-------|
| **4 OCPUs** | ARM Ampere A1 |
| **24GB RAM** | 충분한 용량 |
| **200GB 스토리지** | Block volume |
| **영구 무료** | 신용카드 청구 없음 |

**주의사항:**
- 가입이 까다로울 수 있음 (실패 시 재시도)
- ARM 아키텍처 — 대부분 작동하지만 일부 바이너리는 ARM 빌드 필요

전체 설정 가이드는 [Oracle Cloud](/platforms/oracle)를 참조하세요. 가입 팁 및 등록 프로세스 문제 해결은 이 [커뮤니티 가이드](https://gist.github.com/rssnyder/51e3cfedd730e7dd5f4a816143b25dbd)를 참조하세요.

## 문제 해결

### Gateway가 시작되지 않음
```bash
openclaw gateway status
openclaw doctor --non-interactive
journalctl -u openclaw --no-pager -n 50
```

### 포트가 이미 사용 중
```bash
lsof -i :18789
kill <PID>
```

### 메모리 부족
```bash
# 메모리 확인
free -h

# Swap 추가
# 또는 $12/월 droplet으로 업그레이드 (2GB RAM)
```
