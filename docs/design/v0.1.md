# LearnClawdbot.org v0.1 MVP 设计文档

## 项目概述

**目标**: 快速上线一个第三方 Clawdbot 新手指南站，蹭 "Clawdbot" 关键词热度

**定位**: 第三方 · 新手向 · Clawdbot 使用指南站

**成功标准**:
- [ ] 访问 `https://learnclawdbot.org` 不报错
- [ ] 有首页（哪怕很丑）
- [ ] 有 `/about` 页面
- [ ] 有 `/clawdbot` 页面占位
- [ ] 有 `/privacy-policy` 页面
- [ ] Google 能访问（无登录墙）

---

## 技术栈（复用现有模板）

| 技术 | 版本 | 用途 |
|------|------|------|

## 页面结构

```
learnclawdbot.org
├── /                   # 首页
├── /about              # 关于页面
├── /clawdbot           # Clawdbot 主内容页
├── /blog               # 博客列表（可选，后续启用）
└── /privacy-policy     # 隐私政策
```

### 页面内容规划

#### 1. 首页 `/`

**H1**: `Learn Clawdbot`

**描述**:
```
Learn Clawdbot is a beginner-friendly, third-party guide that helps you
understand and use Clawdbot in simple terms.

This site is not affiliated with the official Clawdbot project.
```

**CTA**: `→ What is Clawdbot?` → 链接到 `/clawdbot`

#### 2. About 页面 `/about`

```
Learn Clawdbot is an independent, third-party website created to help
beginners understand and use Clawdbot.

This site is not affiliated with, endorsed by, or connected to the
official Clawdbot project.

All content is based on publicly available information and personal research.
```

#### 3. Clawdbot 主页 `/clawdbot`（占位版）

**H1**: `What Is Clawdbot?`

**内容**:
```
Clawdbot is a new AI agent tool designed to perform tasks rather than just chat.

This page will be updated with a full beginner-friendly guide explaining
what Clawdbot is, what it can do, and who it is for.
```

#### 4. Privacy Policy `/privacy-policy`

```
This website does not collect personal data.

If this changes in the future, this page will be updated accordingly.
```

## 需要修改的文件

### 高优先级（必须修改）

| 文件 | 修改内容 |
|------|---------|
| `config/site.ts` | 站点名称、URL、作者信息 |
| `i18n/messages/en/common.json` | Header 导航链接、Footer 链接、站点标题 |
| `i18n/messages/en/Landing.json` | 首页 Hero、Features 内容（或隐藏） |
| `public/logo.png` | 站点 Logo |
| `public/favicon.ico` | 网站图标 |

### 中优先级（隐藏功能）

| 文件/目录 | 操作 |
|----------|------|
| `components/header/UserAvatar.tsx` | 隐藏用户头像/登录按钮 |
| `components/home/` 下的定价相关组件 | 隐藏或移除 |
| `app/[locale]/(basic-layout)/login/` | 隐藏入口 |
| `app/[locale]/(basic-layout)/ai-demo/` | 隐藏入口 |
| `app/[locale]/(protected)/` | 不影响，自然隐藏（无入口） |

### 低优先级（后续处理）

| 文件 | 操作 |
|------|------|
| `package.json` | 清理无用依赖 |
| `app/api/` | 可保留，不影响前端 |
| 数据库相关 | 不删除，只是不使用 |

## 实施步骤

### Phase 1: 配置修改 (10-15 分钟)

1. **修改 `config/site.ts`**
   - name: "Learn Clawdbot"
   - url: "https://learnclawdbot.org"
   - 清空社交链接

2. **修改 `i18n/messages/en/common.json`**
   - Home.title: "Learn Clawdbot"
   - Home.tagLine: "A Beginner-Friendly Guide to Clawdbot"
   - Header.links: 新导航结构
   - Footer.Links: 新底部链接

### Phase 2: 首页改造 (15-20 分钟)

1. **修改 `i18n/messages/en/Landing.json`**
   - 简化 Hero 区域内容
   - 移除/隐藏 Features 区域
   - 移除 UseCases, Testimonials, FAQ, CTA

2. **或直接修改首页组件**
   - 创建简洁的 Hero 组件
   - 只保留核心 CTA

### Phase 3: 创建新页面 (10-15 分钟)

1. **创建 `/clawdbot` 页面**
   - 位置: `app/[locale]/(basic-layout)/clawdbot/page.tsx`
   - 复用 about 页面结构
   - 使用 MDX 内容

2. **修改 `/about` 页面内容**
   - 位置: `content/about/en.mdx`

### Phase 4: 隐藏不需要的功能 (5-10 分钟)

1. **Header 组件**
   - 移除 `UserAvatar` 组件渲染
   - 或隐藏登录相关 UI

2. **导航链接**
   - 通过 i18n 配置移除 Pricing、AI Demo 等链接

### Phase 5: 资源替换 (5 分钟)

1. **Logo**: 替换 `public/logo.png`
2. **Favicon**: 替换 `public/favicon.ico`
