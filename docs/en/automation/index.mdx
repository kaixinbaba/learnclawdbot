# Automation

OpenClaw provides powerful automation capabilities to help you build proactive, autonomous agents.

## Topics

- **[Cron Jobs](cron-jobs)** - Schedule tasks to run at specific times
- **[Heartbeats](poll)** - Periodic checks and proactive monitoring
- **[Cron vs Heartbeat](cron-vs-heartbeat)** - When to use each automation method
- **[Gmail Pub/Sub](gmail-pubsub)** - Real-time email monitoring with Google Pub/Sub
- **[Auth Monitoring](auth-monitoring)** - Track authentication status and events

## Getting Started

Start with [Cron Jobs](cron-jobs) to learn how to schedule one-time or recurring tasks, then explore [Heartbeats](poll) for implementing proactive monitoring patterns.
