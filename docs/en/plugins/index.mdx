# Plugins

Extend OpenClaw's capabilities with plugins and custom tools.

## Available Plugins

- **[Plugin Manifest](manifest)** - Define plugin metadata and requirements
- **[Agent Tools](agent-tools)** - Create custom tools for agents
- **[Voice Call Plugin](voice-call)** - Add voice calling capabilities
- **[Zalo User Plugin](zalouser)** - Integration with Zalo messaging

## Creating Plugins

Plugins allow you to add new tools, integrations, and capabilities to OpenClaw. Start with the [Plugin Manifest](manifest) documentation to learn how to structure your plugin.
