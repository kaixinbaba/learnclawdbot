# Автоматизация

OpenClaw предоставляет мощные возможности автоматизации для создания проактивных, автономных агентов.

## Темы

- **[Задачи Cron](cron-jobs)** - Планирование задач на определенное время
- **[Heartbeats](poll)** - Периодические проверки и проактивный мониторинг
- **[Cron vs Heartbeat](cron-vs-heartbeat)** - Когда использовать каждый метод автоматизации
- **[Gmail Pub/Sub](gmail-pubsub)** - Мониторинг электронной почты в реальном времени с Google Pub/Sub
- **[Мониторинг аутентификации](auth-monitoring)** - Отслеживание состояния и событий аутентификации

## Начало работы

Начните с [Задачи Cron](cron-jobs), чтобы узнать, как планировать одноразовые или повторяющиеся задачи, затем изучите [Heartbeats](poll) для реализации шаблонов проактивного мониторинга.
