Локальное доверие:
- Локальные подключения (loopback или собственный адрес tailnet хоста шлюза) могут быть
  автоматически одобрены для сопряжения, чтобы обеспечить плавный UX на том же хосте.
- Не-локальные клиенты tailnet/LAN по-прежнему требуют явного одобрения сопряжения.
