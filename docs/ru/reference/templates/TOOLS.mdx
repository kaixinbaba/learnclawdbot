# TOOLS.md - Локальные заметки

Skills определяют, как работают инструменты. Этот файл для ваших особенностей — того, что уникально для вашей установки.

## Что сюда записывать

Например:

- Названия и местоположения камер
- SSH-хосты и псевдонимы
- Предпочитаемые голоса для TTS
- Названия динамиков/комнат
- Прозвища устройств
- Любая информация, специфичная для среды

## Примеры

### Камеры

- `living-room` → Основная зона, широкий угол 180°
- `front-door` → Вход, активация по движению

### SSH

- `home-server` → 192.168.1.100, пользователь: admin

### TTS

- Предпочитаемый голос: «Nova» (теплый, слегка британский)
- Динамик по умолчанию: кухонный HomePod

## Почему отдельно?

Skills общие. Ваша установка — ваша. Разделение означает, что вы можете обновлять skills, не теряя заметки, и делиться skills, не раскрывая свою инфраструктуру.

Добавляйте все, что помогает вам в работе. Это ваша шпаргалка.
