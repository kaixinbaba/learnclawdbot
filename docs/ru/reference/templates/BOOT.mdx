# BOOT.md

Добавьте короткие, явные инструкции для того, что OpenClaw должен делать при запуске (включите `hooks.internal.enabled`).

Если задача отправляет сообщение, используйте инструмент `message`, а затем ответьте `NO_REPLY`.
