# Плагины

Расширение возможностей OpenClaw с помощью плагинов и пользовательских инструментов.

## Доступные плагины

- **[Манифест плагина](manifest)** - Определение метаданных и требований плагина
- **[Инструменты агента](agent-tools)** - Создание пользовательских инструментов для агентов
- **[Плагин голосовых вызовов](voice-call)** - Добавление возможности голосовых вызовов
- **[Плагин пользователя Zalo](zalouser)** - Интеграция с мессенджером Zalo

## Создание плагинов

Плагины позволяют добавлять новые инструменты, интеграции и возможности в OpenClaw. Начните с документации [Манифест плагина](manifest), чтобы узнать, как структурировать ваш плагин.
