# Инструментарий реакций

Общая семантика реакций для всех каналов:

- `emoji` обязателен при добавлении реакции.
- `emoji=""` удаляет реакцию(и) бота, когда поддерживается.
- `remove: true` удаляет указанный emoji, когда поддерживается (требуется `emoji`).

Примечания по каналам:

- **Discord/Slack**: пустой `emoji` удаляет все реакции бота на сообщении; `remove: true` удаляет только этот emoji.
- **Google Chat**: пустой `emoji` удаляет реакции приложения на сообщении; `remove: true` удаляет только этот emoji.
- **Telegram**: пустой `emoji` удаляет реакции бота; `remove: true` также удаляет реакции, но все еще требует непустой `emoji` для валидации инструмента.
- **WhatsApp**: пустой `emoji` удаляет реакцию бота; `remove: true` отображается на пустой emoji (все еще требуется `emoji`).
- **Signal**: входящие уведомления о реакциях выдают системные события, когда включен `channels.signal.reactionNotifications`.
