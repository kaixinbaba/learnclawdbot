# `openclaw health`

Получить состояние от запущенного Шлюза.

```bash
openclaw health
openclaw health --json
openclaw health --verbose
```

Примечания:
- `--verbose` запускает живые проверки и выводит тайминги для каждого аккаунта, когда настроено несколько аккаунтов.
- Вывод включает хранилища сессий для каждого агента, когда настроено несколько агентов.
