# `openclaw onboard`

Интерактивный мастер настройки (локальная или удаленная настройка Gateway).

Связанные темы:
- Руководство по мастеру: [Onboarding](/start/onboarding)

## Примеры

```bash
openclaw onboard
openclaw onboard --flow quickstart
openclaw onboard --flow manual
openclaw onboard --mode remote --remote-url ws://gateway-host:18789
```

Примечания к потокам:
- `quickstart`: минимум подсказок, автоматическая генерация токена gateway.
- `manual`: полные подсказки для port/bind/auth (псевдоним `advanced`).
- Самый быстрый первый чат: `openclaw dashboard` (Control UI, без настройки канала).
