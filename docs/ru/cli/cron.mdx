# `openclaw cron`

Управление задачами cron для планировщика Шлюза.

Связанные разделы:
- Задачи Cron: [Cron jobs](/automation/cron-jobs)

Совет: запустите `openclaw cron --help` для полной поверхности команд.

## Общие правки

Обновление настроек доставки без изменения сообщения:

```bash
openclaw cron edit <job-id> --deliver --channel telegram --to "123456789"
```

Отключение доставки для изолированной задачи:

```bash
openclaw cron edit <job-id> --no-deliver
```
