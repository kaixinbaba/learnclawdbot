# `openclaw dns`

Помощники DNS для wide-area discovery (Tailscale + CoreDNS). В настоящее время сфокусированы на macOS + Homebrew CoreDNS.

Связанные разделы:
- Обнаружение шлюза: [Discovery](/gateway/discovery)
- Конфигурация wide-area discovery: [Configuration](/gateway/configuration)

## Настройка

```bash
openclaw dns setup
openclaw dns setup --apply
```
