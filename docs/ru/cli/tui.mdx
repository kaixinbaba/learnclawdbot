# `openclaw tui`

Открытие терминального UI, подключенного к Gateway.

Связанные разделы:
- Руководство по TUI: [TUI](/tui)

## Примеры

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
