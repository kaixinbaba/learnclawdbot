# `openclaw status`

Диагностика для каналов + сессий.

```bash
openclaw status
openclaw status --all
openclaw status --deep
openclaw status --usage
```

Примечания:
- `--deep` выполняет живые проверки (WhatsApp Web + Telegram + Discord + Google Chat + Slack + Signal).
- Вывод включает хранилища сессий для каждого агента, когда настроено несколько агентов.
- Обзор включает статус установки/среды выполнения службы шлюза + хоста узла, когда доступно.
- Обзор включает канал обновлений + git SHA (для исходных checkout).
- Информация об обновлениях отображается в обзоре; если доступно обновление, status выводит подсказку запустить `openclaw update` (см. [Обновление](/install/updating)).
