# `openclaw skills`

Проверка навыков (встроенных + рабочего пространства + управляемых переопределений) и просмотр того, что подходит или не соответствует требованиям.

Связанное:
- Система навыков: [Навыки](/tools/skills)
- Конфигурация навыков: [Конфигурация навыков](/tools/skills-config)
- Установки ClawdHub: [ClawdHub](/tools/clawdhub)

## Команды

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
