# `openclaw setup`

Инициализация `~/.openclaw/openclaw.json` и рабочего пространства агента.

Связанные разделы:
- Начало работы: [Начало работы](/start/getting-started)
- Мастер настройки: [Первоначальная настройка](/start/onboarding)

## Примеры

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Для запуска мастера через setup:

```bash
openclaw setup --wizard
```
