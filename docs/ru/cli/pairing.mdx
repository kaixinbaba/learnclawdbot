# `openclaw pairing`

Одобрение или проверка запросов на сопряжение DM (для каналов, поддерживающих сопряжение).

Связанные темы:
- Процесс сопряжения: [Pairing](/start/pairing)

## Команды

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
