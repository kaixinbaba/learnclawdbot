# `openclaw message`

Единая исходящая команда для отправки сообщений и действий каналов
(Discord/Google Chat/Slack/Mattermost (плагин)/Telegram/WhatsApp/Signal/iMessage/MS Teams).

## Использование

```
openclaw message <subcommand> [flags]
```

Выбор канала:
- `--channel` требуется если настроено более одного канала.
- Если настроен ровно один канал, он становится значением по умолчанию.
- Значения: `whatsapp|telegram|discord|googlechat|slack|mattermost|signal|imessage|msteams` (Mattermost требует плагин)

Форматы целей (`--target`):
- WhatsApp: E.164 или JID группы
- Telegram: id чата или `@username`
- Discord: `channel:<id>` или `user:<id>` (или упоминание `<@id>`; необработанные числовые id обрабатываются как каналы)
- Google Chat: `spaces/<spaceId>` или `users/<userId>`
- Slack: `channel:<id>` или `user:<id>` (принимается необработанный id канала)
- Mattermost (плагин): `channel:<id>`, `user:<id>`, или `@username` (голые id обрабатываются как каналы)
- Signal: `+E.164`, `group:<id>`, `signal:+E.164`, `signal:group:<id>`, или `username:<name>`/`u:<name>`
- iMessage: handle, `chat_id:<id>`, `chat_guid:<guid>`, или `chat_identifier:<id>`
- MS Teams: id беседы (`19:...@thread.tacv2`) или `conversation:<id>` или `user:<aad-object-id>`

Поиск имени:
- Для поддерживаемых провайдеров (Discord/Slack/и т.д.), имена каналов типа `Help` или `#help` разрешаются через кэш справочника.
- При промахе кэша, OpenClaw попытается выполнить живой поиск в справочнике, когда провайдер это поддерживает.

## Общие флаги

- `--channel <name>`
- `--account <id>`
- `--target <dest>` (целевой канал или пользователь для send/poll/read/и т.д.)
- `--targets <name>` (повторить; только broadcast)
- `--json`
- `--dry-run`
- `--verbose`

## Действия

### Основные

- `send`
  - Каналы: WhatsApp/Telegram/Discord/Google Chat/Slack/Mattermost (плагин)/Signal/iMessage/MS Teams
  - Требуется: `--target`, плюс `--message` или `--media`
  - Опционально: `--media`, `--reply-to`, `--thread-id`, `--gif-playback`
  - Только Telegram: `--buttons` (требует `channels.telegram.capabilities.inlineButtons` для разрешения)
  - Только Telegram: `--thread-id` (id темы форума)
  - Только Slack: `--thread-id` (метка времени треда; `--reply-to` использует то же поле)
  - Только WhatsApp: `--gif-playback`

- `poll`
  - Каналы: WhatsApp/Discord/MS Teams
  - Требуется: `--target`, `--poll-question`, `--poll-option` (повторить)
  - Опционально: `--poll-multi`
  - Только Discord: `--poll-duration-hours`, `--message`

- `react`
  - Каналы: Discord/Google Chat/Slack/Telegram/WhatsApp/Signal
  - Требуется: `--message-id`, `--target`
  - Опционально: `--emoji`, `--remove`, `--participant`, `--from-me`, `--target-author`, `--target-author-uuid`
  - Примечание: `--remove` требует `--emoji` (опустите `--emoji` для очистки собственных реакций где поддерживается; см. /tools/reactions)
  - Только WhatsApp: `--participant`, `--from-me`
  - Групповые реакции Signal: требуется `--target-author` или `--target-author-uuid`

- `reactions`
  - Каналы: Discord/Google Chat/Slack
  - Требуется: `--message-id`, `--target`
  - Опционально: `--limit`

- `read`
  - Каналы: Discord/Slack
  - Требуется: `--target`
  - Опционально: `--limit`, `--before`, `--after`
  - Только Discord: `--around`

- `edit`
  - Каналы: Discord/Slack
  - Требуется: `--message-id`, `--message`, `--target`

- `delete`
  - Каналы: Discord/Slack/Telegram
  - Требуется: `--message-id`, `--target`

- `pin` / `unpin`
  - Каналы: Discord/Slack
  - Требуется: `--message-id`, `--target`

- `pins` (список)
  - Каналы: Discord/Slack
  - Требуется: `--target`

- `permissions`
  - Каналы: Discord
  - Требуется: `--target`

- `search`
  - Каналы: Discord
  - Требуется: `--guild-id`, `--query`
  - Опционально: `--channel-id`, `--channel-ids` (повторить), `--author-id`, `--author-ids` (повторить), `--limit`

### Треды

- `thread create`
  - Каналы: Discord
  - Требуется: `--thread-name`, `--target` (id канала)
  - Опционально: `--message-id`, `--auto-archive-min`

- `thread list`
  - Каналы: Discord
  - Требуется: `--guild-id`
  - Опционально: `--channel-id`, `--include-archived`, `--before`, `--limit`

- `thread reply`
  - Каналы: Discord
  - Требуется: `--target` (id треда), `--message`
  - Опционально: `--media`, `--reply-to`

### Эмодзи

- `emoji list`
  - Discord: `--guild-id`
  - Slack: дополнительные флаги не требуются

- `emoji upload`
  - Каналы: Discord
  - Требуется: `--guild-id`, `--emoji-name`, `--media`
  - Опционально: `--role-ids` (повторить)

### Стикеры

- `sticker send`
  - Каналы: Discord
  - Требуется: `--target`, `--sticker-id` (повторить)
  - Опционально: `--message`

- `sticker upload`
  - Каналы: Discord
  - Требуется: `--guild-id`, `--sticker-name`, `--sticker-desc`, `--sticker-tags`, `--media`

### Роли / Каналы / Участники / Голос

- `role info` (Discord): `--guild-id`
- `role add` / `role remove` (Discord): `--guild-id`, `--user-id`, `--role-id`
- `channel info` (Discord): `--target`
- `channel list` (Discord): `--guild-id`
- `member info` (Discord/Slack): `--user-id` (+ `--guild-id` для Discord)
- `voice status` (Discord): `--guild-id`, `--user-id`

### События

- `event list` (Discord): `--guild-id`
- `event create` (Discord): `--guild-id`, `--event-name`, `--start-time`
  - Опционально: `--end-time`, `--desc`, `--channel-id`, `--location`, `--event-type`

### Модерация (Discord)

- `timeout`: `--guild-id`, `--user-id` (опционально `--duration-min` или `--until`; опустите оба для очистки таймаута)
- `kick`: `--guild-id`, `--user-id` (+ `--reason`)
- `ban`: `--guild-id`, `--user-id` (+ `--delete-days`, `--reason`)
  - `timeout` также поддерживает `--reason`

### Broadcast

- `broadcast`
  - Каналы: любой настроенный канал; используйте `--channel all` для нацеливания на всех провайдеров
  - Требуется: `--targets` (повторить)
  - Опционально: `--message`, `--media`, `--dry-run`

## Примеры

Отправить ответ Discord:
```
openclaw message send --channel discord \
  --target channel:123 --message "hi" --reply-to 456
```

Создать опрос Discord:
```
openclaw message poll --channel discord \
  --target channel:123 \
  --poll-question "Snack?" \
  --poll-option Pizza --poll-option Sushi \
  --poll-multi --poll-duration-hours 48
```

Отправить проактивное сообщение Teams:
```
openclaw message send --channel msteams \
  --target conversation:19:abc@thread.tacv2 --message "hi"
```

Создать опрос Teams:
```
openclaw message poll --channel msteams \
  --target conversation:19:abc@thread.tacv2 \
  --poll-question "Lunch?" \
  --poll-option Pizza --poll-option Sushi
```

Реагировать в Slack:
```
openclaw message react --channel slack \
  --target C123 --message-id 456 --emoji "✅"
```

Реагировать в группе Signal:
```
openclaw message react --channel signal \
  --target signal:group:abc123 --message-id 1737630212345 \
  --emoji "✅" --target-author-uuid 123e4567-e89b-12d3-a456-426614174000
```

Отправить встроенные кнопки Telegram:
```
openclaw message send --channel telegram --target @mychat --message "Choose:" \
  --buttons '[ [\{"text":"Yes","callback_data":"cmd:yes"\}], [\{"text":"No","callback_data":"cmd:no"\}] ]'
```
