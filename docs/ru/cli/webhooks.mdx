# `openclaw webhooks`

Помощники вебхуков и интеграции (Gmail Pub/Sub, помощники вебхуков).

Связанные разделы:
- Вебхуки: [Вебхуки](/automation/webhook)
- Gmail Pub/Sub: [Gmail Pub/Sub](/automation/gmail-pubsub)

## Gmail

```bash
openclaw webhooks gmail setup --account you@example.com
openclaw webhooks gmail run
```

См. [документацию Gmail Pub/Sub](/automation/gmail-pubsub) для подробностей.
