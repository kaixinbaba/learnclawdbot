# `openclaw browser`

Управление сервером управления браузером OpenClaw и выполнение действий браузера (вкладки, снимки, скриншоты, навигация, клики, ввод текста).

Связанные разделы:
- Инструмент браузера + API: [Browser tool](/tools/browser)
- Relay расширения Chrome: [Chrome extension](/tools/chrome-extension)

## Общие флаги

- `--url <gatewayWsUrl>`: URL WebSocket шлюза (по умолчанию из конфига).
- `--token <token>`: Токен шлюза (если требуется).
- `--timeout <ms>`: таймаут запроса (мс).
- `--browser-profile <name>`: выбор профиля браузера (по умолчанию из конфига).
- `--json`: машинночитаемый вывод (где поддерживается).

## Быстрый старт (локально)

```bash
openclaw browser --browser-profile chrome tabs
openclaw browser --browser-profile openclaw start
openclaw browser --browser-profile openclaw open https://example.com
openclaw browser --browser-profile openclaw snapshot
```

## Профили

Профили — это именованные конфигурации маршрутизации браузера. На практике:
- `openclaw`: запускает/подключается к выделенному экземпляру Chrome, управляемому OpenClaw (изолированный каталог пользовательских данных).
- `chrome`: управляет вашими существующими вкладками Chrome через relay расширения Chrome.

```bash
openclaw browser profiles
openclaw browser create-profile --name work --color "#FF5A36"
openclaw browser delete-profile --name work
```

Использование конкретного профиля:

```bash
openclaw browser --browser-profile work tabs
```

## Вкладки

```bash
openclaw browser tabs
openclaw browser open https://docs.openclaw.ai
openclaw browser focus <targetId>
openclaw browser close <targetId>
```

## Snapshot / screenshot / действия

Snapshot:

```bash
openclaw browser snapshot
```

Screenshot:

```bash
openclaw browser screenshot
```

Navigate/click/type (автоматизация UI на основе ссылок):

```bash
openclaw browser navigate https://example.com
openclaw browser click <ref>
openclaw browser type <ref> "hello"
```

## Relay расширения Chrome (подключение через кнопку на панели инструментов)

Этот режим позволяет агенту управлять существующей вкладкой Chrome, которую вы подключаете вручную (автоматическое подключение не выполняется).

Установите распакованное расширение в стабильный путь:

```bash
openclaw browser extension install
openclaw browser extension path
```

Затем Chrome → `chrome://extensions` → включите "Developer mode" → "Load unpacked" → выберите напечатанную папку.

Полное руководство: [Chrome extension](/tools/chrome-extension)

## Удалённое управление браузером (прокси узлового хоста)

Если Шлюз работает на другой машине, чем браузер, запустите **node host** на машине с Chrome/Brave/Edge/Chromium. Шлюз будет проксировать действия браузера на этот узел (отдельный сервер управления браузером не требуется).

Используйте `gateway.nodes.browser.mode` для управления авто-маршрутизацией и `gateway.nodes.browser.node` для закрепления конкретного узла, если подключено несколько.

Безопасность + удалённая настройка: [Browser tool](/tools/browser), [Remote access](/gateway/remote), [Tailscale](/gateway/tailscale), [Security](/gateway/security)
