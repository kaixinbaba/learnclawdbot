# `openclaw logs`

Просмотр файловых логов Шлюза через RPC (работает в удалённом режиме).

Связанные разделы:
- Обзор логирования: [Logging](/logging)

## Примеры

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
