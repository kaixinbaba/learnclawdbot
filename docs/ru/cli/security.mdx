# `openclaw security`

Инструменты безопасности (аудит + опциональные исправления).

Связанные разделы:
- Руководство по безопасности: [Безопасность](/gateway/security)

## Аудит

```bash
openclaw security audit
openclaw security audit --deep
openclaw security audit --fix
```

Аудит предупреждает, когда несколько отправителей личных сообщений используют общую основную сессию, и рекомендует `session.dmScope="per-channel-peer"` (или `per-account-channel-peer` для многоаккаунтных каналов) для общих почтовых ящиков.
Также предупреждает, когда используются малые модели (`<=300B`) без песочницы и с включенными веб/браузерными инструментами.
