---
title: Развертывание на Northflank
---

Разверните OpenClaw на Northflank с помощью шаблона в один клик и завершите настройку в браузере.
Это самый простой путь "без терминала на сервере": Northflank запускает шлюз за вас,
и вы настраиваете все через веб-мастер `/setup`.

## Как начать

1. Нажмите [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw), чтобы открыть шаблон.  
2. Создайте [учетную запись на Northflank](https://app.northflank.com/signup), если у вас ее еще нет.  
3. Нажмите **Deploy OpenClaw now**.  
4. Установите требуемую переменную окружения: `SETUP_PASSWORD`.
5. Нажмите **Deploy stack**, чтобы собрать и запустить шаблон OpenClaw.
6. Дождитесь завершения развертывания, затем нажмите **View resources**.
7. Откройте службу OpenClaw.
8. Откройте публичный URL OpenClaw и завершите настройку по адресу `/setup`.
9. Откройте UI управления по адресу `/openclaw`.

## Что вы получаете

- Хостинг шлюза OpenClaw + UI управления
- Веб-мастер настройки по адресу `/setup` (без команд терминала)
- Постоянное хранилище через Northflank Volume (`/data`), чтобы конфигурация/учетные данные/рабочее пространство пережили переразвертывания

## Поток настройки

1) Посетите `https://<your-northflank-domain>/setup` и введите ваш `SETUP_PASSWORD`.
2) Выберите модель/провайдера аутентификации и вставьте ваш ключ.
3) (Опционально) Добавьте токены Telegram/Discord/Slack.
4) Нажмите **Run setup**.
5) Откройте UI управления по адресу `https://<your-northflank-domain>/openclaw`

Если DM Telegram установлены на сопряжение, мастер настройки может одобрить код сопряжения.

## Получение токенов чата

### Токен бота Telegram

1) Напишите сообщение `@BotFather` в Telegram
2) Запустите `/newbot`
3) Скопируйте токен (выглядит как `123456789:AA...`)
4) Вставьте его в `/setup`

### Токен бота Discord

1) Перейдите на https://discord.com/developers/applications
2) **New Application** → выберите имя
3) **Bot** → **Add Bot**
4) **Enable MESSAGE CONTENT INTENT** под Bot → Privileged Gateway Intents (требуется, иначе бот вылетит при запуске)
5) Скопируйте **Bot Token** и вставьте в `/setup`
6) Пригласите бота на ваш сервер (OAuth2 URL Generator; области: `bot`, `applications.commands`)
