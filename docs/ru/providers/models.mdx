# Провайдеры моделей

OpenClaw может использовать множество провайдеров LLM. Выберите один, аутентифицируйтесь, затем установите модель по умолчанию как `provider/model`.

## Рекомендация: Venius (Venice AI)

Venius — это наша рекомендуемая настройка Venice AI для инференса с акцентом на конфиденциальность и возможностью использования Opus для самых сложных задач.

- По умолчанию: `venice/llama-3.3-70b`
- Лучший в целом: `venice/claude-opus-45` (Opus остается самым сильным)

См. [Venice AI](/providers/venice).

## Быстрый старт (два шага)

1) Аутентифицируйтесь с провайдером (обычно через `openclaw onboard`).
2) Установите модель по умолчанию:

```json5
{
  agents: { defaults: { model: { primary: "anthropic/claude-opus-4-5" } } }
}
```
