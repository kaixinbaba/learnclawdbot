# Модели GLM

GLM — это **семейство моделей** (не компания), доступное через платформу Z.AI. В OpenClaw модели GLM доступны через провайдера `zai` и ID моделей типа `zai/glm-4.7`.

## Настройка через CLI

```bash
openclaw onboard --auth-choice zai-api-key
```

## Фрагмент конфигурации

```json5
{
  env: { ZAI_API_KEY: "sk-..." },
  agents: { defaults: { model: { primary: "zai/glm-4.7" } } }
}
```

## Примечания

- Версии GLM и доступность могут меняться; проверьте документацию Z.AI для получения последней информации.
- Примеры ID моделей включают `glm-4.7` и `glm-4.6`.
- Для получения подробной информации о провайдере см. [/providers/zai](/providers/zai).
