# OpenAI

OpenAI предоставляет API для разработчиков для моделей GPT. Codex поддерживает **вход в ChatGPT** для доступа по подписке или **вход с API ключом** для доступа с оплатой по использованию. Облако Codex требует входа в ChatGPT.

## Вариант A: API ключ OpenAI (платформа OpenAI)

**Лучше для:** прямого доступа к API и оплаты по использованию.
Получите свой API ключ на панели управления OpenAI.

### Настройка через CLI

```bash
openclaw onboard --auth-choice openai-api-key
# или неинтерактивно
openclaw onboard --openai-api-key "$OPENAI_API_KEY"
```

### Фрагмент конфигурации

```json5
{
  env: { OPENAI_API_KEY: "sk-..." },
  agents: { defaults: { model: { primary: "openai/gpt-5.2" } } }
}
```

## Вариант B: подписка OpenAI Code (Codex)

**Лучше для:** использования доступа по подписке ChatGPT/Codex вместо API ключа.
Облако Codex требует входа в ChatGPT, в то время как CLI Codex поддерживает вход в ChatGPT или с API ключом.

### Настройка через CLI

```bash
# Запустите OAuth Codex в мастере
openclaw onboard --auth-choice openai-codex

# Или запустите OAuth напрямую
openclaw models auth login --provider openai-codex
```

### Фрагмент конфигурации

```json5
{
  agents: { defaults: { model: { primary: "openai-codex/gpt-5.2" } } }
}
```

## Примечания

- Ссылки на модели всегда используют `provider/model` (см. [/concepts/models](/concepts/models)).
- Детали аутентификации + правила повторного использования в [/concepts/oauth](/concepts/oauth).
