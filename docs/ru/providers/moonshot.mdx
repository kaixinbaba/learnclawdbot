# Moonshot AI (Kimi)

Moonshot предоставляет API Kimi с конечными точками, совместимыми с OpenAI. Настройте провайдера и установите модель по умолчанию на `moonshot/kimi-k2.5`, или используйте Kimi Code с `kimi-code/kimi-for-coding`.

Текущие ID моделей Kimi K2:
{/* moonshot-kimi-k2-ids:start */}
- `kimi-k2.5`
- `kimi-k2-0905-preview`
- `kimi-k2-turbo-preview`
- `kimi-k2-thinking`
- `kimi-k2-thinking-turbo`
{/* moonshot-kimi-k2-ids:end */}

```bash
openclaw onboard --auth-choice moonshot-api-key
```

Kimi Code:

```bash
openclaw onboard --auth-choice kimi-code-api-key
```

Примечание: Moonshot и Kimi Code — это отдельные провайдеры. Ключи не взаимозаменяемы, конечные точки различаются, и ссылки на модели различаются (Moonshot использует `moonshot/...`, Kimi Code использует `kimi-code/...`).

```json5

  agents: {

      model: { primary: "moonshot/kimi-k2.5" },

        // moonshot-kimi-k2-aliases:start
        "moonshot/kimi-k2.5": { alias: "Kimi K2.5" },
        "moonshot/kimi-k2-0905-preview": { alias: "Kimi K2" },
        "moonshot/kimi-k2-turbo-preview": { alias: "Kimi K2 Turbo" },
```

## Kimi Code

```json5
{
  env: { KIMICODE_API_KEY: "sk-..." },
  agents: {
    defaults: {
      model: { primary: "kimi-code/kimi-for-coding" },
      models: {
        "kimi-code/kimi-for-coding": { alias: "Kimi Code" }
      }
    }
  },
  models: {
    mode: "merge",
    providers: {
      "kimi-code": {
        baseUrl: "https://api.kimi.com/coding/v1",
        apiKey: "${KIMICODE_API_KEY}",
        api: "openai-completions",
        models: [
          {
            id: "kimi-for-coding",
            name: "Kimi For Coding",
            reasoning: true,
            input: ["text"],
            cost: { input: 0, output: 0, cacheRead: 0, cacheWrite: 0 },
            contextWindow: 262144,
            maxTokens: 32768,
            headers: { "User-Agent": "KimiCLI/0.77" },
            compat: { supportsDeveloperRole: false }
          }
        ]
      }
    }
  }
}
```

## Примечания

- Ссылки на модели Moonshot используют `moonshot/<modelId>`. Ссылки на модели Kimi Code используют `kimi-code/<modelId>`.
- Переопределите цены и метаданные контекста в `models.providers` при необходимости.
- Если Moonshot публикует другие ограничения контекста для модели, отрегулируйте `contextWindow` соответственно.
- Используйте `https://api.moonshot.cn/v1`, если вам нужна конечная точка Китая.
